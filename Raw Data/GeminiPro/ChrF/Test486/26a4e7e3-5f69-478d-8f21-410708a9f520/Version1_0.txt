```java
public class MiddleCharacter {

    public static void main(String[] args) {
        String str1 = "hello";
        String str2 = "java";

        System.out.println("Middle character of " + str1 + ": " + middle(str1));
        System.out.println("Middle character of " + str2 + ": " + middle(str2));


        String str3 = "coding"; // Example with even length
        System.out.println("Middle character of " + str3 + ": " + middle(str3));

        String str4 = "code";  // Example with odd length
        System.out.println("Middle character of " + str4 + ": " + middle(str4));


    }

    public static String middle(String str) {
        int len = str.length();
        if (len % 2 == 0) { // Even length
            return String.valueOf(str.charAt(len / 2 - 1)); // Corrected for 0-based indexing
        } else { // Odd length
            return String.valueOf(str.charAt(len / 2));
        }
    }
}
```


**Explanation and Corrections:**

1. **Even Length Logic:** The original prompt had a typo stating there would be *six* middle characters for an odd-length string; it should be *one* middle character for an *even* length string. The code is corrected to reflect this.  To get the correct middle character when the length is even, we need to use `len / 2 - 1` because string indices are zero-based.  For example, in "coding", 'd' is at index 2, and `6/2 -1` is `3-1 = 2`.

2. **Odd Length Logic:** For odd length strings, `len / 2` correctly identifies the middle character's index.

3. **`String.valueOf()`:** I've used `String.valueOf()` to convert the `char` returned by `str.charAt()` to a `String`. This is generally better practice than simply concatenating the character with an empty string because it handles potential `null` characters more gracefully.

4. **Clearer Examples:**  I've added more examples in the `main` method to demonstrate the function with both even and odd length strings, which makes it easier to verify the logic.
