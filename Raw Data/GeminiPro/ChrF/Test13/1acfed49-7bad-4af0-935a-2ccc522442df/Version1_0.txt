```java
import java.util.Scanner;

public class BinaryMultiplication {

    public static String multiplyBinary(String binary1, String binary2) {
        // Handle null or empty inputs
        if (binary1 == null || binary2 == null || binary1.isEmpty() || binary2.isEmpty()) {
            return "0"; 
        }

        // Validate binary strings (only 0s and 1s)
        if (!binary1.matches("[01]+") || !binary2.matches("[01]+")) {
            return "Invalid Input: Not binary strings";
        }

        int num1 = Integer.parseInt(binary1, 2);
        int num2 = Integer.parseInt(binary2, 2);
        int product = num1 * num2;

        return Integer.toBinaryString(product);
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the first binary number: ");
        String binary1 = scanner.nextLine();

        System.out.print("Enter the second binary number: ");
        String binary2 = scanner.nextLine();

        String result = multiplyBinary(binary1, binary2);
        System.out.println("Product: " + result);

        scanner.close();
    }
}
```

**Explanation and Educational Points:**

1. **Input Handling and Validation:**
   - The code first checks for `null` or empty input strings and returns "0" in such cases. This is important for robustness.
   - It also validates that the input strings contain only '0' and '1' characters using a regular expression (`[01]+`).  If the input is not valid, it returns an error message.

2. **Binary to Integer Conversion:**
   - `Integer.parseInt(binaryString, 2)` is used to convert the binary strings to their integer equivalents. The `2` specifies that the string is in base 2 (binary).

3. **Multiplication:**
   - The integer multiplication is performed using the standard `*` operator.

4. **Integer to Binary Conversion:**
   - `Integer.toBinaryString(product)` converts the integer product back into a binary string.

5. **User Input:**
   - The `Scanner` class is used to take user input from the console, making the program interactive.
