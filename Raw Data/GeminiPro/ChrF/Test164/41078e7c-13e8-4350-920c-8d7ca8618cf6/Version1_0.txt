**Problem Description:**

Write a Java program that trims leading and trailing spaces from a given string.

**Java Code:**

```java
import java.util.Scanner;

public class TrimString {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Get the input string
        System.out.println("Enter a string: ");
        String inputString = scanner.nextLine();

        // Trim the leading and trailing spaces
        String trimmedString = inputString.trim();

        // Print the trimmed string
        System.out.println("Trimmed string: " + trimmedString);
    }
}
```

**Test Cases:**

**Corner Cases:**

* Empty string: ""
* String with leading spaces: "    Hello"
* String with trailing spaces: "Hello    "
* String with both leading and trailing spaces: "    Hello    "

**Other Test Cases:**

* "Hello World"
* "12345"
* " " (single space)
* "  \n\t" (combination of spaces, newlines, and tabs)
