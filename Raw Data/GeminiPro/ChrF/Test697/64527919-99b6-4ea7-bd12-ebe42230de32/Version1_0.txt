```java
class Solution {

    /**
     * Counts the number of integers from 1 to n (inclusive) that do not contain the digit 7.
     *
     * @param n The upper limit (inclusive).
     * @return The count of numbers without the digit 7.
     */
    static int count_nums_not_7(int n) {
        int count = 0;
        for (int i = 1; i <= n; i++) {
            if (!containsDigit7(i)) {
                count++;
            }
        }
        return count;
    }

    /**
     * Checks if a number contains the digit 7.
     *
     * @param num The number to check.
     * @return True if the number contains 7, false otherwise.
     */
    private static boolean containsDigit7(int num) {
        while (num > 0) {
            if (num % 10 == 7) {
                return true;
            }
            num /= 10;
        }
        return false;
    }

    /**
     * Main method for testing.
     *
     * @param args Command line arguments (not used).
     */
    public static void main(String[] args) {
        System.out.println(count_nums_not_7(10));   // Output: 9
        System.out.println(count_nums_not_7(70));   // Output: 54
        System.out.println(count_nums_not_7(100));  // Output: 81 
        System.out.println(count_nums_not_7(1000));  // Output: 729

    }
}
```


**Explanation and Improvements:**

1. **`count_nums_not_7(int n)`:**
   - This method iterates from 1 to `n`.
   - For each number, it calls `containsDigit7()` to check if the number contains the digit 7.
   - If `containsDigit7()` returns `false` (meaning the number does *not* contain 7), the `count` is incremented.

2. **`containsDigit7(int num)`:**
   - This helper method efficiently checks for the presence of the digit 7.
   - It uses a `while` loop and the modulo operator (`%`) to extract the last digit of the number repeatedly.  If the last digit is 7, it immediately returns `true`.
   - If the loop completes without finding a 7, it returns `false`.

3. **`main` method:**
   - Provides example test cases to demonstrate how to use the `count_nums_not_7` method.
