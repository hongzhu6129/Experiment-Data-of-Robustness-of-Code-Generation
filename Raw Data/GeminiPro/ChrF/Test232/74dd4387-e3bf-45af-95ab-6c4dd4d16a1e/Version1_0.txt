```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueIndex = 1; // Index to track unique elements

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueIndex] = nums[i];
                uniqueIndex++;
            }
        }

        return uniqueIndex;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int newLength1 = removeDuplicates(nums1);
        System.out.println("New Length: " + newLength1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, newLength1)));

        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int newLength2 = removeDuplicates(nums2);
        System.out.println("New Length: " + newLength2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, newLength2)));
        
        int[] nums3 = {};
        int newLength3 = removeDuplicates(nums3);
        System.out.println("New Length: " + newLength3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, newLength3)));

        int[] nums4 = null;
        int newLength4 = removeDuplicates(nums4);
        System.out.println("New Length: " + newLength4 + (nums4 != null ? ", Array: " + Arrays.toString(Arrays.copyOfRange(nums4, 0, newLength4)) : "")); // Handle null input
    }
}
```


**Explanation:**

1. **`removeDuplicates(int[] nums)`:**
   - Handles `null` or empty arrays.
   - `uniqueIndex`: Tracks the index where the next unique element should be placed. It starts at 1 because the first element is always unique.
   - Iterates through the array starting from the second element.
   - If the current element `nums[i]` is different from the previous element `nums[i-1]`, it's a unique element.  So, we copy it to `nums[uniqueIndex]` and increment `uniqueIndex`.
   - Finally, `uniqueIndex` represents the length of the array with duplicates removed.

2. **`main(String[] args)`:**
   - Example usages with different input arrays, including empty and `null` arrays.  Demonstrates how to use the `removeDuplicates` function and print the results.
   - Uses `Arrays.copyOfRange` to create a new array containing only the unique elements for printing. This is important because the original array is modified in place, but its original length remains the same. The new array represents the array with duplicates removed.
   - Added handling for `null` input to avoid `NullPointerException`.
