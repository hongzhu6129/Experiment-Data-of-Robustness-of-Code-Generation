```java
import java.util.Scanner;

public class BinaryMultiplication {

    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Input the first binary number: ");
        String binary1 = input.nextLine();

        System.out.print("Input the second binary number: ");
        String binary2 = input.nextLine();

        int decimal1 = Integer.parseInt(binary1, 2); // Convert binary string to decimal integer
        int decimal2 = Integer.parseInt(binary2, 2);

        int product = binaryproduct(decimal1, decimal2);

        String binaryProduct = Integer.toBinaryString(product); // Convert product back to binary string
        System.out.println("Product of two binary numbers: " + binaryProduct);

        input.close();
    }

    static int binaryproduct(int a, int b) {
        return a * b;
    }
}
```

```java
static int binaryproduct(int a, int b) {
    int result = 0;
    int power = 1; // Represents powers of 2

    while (b > 0) {
        if ((b & 1) == 1) { // Check if the last bit of b is 1
            result += a * power;
        }
        power <<= 1; // Equivalent to power *= 2;
        b >>= 1;      // Equivalent to b /= 2 (integer division);
    }
    return result;
}
```


This version of `binaryproduct` performs the multiplication using the same logic as you would multiply binary numbers by hand.  It uses bitwise operators (`&`, `<<`, `>>`) for efficiency.  However, as mentioned, the initial version using decimal multiplication is perfectly valid for the given prompt.
