```java
public class LeapYear {

    public static void main(String[] args) {
        // Get the year from command-line arguments.  Handle potential errors.
        if (args.length != 1) {
            System.out.println("Usage: java LeapYear <year>");
            return; // Exit if no year is provided or too many arguments
        }

        try {
            int year = Integer.parseInt(args[0]);

            // Leap year rules:
            // 1. Divisible by 4
            // 2. Not divisible by 100 unless also divisible by 400

            boolean isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);


            if (isLeap) {
                System.out.println(year + " is a leap year.");
            } else {
                System.out.println(year + " is not a leap year.");
            }

        } catch (NumberFormatException e) {
            System.out.println("Invalid input: Please enter a valid year (integer).");
        }
    }
}
```


To compile and run this:

1. **Save:** Save the code as `LeapYear.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile it using:  `javac LeapYear.java`
3. **Run:** Execute the compiled code with the year as a command-line argument:  `java LeapYear 2024`  (Replace 2024 with the year you want to check).
