```java
public class Main {
    public static void main(String[] args) {
        // Swapping seven variables (a, b, c, d, e, f, g) using a temporary variable 'temp'
        int a = 1, b = 2, c = 3, d = 4, e = 5, f = 6, g = 7;
        int temp;

        System.out.println("Before swapping:");
        System.out.println("a = " + a + ", b = " + b + ", c = " + c + ", d = " + d + ", e = " + e + ", f = " + f + ", g = " + g);

        temp = a;
        a = b;
        b = c;
        c = d;
        d = e;
        e = f;
        f = g;
        g = temp;


        System.out.println("After swapping:");
        System.out.println("a = " + a + ", b = " + b + ", c = " + c + ", d = " + d + ", e = " + e + ", f = " + f + ", g = " + g);


        // Printing a face
        System.out.println("  +\"\"\"\"\"+\n [| o o |] \n  |  ^  | \n  | '-' | \n  +-----+");


    }
}
```


This code demonstrates two separate tasks within the `main` method:

1. **Swapping Seven Variables:**  It initializes seven integer variables (a through g) with different values.  It then uses a `temp` variable to swap their values in a circular manner. The values are printed before and after the swap to demonstrate the change.

2. **Printing a Face:**  It uses `System.out.println()` statements with carefully placed characters to create a simple ASCII art representation of a face.  The output will look like this:

```
  +\"\"\"\"+
 [| o o |] 
  |  ^  | 
  | '-' | 
  +-----+
```
