


**Test Cases:**

Here's a set of test cases covering various scenarios, including corner cases:

| Test Case | Input 1 | Input 2 | Expected Output |
|---|---|---|---|
| Positive Numbers | 10 | 5 | 10 |
| Negative Numbers | -5 | -10 | -5 |
| Zero and Positive | 0 | 5 | 5 |
| Zero and Negative | 0 | -5 | 0 |
| Equal Positive Numbers | 5 | 5 | 5 |
| Equal Negative Numbers | -5 | -5 | -5 |
| Positive and Negative | 10 | -5 | 10 |
| Negative and Positive | -10 | 5 | 5 |
| Large Positive Numbers | 2147483647 (Integer.MAX_VALUE) | 2147483646 | 2147483647 |  // Corner case: Max Value
| Large Negative Numbers | -2147483648 (Integer.MIN_VALUE) | -2147483647 | -2147483647 | // Corner case: Min Value
| Min and Max Value | -2147483648 | 2147483647 | 2147483647 | // Corner case: Min and Max


**How to run the tests (manually):**

1.  Compile the Java code: `javac FindLargest.java`
2.  Run the compiled code: `java FindLargest`
3.  The program will prompt you to enter the two integer values.  Enter the values from the test cases one by one and verify the output against the expected output.
