```java
import java.util.Scanner;

public class ProductOfTwoNumbers {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the first number: ");
        long num1 = scanner.nextLong();  // Use long to handle larger numbers and potential overflow

        System.out.print("Enter the second number: ");
        long num2 = scanner.nextLong();

        long product = num1 * num2;

        System.out.println("The product of " + num1 + " and " + num2 + " is: " + product);

        scanner.close(); 
    }
}
```

   ```java
   import org.junit.jupiter.api.Test;
   import static org.junit.jupiter.api.Assertions.*;

   public class ProductOfTwoNumbersTest {

       @Test

       }

       // ... other test methods for different cases ...

       @Test
       void testOverflow() {
           assertEquals(-2, ProductOfTwoNumbers.calculateProduct(9223372036854775807L, 2)); 
       }


       //Helper function to avoid running main method inside the test
       private long calculateProduct(long num1, long num2)
       {
           return num1*num2;
       }
   }
   ```

   This JUnit example shows how you can automate the process of testing various inputs and asserting (checking) that the output is what you expect.  You would need to add the JUnit library to your project.
