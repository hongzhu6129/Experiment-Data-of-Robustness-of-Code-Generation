```java
public class MatrixPrinter {

    public static void main(String[] args) {
        // Example usage:
        printMatrix(5); 
        System.out.println(); // Add a newline for clarity
        printMatrix(3);
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("[" + i + "," + j + "] "); // Print the matrix elements
            }
            System.out.println(); // Newline after each row
        }
    }
}
```


This code adheres strictly to the specified method signatures:

* **`public static void main(String[] args)`:**  This is the entry point for the program.  It demonstrates how to use the `printMatrix` method.
* **`public static void printMatrix(int n)`:**  This method takes an integer `n` as input and prints an n-by-n matrix to the console.  Each element in the matrix is displayed in the format `[row,column]`.


**Example Output (for `printMatrix(5)`):**

```
[0,0] [0,1] [0,2] [0,3] [0,4] 
[1,0] [1,1] [1,2] [1,3] [1,4] 
[2,0] [2,1] [2,2] [2,3] [2,4] 
[3,0] [3,1] [3,2] [3,3] [3,4] 
[4,0] [4,1] [4,2] [4,3] [4,4] 
```
