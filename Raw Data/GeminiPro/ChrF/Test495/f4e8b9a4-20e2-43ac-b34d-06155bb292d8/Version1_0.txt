```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input: n must be an integer.");
        }
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print((i * n + j +1) + " "); // Print element value (example: 1, 2, 3... ) You can modify this as needed.
            }
            System.out.println(); // New line after each row
        }
    }
}
```


**Explanation and How to use:**

1. **Compilation:** Save the code as `MatrixPrinter.java` and compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java 
   ```

2. **Running:**  Run the compiled code from your terminal, providing the matrix size `n` as a command-line argument:
   ```bash
   java MatrixPrinter 5  // Creates a 5x5 matrix
   java MatrixPrinter 3  // Creates a 3x3 matrix
   ```

3. **Inside `main`:**
   - The `main` method first checks if exactly one command-line argument (the value of `n`) is provided. It handles incorrect usage by printing an error message.
   - It then uses `Integer.parseInt()` to convert the string argument to an integer.  A `try-catch` block handles potential `NumberFormatException` if the input is not a valid integer.
   - Finally, it calls the `printMatrix` method to generate and print the matrix.

4. **Inside `printMatrix`:**
   - The `printMatrix` method takes the integer `n` as input.
   - It uses nested loops to iterate through the rows and columns of the matrix (from 0 to n-1).
   - Inside the inner loop, it prints the value of each matrix element.  In this example, it prints `(i * n + j + 1)`, which generates a sequence of numbers from 1 up to n*n. You can easily modify the value being printed here to display other patterns or data as required.  
   - `System.out.println()` is used after each row to create the matrix structure.


**Example Output (for `java MatrixPrinter 3`):**

```
1 2 3 
4 5 6 
7 8 9 
```
