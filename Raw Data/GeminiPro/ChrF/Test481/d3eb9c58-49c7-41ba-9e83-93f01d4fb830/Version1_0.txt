```java
import java.time.ZoneId;
import java.time.ZoneOffset;
import java.time.ZonedDateTime;
import java.util.Set;

public class ExtractZoneOffsets {

    public static void main(String[] args) {
        Set<String> allZoneIds = ZoneId.getAvailableZoneIds();

        for (String zoneIdString : allZoneIds) {
            ZoneId zoneId = ZoneId.of(zoneIdString);
            ZonedDateTime zonedDateTime = ZonedDateTime.now(zoneId);
            ZoneOffset zoneOffset = zonedDateTime.getOffset();

            int offsetSeconds = zoneOffset.getTotalSeconds();
            String formattedOffset = formatOffset(offsetSeconds);

            System.out.println("Zone ID: " + zoneIdString + ", Offset: " + formattedOffset);
        }
    }

    private static String formatOffset(int offsetSeconds) {
        String sign = offsetSeconds >= 0 ? "+" : "-";
        int absOffsetSeconds = Math.abs(offsetSeconds);
        int hours = absOffsetSeconds / 3600;
        int minutes = (absOffsetSeconds % 3600) / 60;
        int seconds = absOffsetSeconds % 60;

        return String.format("%s%02d:%02d:%02d", sign, hours, minutes, seconds); // Format as +HH:mm:ss or -HH:mm:ss
    }
}
```


This code fulfills the requirements:

1. **Uses only `main` and `formatOffset`:**  The core logic resides within the `main` method, and the offset formatting is delegated to the `formatOffset` method.

2. **Extracts Zone Offsets:**  It iterates through available `ZoneId`s, gets the current `ZonedDateTime` for each zone, extracts the `ZoneOffset`, and prints it in a formatted way.

3. **`formatOffset` Method Signature:** The `formatOffset` method takes an integer (offset in seconds) and returns a formatted string as required.

4. **Clear Formatting:**  The `formatOffset` method ensures the output is in the format `+HH:mm:ss` or `-HH:mm:ss`.


Key improvements:

- **Handles all available zones:** Uses `ZoneId.getAvailableZoneIds()` to get a comprehensive list.
- **Correct offset calculation:** Uses `getTotalSeconds()` to reliably get the offset.
- **Clearer formatting:**  Uses `String.format()` for clean and consistent output.
- **Handles both positive and negative offsets:** Includes logic for the sign of the offset.
- **Handles seconds correctly:** The formatting now includes seconds (which can be non-zero for some zones).


This revised code is more robust, comprehensive, and adheres more strictly to the problem's constraints.
