```java
class Node {
    int data;
    Node left, mid, right;

    public Node(int item) {
        data = item;
        left = mid = right = null;
    }
}

public class TernaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            int leftDepth = maxDepth(node.left);
            int midDepth = maxDepth(node.mid);
            int rightDepth = maxDepth(node.right);

            return Math.max(leftDepth, Math.max(midDepth, rightDepth)) + 1; 
        }
    }

    public static void main(String[] args) {
        TernaryTreeMaxDepth tree = new TernaryTreeMaxDepth();
        Node root = new Node(1);
        root.left = new Node(2);
        root.mid = new Node(3);
        root.right = new Node(4);
        root.left.left = new Node(5);
        root.left.mid = new Node(6);
        root.left.right = new Node(7);


        System.out.println("Maximum depth of the ternary tree is: " + tree.maxDepth(root));


         root = new Node(10);
        root.left = new Node(20);
        root.mid = new Node(30);
        root.right = new Node(40);
        root.left.left = new Node(50);
        root.mid.mid = new Node(60);
        root.right.right = new Node(70);

        System.out.println("Maximum depth of the ternary tree is: " + tree.maxDepth(root));


        // Test with an empty tree
        root = null;
        System.out.println("Maximum depth of the ternary tree is: " + tree.maxDepth(root));

        // Test with a single node tree
        root = new Node(100);
        System.out.println("Maximum depth of the ternary tree is: " + tree.maxDepth(root));


    }
}
```


Key improvements and explanations:

1. **Clearer Recursion:** The `maxDepth` method uses recursion to calculate the depth. The base case is when the `node` is `null` (empty subtree), returning 0.  Otherwise, it recursively calculates the depth of the left, middle, and right subtrees and returns the maximum of those depths plus 1 (for the current node).

2. **Comprehensive `main` method:** The `main` method now includes several test cases:
   - A basic ternary tree.
   - A tree with different depths in different branches.
   - An empty tree.
   - A tree with a single node. This helps demonstrate the correctness of the `maxDepth` method in various scenarios.

3. **`Node` class:** The `Node` class encapsulates the structure of a node in the ternary tree.  It has a `data` field and `left`, `mid`, and `right` pointers to its children.

4. **Conciseness and Readability:**  The code is written in a clear and concise manner, making it easy to understand and follow.  Good variable names and comments further enhance readability.
