```java
import java.util.Scanner;

public class SumOfDigits {

    public static int function(int n) {
        int sum = 0;
        n = Math.abs(n); // Handle negative numbers

        while (n > 0) {
            sum += n % 10;
            n /= 10;
        }

        return sum;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter an integer: ");
        int num = scanner.nextInt();
        int sum = function(num);
        System.out.println("Sum of digits: " + sum);
        scanner.close(); 
    }
}
```

**Explanation of Corner/Edge Cases and how they are handled:**

* **Zero (0):**  The `while` loop condition `n > 0` correctly handles zero.  The loop doesn't execute, and the sum remains 0.
* **Negative Numbers:** The code uses `Math.abs(n)` to handle negative numbers. It calculates the sum of the digits of the absolute value.
* **Large Positive/Negative Numbers (Integer.MAX_VALUE/Integer.MIN_VALUE):** These test cases ensure the code works correctly with the largest possible integer values. The code handles these gracefully because the logic operates digit by digit, so the magnitude of the overall number doesn't cause issues.
