```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        }

        int leftDepth = maxDepth(node.left);
        int rightDepth = maxDepth(node.right);

        return Math.max(leftDepth, rightDepth) + 1; 
    }

    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.left.right.left = new Node(6); // Added a deeper node


        System.out.println("Max depth of the tree is: " + tree.maxDepth(root)); // Output: 4

        // Another test case
        Node root2 = null;
        System.out.println("Max depth of the empty tree is: " + tree.maxDepth(root2)); // Output: 0


        Node root3 = new Node(7); // Single node tree
        System.out.println("Max depth of the single node tree is: " + tree.maxDepth(root3)); // Output 1


    }
}
```


**Explanation:**

1. **Node Class:** Represents a node in the binary tree. It has `data`, `left` child, and `right` child.

2. **maxDepth(Node node):**
   - This recursive method calculates the maximum depth.
   - **Base Case:** If the `node` is `null` (empty subtree), the depth is 0.
   - **Recursive Step:**  It recursively calculates the depth of the left subtree (`leftDepth`) and the right subtree (`rightDepth`).
   - **Return:** It returns the maximum of `leftDepth` and `rightDepth`, plus 1 (to account for the current node).

3. **main(String[] args):**
   - Creates a `BinaryTreeMaxDepth` object.
   - Constructs sample binary trees (including an empty tree and a single-node tree for testing).
   - Calls `maxDepth()` to calculate and print the maximum depth of each tree.


**How it works (example):**

Let's trace the first example tree:

```
      1
     / \
    2   3
   / \
  4   5
     /
    6 
```

- `maxDepth(1)`:
  - `leftDepth = maxDepth(2)`
  - `rightDepth = maxDepth(3)`
- `maxDepth(2)`:
  - `leftDepth = maxDepth(4) = 1` (4 is a leaf node)
  - `rightDepth = maxDepth(5)` 
- `maxDepth(5)`:
   - `leftDepth = maxDepth(6) = 1`
   - `rightDepth = maxDepth(null) = 0`
   - returns `max(1, 0) + 1 = 2`
- Back to `maxDepth(2)`: returns `max(1, 2) + 1 = 3`
- `maxDepth(3)`: returns `1` (3 is a leaf node)
- Back to `maxDepth(1)`: returns `max(3, 1) + 1 = 4`



Therefore, the maximum depth is correctly calculated as 4.
