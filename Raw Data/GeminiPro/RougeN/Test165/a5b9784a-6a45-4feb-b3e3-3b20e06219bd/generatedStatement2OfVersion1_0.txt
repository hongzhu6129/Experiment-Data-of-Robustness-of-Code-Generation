

**Test Cases:**

* **Valid:**
    * Input: "abcdefghij", numParts = 3
    * Expected Output:
        * Part 1: abc
        * Part 2: def
        * Part 3: ghi
* **Invalid (number of parts too small):**
    * Input: "abcdefghij", numParts = 0
    * Expected Output: Invalid number of parts
* **Invalid (string length not divisible by number of parts):**
    * Input: "abcdefghij", numParts = 5
    * Expected Output: Invalid number of parts
* **Corner case (number of parts is 1):**
    * Input: "abcdefghij", numParts = 1
    * Expected Output:
        * Part 1: abcdefghij
* **Corner case (string is empty):**
    * Input: "", numParts = 3
    * Expected Output: Invalid number of parts
