```java
import java.util.Scanner;

public class ProductOfTwoNumbers {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        System.out.print("Enter the first number: ");
        long num1 = input.nextLong(); // Use long to handle larger numbers

        System.out.print("Enter the second number: ");
        long num2 = input.nextLong();

        long product = num1 * num2;

        System.out.println("The product of " + num1 + " and " + num2 + " is: " + product);

        input.close(); 
    }
}
```

**Explanation of Corner Cases and Overflow:**

* **Zero:** Multiplying any number by zero results in zero.
* **Maximum/Minimum Long Values:**  Testing with the largest and smallest possible `long` values is crucial to check how your program handles these boundaries.
* **Overflow:** When the product of two numbers exceeds the maximum or minimum value that a `long` can hold, overflow occurs.  The result wraps around.  This is demonstrated in test cases 9 and 10. It is important to be aware of the possibility of overflow in your program, especially when dealing with large numbers.
