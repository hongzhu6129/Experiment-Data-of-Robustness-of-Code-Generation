    - `(m|rap)`: Matches either "m" or "rap".
- **Handles Null/Empty:** Includes checks for `null` or empty input strings to prevent `NullPointerException`.
- **Trimmed Input:** Uses `text.trim()` to handle leading/trailing whitespace, avoiding incorrect counts for spaces.
- **Clearer Variable Names:** Uses more descriptive variable names (e.g., `text`, `count`).
