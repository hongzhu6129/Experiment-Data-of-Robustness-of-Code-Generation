```java
public class PrintChars {

    public static void main(String[] args) {
        // Example usage:
        print_Chars('a', 'f', 3);  // Prints a, d
        print_Chars('x', 'z', 1);  // Prints x, y, z 
        print_Chars('Z', 'A', 2);  // Prints Z, X, V... (backwards)
        print_Chars('m', 'j', 2); // Prints m, k, i... (backwards)
    }

    public static void print_Chars(char ch1, char ch2, int n) {
        if (ch1 < ch2) {
            for (char ch = ch1; ch <= ch2; ch += n) {
                System.out.print(ch + ", ");
            }
        } else {
            for (char ch = ch1; ch >= ch2; ch -= n) {
                System.out.print(ch + ", ");
            }
        }
        System.out.println(); // Add a newline for cleaner output
    }
}
```


**Explanation and Improvements:**

1. **Handles both increasing and decreasing ranges:**  The code now correctly handles cases where `ch1` is greater than `ch2`, printing characters in descending order.

2. **Clearer Output:**  Added `System.out.println()` at the end of the `print_Chars` method to separate the output of different calls, making it easier to read.

3. **Example Usages in `main`:**  Demonstrates how to call the `print_Chars` method with various inputs, including forward and backward printing.

4. **Uses `+=` and `-=`:**  The code uses `ch += n` and `ch -= n` for incrementing/decrementing the character within the loop, which is more concise than `ch = (char)(ch + n)` .
