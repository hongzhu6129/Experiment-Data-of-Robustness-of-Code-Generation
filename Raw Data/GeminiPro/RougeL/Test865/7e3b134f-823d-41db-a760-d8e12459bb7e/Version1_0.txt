```java
public class DisplayFiveMessages {

    public static void main(String[] args) {
        System.out.println("I love Java");
        System.out.println("I love Java");
        System.out.println("I love Java");
        System.out.println("I love Java");
        System.out.println("I love Java"); 
    }
}
```

While a loop would be more concise for a larger number of repetitions,  following the prompt's instruction to *only* use the `main` method means we simply print the message five times explicitly.
