```java
import java.util.Arrays;

public class RemoveDuplicates {

    public static void main(String[] args) {
        String str = "programming";
        System.out.println(removeDuplicates(str)); // "progaming"

        str = "hello";
        System.out.println(removeDuplicates(str)); // "helo"

        str = "aaabbbccc";
        System.out.println(removeDuplicates(str)); // "abc"

        str = "";
        System.out.println(removeDuplicates(str)); // ""

        str = "  ";
        System.out.println(removeDuplicates(str)); // " "
    }

    public static String removeDuplicates(String str) {
        if (str == null || str.length() == 0) {
            return str;
        }

        char[] arr = str.toCharArray();
        int writeIndex = 0;

        for (int i = 0; i < arr.length; i++) {
            if (arr[i] != arr[writeIndex]) {
                arr[writeIndex + 1] = arr[i];
                writeIndex++;
            }
        }

        return new String(Arrays.copyOf(arr, writeIndex + 1));
    }
}
```

**Test Cases:**

* **Normal case:**
    * Input: "programming"
    * Output: "progaming"
* **Case with multiple consecutive duplicates:**
    * Input: "aaabbbccc"
    * Output: "abc"
* **Case with empty string:**
    * Input: ""
    * Output: ""
* **Case with only spaces:**
    * Input: "  "
    * Output: " "

**Corner Cases:**

* **Null input:**
    * Input: null
    * Output: null (or any other appropriate value)
