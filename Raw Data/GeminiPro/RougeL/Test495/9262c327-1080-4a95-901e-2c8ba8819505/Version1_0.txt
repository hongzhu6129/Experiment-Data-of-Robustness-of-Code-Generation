```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter an integer.");
        }
    }


    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("[" + i + "," + j + "] "); // Prints the matrix elements in [row,col] format
            }
            System.out.println();  // New line after each row
        }
    }
}
```


**Explanation and How to Run:**

1. **Compilation:** Save the code as `MatrixPrinter.java` and compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java
   ```

2. **Execution:** Run the compiled code, providing the matrix size `n` as a command-line argument:
   ```bash
   java MatrixPrinter 5  // This will print a 5x5 matrix
   ```

**Key improvements in this version:**

* **Error Handling:**  The `main` method now includes error handling for invalid input:
    * It checks if exactly one command-line argument is provided.
    * It uses a `try-catch` block to handle `NumberFormatException` in case the input is not a valid integer.
    * It checks if the input `n` is positive.
* **Clearer Output:** The `printMatrix` method now prints the matrix elements in a more readable format: `[row,col]`, making it easy to see the row and column indices.
* **Adherence to prompt:** The code strictly adheres to the requested method signatures: `public static void main(String[])` and `public static void printMatrix(int)`.


**Example Output (for `n=3`):**
