```java
import java.io.File;

public class FileSize {

    public static void main(String[] args) {
        String[] filePaths = { "/home/students/abc.txt", "/home/students/test.txt" }; // Replace with your file paths

        for (String filePath : filePaths) {
            File file = new File(filePath);

            if (file.exists()) {
                long fileSizeInBytes = file.length();
                System.out.println(filePath + ": " + fileSizeInBytes + " bytes");
            } else {
                System.out.println(filePath + ": File not found.");
            }
        }
    }
}
```

```java
import java.io.File;

public class FileSize {

    public static void main(String[] args) {
        if (args.length == 0) {
            System.out.println("Please provide file paths as command-line arguments.");
            return;
        }


        for (String filePath : args) {
            File file = new File(filePath);

            if (file.exists()) {
                long fileSizeInBytes = file.length();
                System.out.println(filePath + ": " + fileSizeInBytes + " bytes");
            } else {
                System.out.println(filePath + ": File not found.");
            }
        }
    }
}
```

1. `javac FileSize.java`
2. `java FileSize /path/to/file1.txt /path/to/file2.txt etc.`  (Replace with your actual file paths)


This command-line argument version lets the user specify the file paths when running the program. This addresses the limitations of using `Scanner` within the `main` method's specified signature.
