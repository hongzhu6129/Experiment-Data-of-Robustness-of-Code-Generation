```java
public class Faces {

    public static void printFace(int faceType) {
        switch (faceType) {
            case 1: // Smiling face
                System.out.println("  _,-._");
                System.out.println(" / \_/ \\");
                System.out.println("| ( o o ) |");
                System.out.println(" \\  ^  /");
                System.out.println("  `--' ");
                break;
            case 2: // Frowning face
                System.out.println("  _,-._");
                System.out.println(" / \_/ \\");
                System.out.println("| ( o o ) |");
                System.out.println(" \\  v  /");
                System.out.println("  `--' ");
                break;
            case 3: // Surprised face
                System.out.println("  _,-._");
                System.out.println(" / \_/ \\");
                System.out.println("| ( o o ) |");
                System.out.println(" \\  -  /");
                System.out.println("  `--' ");

                break;
            default: // Default face (Neutral)
                System.out.println("  _,-._");
                System.out.println(" / \_/ \\");
                System.out.println("| ( - - ) |");
                System.out.println(" \\  -  /");
                System.out.println("  `--' ");
                break;

        }
    }


    public static void main(String[] args) {
        // Test cases
        System.out.println("Smiling Face:");
        printFace(1);
        System.out.println("\nFrowning Face:");
        printFace(2);
        System.out.println("\nSurprised Face:");
        printFace(3);
        System.out.println("\nDefault/Neutral Face:");
        printFace(0); // Testing the default case
        System.out.println("\nInvalid Input:");
        printFace(4); // Testing an invalid input (should default)
        System.out.println("\nAnother Invalid Input:");
        printFace(-1);  // Testing another invalid input (should default)

    }
}

```



**Explanation and Test Cases:**

1. **`printFace(int faceType)` method:** This method takes an integer `faceType` as input to determine which face to print. A `switch` statement handles different face types.


2. **Test Cases in `main`:**
   - **Valid Inputs:**  Tests face types 1 (smiling), 2 (frowning), and 3 (surprised).
   - **Default Case:** Tests with `faceType = 0` and other invalid numbers like `4` and `-1` to ensure the default (neutral) face is printed when an invalid `faceType` is provided. This covers corner cases where the user might input an unexpected value.


**How to Run:**

1. Save the code as `Faces.java`.
2. Compile: `javac Faces.java`
3. Run: `java Faces`


**Output:**  You'll see the different faces printed to the console based on the test cases.  The default face will be printed for invalid inputs.
