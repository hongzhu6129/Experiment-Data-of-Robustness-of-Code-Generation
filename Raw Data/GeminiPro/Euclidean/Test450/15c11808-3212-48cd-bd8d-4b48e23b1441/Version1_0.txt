```java
import java.time.DayOfWeek;
import java.time.LocalDate;
import java.time.format.DateTimeFormatter;
import java.time.format.DateTimeParseException;
import java.util.Scanner;

public class GetDayOfWeek {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter a date in yyyy-MM-dd format (e.g., 2024-03-15):");

        String dateString = scanner.nextLine();

        try {
            DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
            LocalDate date = LocalDate.parse(dateString, formatter);
            DayOfWeek dayOfWeek = date.getDayOfWeek();
            System.out.println("The day of the week for " + dateString + " is: " + dayOfWeek);
        } catch (DateTimeParseException e) {
            System.out.println("Invalid date format. Please use yyyy-MM-dd.");
        }
    }
}
```


Explanation:

1. **Import necessary classes:**
   - `java.time.DayOfWeek`: To get the day of the week.
   - `java.time.LocalDate`: To represent a date.
   - `java.time.format.DateTimeFormatter`: To parse the date string.
   - `java.time.format.DateTimeParseException`: To handle invalid date formats.
   - `java.util.Scanner`: To get user input.

2. **Get user input:**
   - Prompts the user to enter a date in `yyyy-MM-dd` format.
   - Reads the input using `scanner.nextLine()`.

3. **Parse the date string:**
   - Creates a `DateTimeFormatter` with the specified pattern.
   - Uses `LocalDate.parse()` to convert the string into a `LocalDate` object.  The `try-catch` block handles potential `DateTimeParseException` if the input is not in the correct format.

4. **Get the day of the week:**
   - Calls `date.getDayOfWeek()` to get the `DayOfWeek` enum representing the day.

5. **Print the result:**
   - Prints the day of the week to the console.

6. **Handle invalid input:**
   - The `try-catch` block catches `DateTimeParseException` and prints an error message if the date format is incorrect.


Example Usage:

```
Enter a date in yyyy-MM-dd format (e.g., 2024-03-15):

invalid-date
Invalid date format. Please use yyyy-MM-dd.
```
