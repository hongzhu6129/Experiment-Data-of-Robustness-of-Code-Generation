```java
import java.util.Arrays;

public class CreateCharacterArray {

    public static void main(String[] args) {
        String str = "Hello World";

        // Create a character array from the string
        char[] charArray = str.toCharArray();

        // Print the character array
        System.out.println(Arrays.toString(charArray));
    }
}
```

**Test cases:**

* **Corner cases:**
    * Empty string: ""
    * String with only spaces: " "
    * String with special characters: "!@#$%^&*()"
* **Normal cases:**
    * String with lowercase letters: "hello world"
    * String with uppercase letters: "HELLO WORLD"
    * String with mixed case letters: "Hello World"
    * String with numbers: "1234567890"
    * String with punctuation: ".,:;?!"
