```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Start with 1 as the first element is always unique.

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueCount] = nums[i]; // Overwrite duplicate with the unique element.
                uniqueCount++;
            }
        }

        return uniqueCount;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Updated length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: 2, [1, 2]


        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Updated length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: 5, [0, 1, 2, 3, 4]

        // Test case with empty array
        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Updated length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Output: 0, []



    }
}
```


Key improvements and explanations:

1. **Handles Empty Array:**  Added a check at the beginning of `removeDuplicates` to handle the case where the input array is empty. This prevents potential `ArrayIndexOutOfBoundsException`.

2. **Clearer Variable Names:**  `uniqueCount` is used instead of just `count` to clearly represent the count of unique elements.

3. **In-Place Modification:** The code modifies the original array `nums` in-place as requested by the problem (overwriting duplicates with unique elements).

4. **Return Value:** The function returns the updated length (number of unique elements).

5. **Comprehensive `main` Method:** The `main` method now includes test cases with different array inputs, including an empty array and more complex examples with repeating duplicates, demonstrating the correctness of the `removeDuplicates` function.

6. **Output Formatting:** The `main` method uses `Arrays.toString()` and `Arrays.copyOfRange()` to print only the relevant part of the modified array (up to the new length), making the output easier to understand.



This revised code is more robust, efficient, and adheres more closely to best practices. It is also more thoroughly tested.
