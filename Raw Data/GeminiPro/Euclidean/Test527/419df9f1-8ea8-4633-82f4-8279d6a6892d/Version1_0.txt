            if (a3 > n) break; // Optimization: No need to check further if a^3 exceeds n

            for (int b = a + 1; b <= n; b++) {  // Start b from a+1 to avoid duplicates and ensure distinct pairs
                int b3 = b * b * b;
                int sum = a3 + b3;
                if (sum > n) break; // Optimization: No need to check further if the sum exceeds n

                    if (c3 > sum) break; // Optimization

                        if (c3 + d3 > sum) break; // Optimization

1. **Distinct Pairs:** The code ensures that the pairs (a, b) and (c, d) are distinct by starting the inner loops from `a + 1` and `c + 1` respectively. This prevents solutions like `1^3 + 12^3 = 9^3 + 10^3` being reported as a solution.  It also avoids duplicate pairs like  `12^3 + 1^3` and `1^3 + 12^3`.

2. **Optimizations:** Several `break` statements are added to the loops to exit early when it's clear that no solution can be found. This significantly improves performance, especially for larger values of `n`.  We check if `a^3`, `b^3`, the `sum`, `c^3`, and `d^3` exceed `n` or the current `sum` to stop unnecessary calculations.
