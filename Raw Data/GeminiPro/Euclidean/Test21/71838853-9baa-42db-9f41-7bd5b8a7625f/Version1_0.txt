```java
import java.util.Scanner;

public class OctalToBinary {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter an octal number: ");
        String octal = scanner.nextLine();

        try {
            int decimal = 0;
            int power = 0;
            for (int i = octal.length() - 1; i >= 0; i--) {
                int digit = octal.charAt(i) - '0';
                if (digit < 0 || digit > 7) {
                    System.out.println("Invalid octal number.");
                    return;
                }
                decimal += digit * Math.pow(8, power++);
            }

            String binary = "";
            if (decimal == 0) {
                binary = "0"; // Handle the case of octal 0
            } else {
                while (decimal > 0) {
                    binary = (decimal % 2) + binary;
                    decimal /= 2;
                }
            }


            System.out.println("Binary equivalent: " + binary);

        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter a valid octal number.");
        }

    }
}
```

**Test Cases:**

1. **Valid Octal Number:**
   - Input: 123
   - Expected Output: 1010011

2. **Octal Number with Leading Zeroes:**
   - Input: 0075
   - Expected Output: 111101

3. **Octal Number with Zero:**
   - Input: 0
   - Expected Output: 0   (Important corner case)

4. **Invalid Octal Number (Contains digits other than 0-7):**
   - Input: 128 
   - Expected Output: Invalid octal number.

5. **Invalid Octal Number (Empty String):**
   - Input:  (just press enter)
   - Expected Output: Invalid input. Please enter a valid octal number. (Or a similar error message depending on how Scanner handles empty input).

6. **Invalid Octal Number (Non-numeric input):**
   - Input: abc
   - Expected Output: Invalid input. Please enter a valid octal number.

7. **Large Octal Number:**
    - Input: 7777777
    - Expected Output: 111111111111111111

8. **Single Digit Octal Number:**
    - Input: 5
    - Expected Output: 101


These test cases cover various scenarios, including valid octal inputs, invalid inputs (with characters outside the range 0-7, empty input, non-numeric input), and corner cases like zero and leading zeros.  Testing with these different inputs will help ensure the program is robust.
