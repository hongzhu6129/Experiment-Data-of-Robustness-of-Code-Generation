**Problem:**

Write a Java program that takes a string as input and returns the character with the highest score in the alphabet.

**Solution:**

```java
import java.util.Scanner;

public class HighestScoreCharacter {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Get the input string
        System.out.println("Enter a string: ");
        String str = sc.nextLine();

        // Find the highest score character
        int highestScore = 0;
        char highestScoreCharacter = ' ';
        for (int i = 0; i < str.length(); i++) {
            char c = str.charAt(i);
            int score = (int)c - 96;
            if (score > highestScore) {
                highestScore = score;
                highestScoreCharacter = c;
            }
        }

        // Print the highest score character
        System.out.println("The character with the highest score is: " + highestScoreCharacter);
    }
}
```

**Test Cases:**

**Input 1:** java
**Output:** a

**Input 2:** programming
**Output:** o

**Input 3:** the quick brown fox jumps over the lazy dog
**Output:** z

**Corner Cases:**

**Input:** ""
**Output:** (Empty string. No character with the highest score)

**Input:** "123"
**Output:** (No alphabetical characters. No character with the highest score)
