```java
import java.time.LocalDate;
import java.time.Period;
import java.time.format.DateTimeFormatter;
import java.time.format.DateTimeParseException;
import java.util.Scanner;

public class DateDifference {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter the first date (yyyy-MM-dd):");
        String date1String = scanner.nextLine();

        System.out.println("Enter the second date (yyyy-MM-dd):");
        String date2String = scanner.nextLine();

        try {
            DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
            LocalDate date1 = LocalDate.parse(date1String, formatter);
            LocalDate date2 = LocalDate.parse(date2String, formatter);

            Period period = Period.between(date1, date2);

            System.out.println("Difference:");
            System.out.println("Years: " + period.getYears());
            System.out.println("Months: " + period.getMonths());
           // System.out.println("Days: " + period.getDays()); // If you need days as well

        } catch (DateTimeParseException e) {
            System.out.println("Invalid date format. Please use yyyy-MM-dd.");
        }

        scanner.close();


    }
}
```


Key improvements and explanations:

1. **`java.time` API:**  Uses the modern `java.time` API (introduced in Java 8) for date/time handling. This is significantly better than the older `Date` and `Calendar` classes.

2. **Error Handling:** Includes a `try-catch` block to handle `DateTimeParseException`.  This is crucial.  If the user enters dates in the wrong format, the program will gracefully inform them instead of crashing.

3. **Clear Prompts:** The `System.out.println()` statements provide clear instructions to the user about the expected date format.

4. **`Period` for Difference:** Uses the `Period` class to calculate the difference between the dates, directly giving you years and months.

5. **Scanner Closed:** Closes the `Scanner` at the end to release resources.

6. **Example Usage and Output:** Included below to illustrate how to interact with the program.
