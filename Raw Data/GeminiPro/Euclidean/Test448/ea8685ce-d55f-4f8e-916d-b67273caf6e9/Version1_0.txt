```java
import java.util.Scanner;

public class DaysInMonth {

    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter the month (1-12): ");
        int month = input.nextInt();

        System.out.print("Enter the year: ");
        int year = input.nextInt();

        int days = 0;

        switch (month) {
            case 1: // January
            case 3: // March
            case 5: // May
            case 7: // July
            case 8: // August
            case 10: // October
            case 12: // December
                days = 31;
                break;
            case 4: // April
            case 6: // June
            case 9: // September
            case 11: // November
                days = 30;
                break;
            case 2: // February
                if (((year % 4 == 0) && !(year % 100 == 0)) || (year % 400 == 0)) {
                    days = 29;
                } else {
                    days = 28;
                }
                break;
            default:
                System.out.println("Invalid month.");
                return; // Exit the program if the month is invalid
        }

        System.out.println("Number of days in the specified month: " + days);
        input.close(); // Close the Scanner
    }
}
```


Key improvements in this version:

* **Input Validation:** The code now checks for invalid month input (outside the 1-12 range) and exits gracefully with an error message if an invalid month is entered.  This prevents unexpected behavior.
* **Resource Management:** The `Scanner` is closed using `input.close()` to release system resources. This is good practice, especially in larger programs.
* **Clearer Output:** The output message is more informative.
* **Standard Formatting:**  Improved indentation and code style for better readability.


This revised code provides a more robust and user-friendly solution for determining the number of days in a month.
