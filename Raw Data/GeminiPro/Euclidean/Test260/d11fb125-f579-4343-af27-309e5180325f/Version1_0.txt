```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        int uniqueIndex = 1; // Index for storing unique elements

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueIndex] = nums[i];
                uniqueIndex++;
            }
        }

        return uniqueIndex;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Updated length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1)));

        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Updated length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2)));
        
        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Updated length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Empty array handling

        int[] nums4 = {1, 2, 3, 4, 5}; // Already unique elements
        int len4 = removeDuplicates(nums4);
        System.out.println("Updated length: " + len4 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums4, 0, len4)));
    }
}
```


Explanation:

1. **`removeDuplicates(int[] nums)`:**
   - Handles the edge case of an empty input array.
   - `uniqueIndex` keeps track of where to place the next unique element. It starts at 1 because the first element is always unique initially.
   - Iterates through the array from the second element (`i = 1`).
   - If the current element (`nums[i]`) is different from the previous element (`nums[i-1]`), it's a unique element. So, we place it at `nums[uniqueIndex]` and increment `uniqueIndex`.
   - Finally, returns `uniqueIndex`, which is the new length of the array containing only unique elements.

2. **`main(String[] args)`:**
   - Provides example usage of the `removeDuplicates` function with various test cases, including an empty array and an array with already unique elements.
   - Prints the updated length and the modified array using `Arrays.copyOfRange` to display only the relevant part of the array (up to `len`).  This is important because the original array is modified in-place, but some elements after `len` might still contain old values that are not part of the unique portion of the array.


Key Improvements:

- **Handles empty array:** The code now correctly handles the case where the input array is empty.
- **Clearer variable naming:** Using `uniqueIndex` instead of just `i` for the tracking index improves readability.
- **Comprehensive test cases:** Demonstrates the function's behavior with different inputs, including edge cases.
- **Output clarity:** Uses `Arrays.copyOfRange` to show the updated array correctly after duplicate removal.
