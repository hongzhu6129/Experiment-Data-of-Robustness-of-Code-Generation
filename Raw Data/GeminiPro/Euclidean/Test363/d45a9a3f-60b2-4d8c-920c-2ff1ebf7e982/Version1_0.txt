```java
public class ReverseWordsInString {

    public void reverseEachWordInString(String str) {
        if (str == null || str.trim().isEmpty()) {
            System.out.println(""); // Handle empty or null input
            return;
        }

        String[] words = str.split("\\s+"); // Split the string into words
        StringBuilder reversedString = new StringBuilder();

        for (String word : words) {
            reversedString.append(new StringBuilder(word).reverse().toString()).append(" ");
        }

        System.out.println(reversedString.toString().trim()); // Print the reversed string
    }

    public static void main(String[] args) {
        ReverseWordsInString reverser = new ReverseWordsInString();

        reverser.reverseEachWordInString("This is a test string"); // Example usage 1
        reverser.reverseEachWordInString("Hello World");        // Example usage 2
        reverser.reverseEachWordInString(null);                // Example usage 3 (null input)
        reverser.reverseEachWordInString(" ");                 // Example usage 4 (empty string)
        reverser.reverseEachWordInString(" leading and trailing spaces "); //Example with spaces
    }
}
```


Explanation and Improvements:

1. **Handles Null and Empty Input:** The `reverseEachWordInString` method now checks for `null` or empty input strings and prints an empty string in those cases, preventing `NullPointerException` or unnecessary processing.

2. **Uses `split("\\s+")`:** This is a more robust way to split the string into words.  `\\s+` matches one or more whitespace characters, so it correctly handles multiple spaces between words.

3. **`StringBuilder` for Efficiency:** Using `StringBuilder` is generally more efficient than string concatenation with `+` within a loop.

4. **`trim()` for Clean Output:** The `trim()` method removes leading and trailing spaces from the final reversed string, providing cleaner output.

5. **Clearer Example Usage:** The `main` method demonstrates several different example usages of the `reverseEachWordInString` method, including handling null and empty input strings, as well as strings with leading and trailing spaces.

6. **Object-Oriented Approach:** The code uses a class (`ReverseWordsInString`) and a method (`reverseEachWordInString`), making it more organized and reusable.
