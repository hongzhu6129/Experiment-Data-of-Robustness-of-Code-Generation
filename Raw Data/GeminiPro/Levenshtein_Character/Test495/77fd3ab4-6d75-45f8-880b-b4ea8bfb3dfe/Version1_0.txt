```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter an integer.");
        }
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("[" + i + "," + j + "] "); // Print matrix elements
            }
            System.out.println(); // Newline after each row
        }
    }
}
```

**Explanation and How to Run:**

1. **Compilation:** Save the code as `MatrixPrinter.java` and compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java
   ```

2. **Execution:** Run the compiled code, providing the size of the matrix (n) as a command-line argument:
   ```bash
   java MatrixPrinter 5  // This will print a 5x5 matrix
   ```

**Code Breakdown:**

* **`main` Method:**
    * Takes the matrix size `n` as a command-line argument.
    * Handles invalid input (non-integer, negative numbers).
    * Calls the `printMatrix` method to generate and display the matrix.

* **`printMatrix` Method:**
    * Uses nested loops to iterate through rows and columns.
    * Prints the matrix elements in the format `[row,column]`.
    * Adds a newline after each row to format the output correctly.



**Example Output (for n=3):**

```
[0,0] [0,1] [0,2] 
[1,0] [1,1] [1,2] 
[2,0] [2,1] [2,2] 
```
