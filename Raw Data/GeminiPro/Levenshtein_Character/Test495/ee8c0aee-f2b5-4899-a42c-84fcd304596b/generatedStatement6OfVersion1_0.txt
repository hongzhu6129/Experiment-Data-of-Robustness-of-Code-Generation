

**How it Works:**

* **`main` method:**
    - Takes the size `n` as a command-line argument.
    - Uses a `try-catch` block to handle potential `NumberFormatException` if the user inputs a non-integer value.
    - Calls the `printMatrix` method to generate and print the matrix.

* **`printMatrix` method:**
    - Uses nested loops to iterate through the rows and columns of the `n x n` matrix.
    - The key logic is in the `if (i + j == n - 1)` condition.  This checks if the current cell is on the *inverse* diagonal (from top-right to bottom-left).  If it is, it prints "1".  Otherwise, it prints "0".
