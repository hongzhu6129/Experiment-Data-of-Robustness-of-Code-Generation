```java
public class DisariumAndUnhappyNumberChecker {

    public static void main(String[] args) {
        int num = 135; // Example input, you can change this

        if (isDisarium(num)) {
            System.out.println(num + " is a Disarium number.");
        } else {
            System.out.println(num + " is not a Disarium number.");
        }

        if (isUnhappy(num)) {
            System.out.println(num + " is an Unhappy number.");
        } else {
            System.out.println(num + " is a Happy number (not Unhappy).");
        }
    }

    public static boolean isDisarium(int num) {
        String numStr = String.valueOf(num);
        int sum = 0;
        for (int i = 0; i < numStr.length(); i++) {
            int digit = Character.getNumericValue(numStr.charAt(i));
            sum += Math.pow(digit, i + 1);
        }
        return sum == num;
    }

    public static boolean isUnhappy(int num) {
        java.util.Set<Integer> seen = new java.util.HashSet<>();
        while (num != 1 && !seen.contains(num)) {
            seen.add(num);
            int sum = 0;
            while (num > 0) {
                int digit = num % 10;
                sum += digit * digit;
                num /= 10;
            }
            num = sum;
        }
        return num != 1;
    }
}
```

1. **Clearer Structure:** The code is organized into separate functions (`isDisarium` and `isUnhappy`) for better readability and maintainability.

2. **Correct Disarium Calculation:** The `isDisarium` function now correctly calculates the sum of digits raised to their respective positions (starting from 1).

3. **Efficient Unhappy Number Check:** The `isUnhappy` function uses a `HashSet` to detect cycles and efficiently determine if a number is unhappy.  It avoids infinite loops.

4. **Informative Output:** The `main` function prints clear messages indicating whether the input number is Disarium and/or Unhappy.

5. **Example Usage:**  The `main` method demonstrates how to use the functions with a sample input.  You can easily change the `num` variable to test other numbers.
