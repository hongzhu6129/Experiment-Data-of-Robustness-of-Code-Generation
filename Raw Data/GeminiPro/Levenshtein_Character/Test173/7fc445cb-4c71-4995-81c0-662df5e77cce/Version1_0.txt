```java
import java.util.HashMap;
import java.util.Map;

public class FrequentCharacters {
    public static void main(String[] args) {
        String str = "Hello World";
        Map<Character, Integer> charCountMap = new HashMap<>();

        // Count the occurrences of each character in the string
        for (char c : str.toCharArray()) {
            if (charCountMap.containsKey(c)) {
                charCountMap.put(c, charCountMap.get(c) + 1);
            } else {
                charCountMap.put(c, 1);
            }
        }

        // Find the character with the highest count
        Character mostFrequentChar = null;
        int maxCount = 0;
        for (Map.Entry<Character, Integer> entry : charCountMap.entrySet()) {
            if (entry.getValue() > maxCount) {
                mostFrequentChar = entry.getKey();
                maxCount = entry.getValue();
            }
        }

        // Print the most frequent character and its count
        System.out.println("Most frequent character: " + mostFrequentChar);
        System.out.println("Count: " + maxCount);
    }
}
```

**Test Cases:**

**Corner Cases:**

* **Empty string:** The program should handle an empty string input and print an appropriate message.
* **Single character string:** The program should handle a string with only one character and print that character as the most frequent.

**Regular Test Cases:**

* **String with multiple occurrences of the same character:** The program should find the character with the highest number of occurrences.
* **String with equal occurrences of multiple characters:** The program should print all the characters with the highest number of occurrences.
* **String with special characters or spaces:** The program should ignore special characters and spaces when counting occurrences.
* **String with uppercase and lowercase characters:** The program should treat uppercase and lowercase characters as the same.
