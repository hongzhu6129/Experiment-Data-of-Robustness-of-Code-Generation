```java
public class RemoveElement {

    public static int removeElement(int[] nums, int val) {
        int k = 0; // Number of elements not equal to val

        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != val) {
                nums[k] = nums[i];
                k++;
            }
        }
        return k;
    }

    public static void main(String[] args) {
        int[] nums = {1, 4, 6, 7, 6, 2};
        int val = 6;

        System.out.print("Original array: [");
        for (int i = 0; i < nums.length; i++) {
            System.out.print(nums[i]);
            if (i < nums.length - 1) {
                System.out.print(", ");
            }
        }
        System.out.println("]");

        int newLength = removeElement(nums, val);

        System.out.println("The length of the new array is: " + newLength);


        System.out.print("New array: ["); // Print the modified array (up to the new length)
        for (int i = 0; i < newLength; i++) {
            System.out.print(nums[i]);
            if (i < newLength - 1) {
                System.out.print(", ");
            }
        }
        System.out.println("]");
    }
}
```


**Explanation:**

1. **`removeElement(int[] nums, int val)`:**
   - This method iterates through the input array `nums`.
   - `k` keeps track of the index where the next element *not equal* to `val` should be placed.
   - If `nums[i]` is not equal to `val`, we copy it to `nums[k]` and increment `k`.  This effectively overwrites the elements equal to `val` with the elements that should be kept.
   - Finally, `k` represents the number of elements in the modified array (excluding the removed elements), so we return it.

2. **`main(String[] args)`:**
   - Creates a sample array `nums` and the value to remove `val`.
   - Prints the original array.
   - Calls `removeElement` to modify the array and get the new length.
   - Prints the new length.
   - Prints the updated array (only up to the `newLength` to avoid showing the unchanged elements beyond the "new" end of the array).


**Key Improvement:**

This version is more efficient because it does the removal in-place (directly modifying the input array) rather than creating a new array.  It's also clearer and easier to understand.
