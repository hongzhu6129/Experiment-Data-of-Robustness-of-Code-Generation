


**How to Run Test Cases (Manually):**

1. Compile the Java code:  `javac MultiplicationTable.java`
2. Run the program multiple times, each time providing a different test case input: `java MultiplicationTable`  (Then enter the input value when prompted).
3. Compare the program's output with the expected output for each test case.
