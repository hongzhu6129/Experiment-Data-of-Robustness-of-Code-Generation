```java
import java.util.Scanner;

public class ScalarPlace {

    public static void printScalarPlaces(int number) {
        if (number == 0) {
            System.out.println("Scalar places of 0: 0");
            return;
        }

        int temp = Math.abs(number); // Handle negative numbers
        int place = 1;

        System.out.print("Scalar places of " + number + ": ");

        while (temp > 0 && place <= 10) {  // Limit to ten places
            int digit = temp % 10;
            System.out.print(digit * place + " ");
            temp /= 10;
            place *= 10;
        }
        System.out.println(); // Newline for clarity
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter a number: ");
        int num = scanner.nextInt();

        printScalarPlaces(num);
        scanner.close(); 
    }
}
```

1. **Positive Number:**

2. **Negative Number:**

3. **Zero:**

4. **Number with more than ten digits:**
   - Input: 123456789012345  (15 digits)
   - Expected Output: `Scalar places of 123456789012345: 5 40 300 2000 10000 0 9000000 80000000 700000000 6000000000 ` (only up to 10 places)

5. **Number with leading zeros (will be treated as the actual value):**
   - Input: 000123
   - Expected Output: `Scalar places of 123: 3 20 100 ` (Java's `nextInt()` handles leading zeros automatically)

6. **Single Digit Number:**
    - Input: 7
    - Expected Output: `Scalar places of 7: 7 `

7. **Large Negative Number:**
    - Input: -2147483648 (Integer.MIN_VALUE)
    - Expected Output: `Scalar places of -2147483648: 8 40 600 3000 80000 400000 7000000 40000000 100000000 2000000000 `



These test cases cover various scenarios, including positive and negative numbers, zero, numbers with more than ten digits, leading zeros, and single-digit numbers. This helps ensure that the program functions correctly under different input conditions, including edge cases.  Testing with `Integer.MIN_VALUE` and `Integer.MAX_VALUE` is also good practice when working with integers.
