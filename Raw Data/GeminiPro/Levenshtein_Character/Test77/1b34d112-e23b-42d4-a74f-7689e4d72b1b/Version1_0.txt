```java
import java.util.Scanner;

public class EndsWith {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Get the input string and ending substring
        System.out.println("Enter a string: ");
        String str = scanner.nextLine();

        System.out.println("Enter an ending substring: ");
        String ending = scanner.nextLine();

        // Check if the string ends with the ending substring
        boolean result = str.endsWith(ending);

        // Print the result
        if (result) {
            System.out.println("The string ends with the ending substring.");
        } else {
            System.out.println("The string does not end with the ending substring.");
        }
    }
}
```

**Test cases:**

* **Normal case:** Input string: "Hello", ending substring: "o" - expected output: "The string ends with the ending substring."
* **Case-insensitive case:** Input string: "Hello", ending substring: "O" - expected output: "The string ends with the ending substring."
* **Whitespace case:** Input string: "Hello ", ending substring: " " - expected output: "The string ends with the ending substring."
* **Corner case: empty string:** Input string: "", ending substring: "o" - expected output: "The string does not end with the ending substring."
* **Corner case: null string:** Input string: null, ending substring: "o" - expected output: "The string does not end with the ending substring."
* **Corner case: null ending substring:** Input string: "Hello", ending substring: null - expected output: "The string does not end with the ending substring."
* **Corner case: ending substring longer than string:** Input string: "Hello", ending substring: "Hello World" - expected output: "The string does not end with the ending substring."
