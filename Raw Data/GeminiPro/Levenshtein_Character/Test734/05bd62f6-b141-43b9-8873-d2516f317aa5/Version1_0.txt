```java
public class SaddlebackSearch {

    public static void main(String[] args) {
        int[][] arr = {
                {1, 4, 7, 11, 15},
                {2, 5, 8, 12, 19},
                {3, 6, 9, 16, 22},
                {10, 13, 14, 17, 24},
                {18, 21, 23, 26, 30}
        };

        int target = 21;
        int m = arr.length;
        int n = arr[0].length;

        int[] result = Saddleback(arr, target, m, n);

        if (result[0] != -1) {
            System.out.println("Element " + target + " found at row: " + result[0] + ", column: " + result[1]);
        } else {
            System.out.println("Element " + target + " not found in the array.");
        }

        target = 50;  // Test with a value not in the array
        result = Saddleback(arr, target, m, n);
        if (result[0] != -1) {
            System.out.println("Element " + target + " found at row: " + result[0] + ", column: " + result[1]);
        } else {
            System.out.println("Element " + target + " not found in the array.");
        }

    }



    private static int[] Saddleback(int[][] arr, int target, int m, int n) {
        int row = 0;
        int col = n - 1;

        while (row < m && col >= 0) {
            if (arr[row][col] == target) {
                return new int[]{row, col}; // Element found
            } else if (arr[row][col] < target) {
                row++; // Move down to the next row
            } else {
                col--; // Move left to the previous column
            }
        }

        return new int[]{-1, -1}; // Element not found
    }
}
```


Key improvements and explanations:

1. **Clearer Variable Names:** Using `m` and `n` for dimensions improves readability.

2. **Returns an Array:** The `Saddleback` method now returns an `int[]` to store both the row and column index. This is much cleaner than using `System.out.println` inside the search function.  If the element isn't found, it returns `{-1, -1}`.

3. **Error Handling in `main`:**  The `main` method now checks the returned array from `Saddleback` to print an appropriate message if the element is not found.

4. **Example Usage and Testing:** The `main` method now demonstrates how to use the `Saddleback` function and includes a test case with a target value that's *not* in the array to show the "not found" handling.


This revised version is more robust, easier to understand, and follows best practices for function design and error handling.
