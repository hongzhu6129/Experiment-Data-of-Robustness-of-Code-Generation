```java
import java.util.Arrays;

public class CreateStringObjectAndCharacterArray {

    public static void main(String[] args) {
        // Create a String object
        String str = "Hello World";

        // Create a characters array
        char[] chars = {'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd'};

        // Print the String object
        System.out.println(str);

        // Print the characters array
        System.out.println(Arrays.toString(chars));
    }
}
```

**Test Cases:**

**Normal case:**

* Input: `String str = "Hello World"`
* Output: `Hello World`

**Empty string case:**

* Input: `String str = ""`
* Output: `""`

**Null string case:**

* Input: `String str = null`
* Output: `null`

**Corner case 1:**

* Input: `char[] chars = null`
* Output: `null`

**Corner case 2:**

* Input: `char[] chars = {}`
* Output: `[]`
