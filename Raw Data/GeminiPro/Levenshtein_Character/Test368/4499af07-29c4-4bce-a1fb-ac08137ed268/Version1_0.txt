```java
import java.util.Arrays;

public class StringDuplicates {

    static void CountCharacters(String str, int[] charCounts) {
        str = str.toLowerCase(); // Case-insensitive counting
        for (char c : str.toCharArray()) {
            if (Character.isLetter(c)) { // Consider only letters
                charCounts[c - 'a']++;
            }
        }
    }

    static void showDuplicates(String str) {
        int[] charCounts = new int[26]; // Assuming only lowercase English letters

        CountCharacters(str, charCounts);

        System.out.print("Duplicate characters in \"" + str + "\": ");
        boolean foundDuplicates = false;

        for (int i = 0; i < 26; i++) {
            if (charCounts[i] > 1) {
                System.out.print((char) ('a' + i) + " ");  // Print the duplicate character
                foundDuplicates = true;
            }
        }

        if (!foundDuplicates) {
            System.out.print("No duplicates found.");
        }
        System.out.println(); // Add a newline for cleaner output
    }


    public static void main(String[] args) {
        showDuplicates("Programming");
        showDuplicates("Java");
        showDuplicates("ABBA"); // Example with uppercase
        showDuplicates("123"); // Example with no duplicates
        showDuplicates("better butter"); // Example with spaces and mixed case


    }
}
```


Key improvements and explanations:

1. **Case-Insensitive Counting:**  The `CountCharacters` method now converts the input string to lowercase using `str.toLowerCase()` to ensure that 'P' and 'p' are considered the same character.
2. **Clearer Output:** Added a newline character (`System.out.println()`) after printing the duplicates to separate the output for different input strings.  Also added a "No duplicates found" message for clarity.
3. **Handles Non-Letters:** Includes `Character.isLetter(c)` in `CountCharacters` to only count letters.  This avoids potential `ArrayIndexOutOfBoundsException` errors if the input string has characters outside the 'a'-'z' range.
4. **Example Usage:** The `main` method demonstrates the functionality with various examples, including strings with uppercase characters, no duplicates, and strings containing spaces.


This improved version addresses potential issues, handles various input scenarios more robustly, and produces clearer output.
