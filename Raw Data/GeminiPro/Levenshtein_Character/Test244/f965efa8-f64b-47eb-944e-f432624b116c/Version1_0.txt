```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueIndex = 0; 

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[uniqueIndex]) {
                uniqueIndex++;
                nums[uniqueIndex] = nums[i];
            }
        }

        return uniqueIndex + 1;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Updated Length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: 2, [1, 2]


        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Updated Length: " + len2+ ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: 5, [0, 1, 2, 3, 4]

         int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Updated Length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Output: 0, []

        int[] nums4 = null;
        int len4 = removeDuplicates(nums4);
        System.out.println("Updated Length: " + len4+ ", Array: null"); // Output: 0, null

        int[] nums5 = {1};
         int len5 = removeDuplicates(nums5);
        System.out.println("Updated Length: " + len5 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums5, 0, len5))); // Output: 1, [1]
    }
}
```


Explanation:

1. **`removeDuplicates(int[] nums)`:**
   - Handles `null` or empty input arrays gracefully.
   - Uses `uniqueIndex` to track the index where the next unique element should be placed.
   - Iterates through the array starting from the second element (`i = 1`).
   - If the current element `nums[i]` is different from the element at `nums[uniqueIndex]`, it means we've found a new unique element.  We increment `uniqueIndex` and copy the new unique element to `nums[uniqueIndex]`.
   - Finally, returns `uniqueIndex + 1`, which represents the length of the updated array (containing only unique elements).

2. **`main(String[] args)`:**
   - Creates example arrays to test the `removeDuplicates` function.
   - Prints the updated length and the updated array (using `Arrays.copyOfRange` to only print the relevant part of the array after removing duplicates.  The original array is modified in place).

- **Clearer Variable Names:**  `uniqueIndex` is more descriptive than just `i` or `j`.
- **Handles Edge Cases:** The code now handles `null` and empty input arrays.
- **In-Place Modification:** The function modifies the original array in place as requested, which is generally more efficient than creating a new array.
- **Comprehensive Testing:** The `main` method includes tests with various scenarios (duplicates, empty array, null array, single-element array).
