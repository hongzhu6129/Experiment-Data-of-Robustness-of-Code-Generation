**How it Works (Two Pointers):**

The two-pointer approach efficiently partitions the array in-place (or into a new array of the same size without needing extra lists):

1. **Initialization:** `evenIndex` is set to 0, `oddIndex` is set to `arr.length - 1`.

2. **Iteration:** The code iterates through the input array `arr`.

3. **Even Number:** If a number is even, it's placed at `partitionedArr[evenIndex]` and `evenIndex` is incremented.

4. **Odd Number:** If a number is odd, it's placed at `partitionedArr[oddIndex]` and `oddIndex` is decremented.


This process continues until all elements are processed, resulting in a partitioned array with even numbers at the beginning and odd numbers at the end.
