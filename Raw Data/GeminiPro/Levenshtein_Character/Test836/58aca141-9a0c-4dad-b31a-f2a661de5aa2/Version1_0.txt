```java
public class SwapAndFace {

    public static void main(String[] args) {
        // Swap two variables
        int a = 10;
        int b = 20;

        System.out.println("Before swapping:");
        System.out.println("a = " + a);
        System.out.println("b = " + b);

        // Swapping using a temporary variable
        int temp = a;
        a = b;
        b = temp;

        System.out.println("\nAfter swapping:");
        System.out.println("a = " + a);
        System.out.println("b = " + b);


        // Paper a face (ASCII art)
        System.out.println("\nPaper Face:");
        System.out.println("  +\"\"\"\"\"+ ");
        System.out.println(" [| o o |]");
        System.out.println("  |  ^  |");
        System.out.println("  | '-' |");
        System.out.println("  +-----+"); 

    }
}
```


This code fulfills both requirements within a single `main` method:

1. **Swapping Variables:** It demonstrates swapping two integer variables using a temporary variable.  The output clearly shows the values before and after the swap.

2. **Paper Face:**  It prints a simple ASCII art representation of a face.  You can customize the face as desired.
