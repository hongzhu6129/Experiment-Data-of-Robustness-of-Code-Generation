```java
public class SquareRoot {

    public static void main(String[] args) {
        int num = 25; // You can change this to any positive integer
        int sqrtResult = sqrt(num);
        System.out.println("Square root of " + num + " is: " + sqrtResult);
    }

    private static int sqrt(int num) {
        if (num < 0) {
            throw new IllegalArgumentException("Input must be a non-negative integer.");
        }
        if (num == 0) {
            return 0;
        }

        int left = 1;
        int right = num;
        int result = 0;

        while (left <= right) {
            int mid = left + (right - left) / 2; // Avoid potential overflow
            int square = mid * mid;

            if (square == num) {
                return mid; // Perfect square
            } else if (square < num) {
                result = mid;  // Store the closest integer square root so far
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return result;

    }
}
```


This code:

1. **Handles Input:** It takes an integer `num` as input (currently set to 25, but you can modify it).
2. **Handles Errors:** It checks for negative input and throws an `IllegalArgumentException` if encountered.
3. **Handles 0:**  It correctly handles the case where the input is 0.
4. **Integer Square Root:**  It uses an efficient binary search approach to find the integer square root. The result will be the largest integer whose square is less than or equal to the input number. This adheres to the typical definition of the integer square root function.
5. **Avoids Overflow:**  The calculation of `mid` uses `left + (right - left) / 2` to avoid potential integer overflow that could happen with `(left + right) / 2`.
6. **Clear Output:** It prints the result to the console in the desired format.
