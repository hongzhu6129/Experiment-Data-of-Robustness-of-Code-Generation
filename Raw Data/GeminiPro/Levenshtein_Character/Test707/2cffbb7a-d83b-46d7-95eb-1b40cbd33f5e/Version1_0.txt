```java
public class ExcelColumnName {

    public static String Column(int n) {
        StringBuilder columnName = new StringBuilder();

        while (n > 0) {
            int rem = n % 26;

            if (rem == 0) {
                columnName.append("Z");
                n = (n / 26) - 1;
            } else {
                columnName.append((char)('A' + rem - 1));
                n = n / 26;
            }
        }

        return columnName.reverse().toString();
    }

    public static void main(String[] args) {
        System.out.println("Column 1: " + Column(1));      // Output: A
        System.out.println("Column 28: " + Column(28));    // Output: AB
        System.out.println("Column 701: " + Column(701));  // Output: ZY
        System.out.println("Column 702: " + Column(702));  // Output: ZZ
        System.out.println("Column 703: " + Column(703));  // Output: AAA
        System.out.println("Column 26: " + Column(26));    // Output: Z 
        System.out.println("Column 52: " + Column(52));    // Output: AZ

    }
}
```


**Explanation:**

1. **`Column(int n)` Method:**
   - Takes an integer `n` (the column number) as input.
   - Uses a `StringBuilder` to efficiently build the column name string.
   - The core logic is in the `while` loop:
     - It calculates the remainder (`rem`) when `n` is divided by 26 (since there are 26 letters).
     - If `rem` is 0, it means we've reached a 'Z'.  We append 'Z', and since 'Z' represents a full cycle of 26, we decrement `n` by 1 after dividing by 26 to account for the 'Z'.
     - Otherwise, we append the corresponding character by adding `rem - 1` to the ASCII value of 'A'.  Then, we divide `n` by 26 to continue to the next character.
   - Finally, it reverses the `StringBuilder` (because we build the string from right to left) and converts it to a `String`.
