```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Start with 1 as the first element is always unique.
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueCount] = nums[i];
                uniqueCount++;
            }
        }

        return uniqueCount;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Updated length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOf(nums1, len1)));  // Output: 2, [1, 2]

        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Updated length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOf(nums2, len2))); // Output: 5, [0, 1, 2, 3, 4]

        int[] nums3 = {}; // Empty array
        int len3 = removeDuplicates(nums3);
        System.out.println("Updated length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOf(nums3, len3))); // Output: 0, []
        
        int[] nums4 = null; // Null array
        int len4 = removeDuplicates(nums4);
        System.out.println("Updated length: " + len4 + ", Array: " + (nums4 == null ? "null" : Arrays.toString(Arrays.copyOf(nums4, len4)))); // Output: 0, null
                
        int[] nums5 = {1, 2, 3, 4, 5};  // Already unique
        int len5 = removeDuplicates(nums5);
        System.out.println("Updated length: " + len5+ ", Array: " + Arrays.toString(Arrays.copyOf(nums5, len5))); // Output: 5, [1, 2, 3, 4, 5]

    }
}
```



Key improvements in this version:

* **Handles edge cases:**  Correctly handles empty and null arrays.
* **Clearer variable names:** `uniqueCount` is more descriptive than just `count`.
* **Improved output in `main`:** Uses `Arrays.copyOf` to show only the relevant part of the modified array, making the output easier to understand and preventing displaying garbage values from the unmodified portion of the array.
* **More test cases:** The `main` method includes various test cases, including empty, null, and already-unique arrays, to demonstrate the robustness of the `removeDuplicates` method.
