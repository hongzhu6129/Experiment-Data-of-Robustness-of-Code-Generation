```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Initialize with 1 as the first element is always unique

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueCount] = nums[i];
                uniqueCount++;
            }
        }

        return uniqueCount;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Updated length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1)));

        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Updated length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2)));


        // Test with empty array
        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Updated length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Empty array

         // Test with single element array
        int[] nums4 = {5};
        int len4 = removeDuplicates(nums4);
        System.out.println("Updated length: " + len4 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums4, 0, len4))); // [5]


    }
}
```


Key improvements:

* **Handles empty arrays:**  Added a check at the beginning to return 0 if the input array is empty, preventing potential `ArrayIndexOutOfBoundsException`.
* **Clearer Variable Names:**  `uniqueCount` is more descriptive than just `count`.
* **In-place Modification and Return Value:**  The function modifies the input array `nums` in place to contain the unique elements at the beginning, and it returns the new length (`uniqueCount`), which represents the number of unique elements.
* **Comprehensive Test Cases:** The `main` method includes various test cases, including empty and single-element arrays, to demonstrate the correctness of the function.
* **Output Formatting:** The output now clearly shows both the updated length and the modified array containing only the unique elements.  The `Arrays.copyOfRange` method is used to prevent displaying the unmodified portion of the array.
