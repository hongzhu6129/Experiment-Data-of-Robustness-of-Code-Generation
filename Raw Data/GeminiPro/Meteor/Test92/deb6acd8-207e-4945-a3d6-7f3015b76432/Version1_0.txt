```java
public class CreateArrayAfter10 {

    public static void main(String[] args) {
        int[] originalArray = {1, 2, 3, 10, 11, 12, 13, 10, 14, 15};
        //int[] originalArray = {1, 2, 3}; // Test case: no 10 found
        //int[] originalArray = {1, 2, 3, 10}; // Test case: 10 is the last element

        int last10Index = -1;
        for (int i = 0; i < originalArray.length; i++) {
            if (originalArray[i] == 10) {
                last10Index = i;
            }
        }

        if (last10Index != -1 && last10Index < originalArray.length - 1) {
            int newArrayLength = originalArray.length - last10Index - 1;
            int[] newArray = new int[newArrayLength];
            for (int i = 0; i < newArrayLength; i++) {
                newArray[i] = originalArray[last10Index + 1 + i];
            }

            System.out.print("New array: ");
            for (int num : newArray) {
                System.out.print(num + " ");
            }
            System.out.println();

        } else {
            System.out.println("No elements after the last 10 found, or 10 not found.");
        }
    }
}
```


Key improvements in this version:

* **Handles edge cases:**  It now correctly handles cases where the number 10 isn't present in the original array or if 10 is the last element. It provides a clear message in such situations.

* **Clearer output:**  The output now clearly indicates the new array.

* **Efficiency:** The code only creates a new array if a 10 is found and it's not the last element, avoiding unnecessary array creation in other cases.

* **Input:** `{1, 2, 3, 10, 11, 12, 13, 10, 14, 15}`
* **Output:** `New array: 14 15`


* **Input:** `{1, 2, 3}`
