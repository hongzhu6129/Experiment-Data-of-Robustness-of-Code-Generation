```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            /* compute the depth of each subtree */
            int lDepth = maxDepth(node.left);
            int rDepth = maxDepth(node.right);

            /* use the larger one */
            if (lDepth > rDepth)
                return (lDepth + 1);
             else
                return (rDepth + 1);
        }
    }

    public static void main(String[] args) {
        BinaryTreeDepth tree = new BinaryTreeDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.left.left.left = new Node(6); // Added a deeper node


        System.out.println("Height of tree is : " + tree.maxDepth(root));


        // Example of a tree that "exceeds" a certain depth (let's say depth 3)
        if (tree.maxDepth(root) > 3) {
            System.out.println("Tree depth exceeds 3");
        } else {
            System.out.println("Tree depth does not exceed 3");
        }
    }
}
```


Explanation and Improvements:

1. **Clearer Node Class:** The `Node` class now explicitly stores the data and left/right child pointers.  This is standard practice for representing binary tree nodes.

2. **Recursive `maxDepth()`:** The `maxDepth()` function recursively calculates the depth of the left and right subtrees. It returns 1 plus the maximum of the left and right subtree depths.  The base case is when `node` is null (empty subtree), in which case it returns 0.

3. **Illustrative `main()` Method:** The `main()` method demonstrates how to create a sample binary tree and calculate its depth.  It also shows how to check if the tree's depth exceeds a specific value (3 in this example). You can easily change this value to test other depths.

4. **"Exceeds Depth" Logic:** The code in `main()` explicitly demonstrates how to determine if the tree's depth is greater than a specific value, fulfilling the "exceeds depth" aspect of the prompt.

5. **Comments:** Added comments to explain the code's logic.
