```java
import java.util.Scanner;

public class OctalToHexadecimal {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter an octal number: ");
        String octalNumber = scanner.nextLine();

        try {
            // Convert octal to decimal
            int decimalNumber = 0;
            int power = 0;
            for (int i = octalNumber.length() - 1; i >= 0; i--) {
                int digit = octalNumber.charAt(i) - '0';
                if (digit < 0 || digit > 7) {
                    throw new NumberFormatException("Invalid octal digit: " + octalNumber.charAt(i));
                }
                decimalNumber += digit * Math.pow(8, power);
                power++;
            }


            // Convert decimal to hexadecimal
            String hexadecimalNumber = "";
            while (decimalNumber > 0) {
                int remainder = decimalNumber % 16;
                char hexDigit = (remainder < 10) ? (char) (remainder + '0') : (char) (remainder - 10 + 'A');
                hexadecimalNumber = hexDigit + hexadecimalNumber;
                decimalNumber /= 16;
            }

            if (hexadecimalNumber.isEmpty()) { //Handle the case where input is 0.
                hexadecimalNumber = "0";
            }


            System.out.println("Hexadecimal equivalent: " + hexadecimalNumber);


        } catch (NumberFormatException e) {
            System.out.println("Invalid octal number: " + e.getMessage());
        }
        scanner.close();
    }
}
```


**Test Cases:**

1. **Valid Octal Number:**
   - Input: 123
   - Expected Output: Hexadecimal equivalent: 53

2. **Octal Number with Leading Zeroes:**
   - Input: 00077
   - Expected Output: Hexadecimal equivalent: 3F


3. **Octal Number with a large value:**
   - Input: 7777777
   - Expected Output: Hexadecimal equivalent: FFFFFF


4. **Invalid Octal Number (Contains digits other than 0-7):**
   - Input: 1283 
   - Expected Output: Invalid octal number: Invalid octal digit: 8

5. **Invalid Octal Number (Empty String):**
   - Input: "" (Just press enter)
   - Expected Output: Hexadecimal equivalent: 0  (Handles this gracefully). Note, it will be 0 unlike some other conversions.


6. **Invalid Octal Number (Negative Number):**  Though not strictly octal, good to test.
   - Input: -123
   - Expected Output: Invalid octal number: Invalid octal digit: -

7. **Zero:**
   - Input: 0
   - Expected Output: Hexadecimal equivalent: 0

8. **Single Digit:**
   - Input: 7
   - Expected Output: Hexadecimal equivalent: 7



These test cases cover various scenarios, including valid octal numbers, invalid input (containing invalid digits, empty input), and edge cases like zero and leading zeros. They help to verify that the code handles different situations correctly and produces the expected output.
