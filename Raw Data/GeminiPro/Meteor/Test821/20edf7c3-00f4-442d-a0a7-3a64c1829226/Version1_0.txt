```java
public class CheckJavaInstallation {

    public static void main(String[] args) {
        System.out.println("Java is installed on your computer.");
        System.out.println("Java Version: " + System.getProperty("java.version"));
        System.out.println("Java Runtime Version: " + System.getProperty("java.runtime.version"));
        System.out.println("Java Home: " + System.getProperty("java.home"));
        System.out.println("Java Vendor: " + System.getProperty("java.vendor"));
        System.out.println("Java Vendor URL: " + System.getProperty("java.vendor.url"));
        System.out.println("Java Class Path: " + System.getProperty("java.class.path")); 
    }
}
```

**Explanation:**

Since the program is successfully compiling and running using a Java compiler and runtime environment, it inherently proves that Java is installed.  The code then accesses and prints various system properties related to the Java installation using `System.getProperty()`, providing details about the version, vendor, and other relevant information.  This confirms not only that Java is present but also reveals specific details about the installation.
