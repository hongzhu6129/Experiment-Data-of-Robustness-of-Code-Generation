```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter an integer.");
        }
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("1 "); // Or any other desired element.  For simplicity, using "1"
            }
            System.out.println();  // Newline after each row
        }
    }
}
```


**Explanation and How to Run:**

1. **Compilation:** Save the code as `MatrixPrinter.java`. Compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java 
   ```

2. **Running:** Run the compiled code, providing the matrix size `n` as a command-line argument:
   ```bash
   java MatrixPrinter 5  // This will print a 5x5 matrix
   ```

**Key Improvements:**

* **Input Validation:** The `main` method now checks for:
    * Correct number of command-line arguments.
    * Whether the input is a valid positive integer.
    * Handles `NumberFormatException` if the input is not an integer.
* **Clear Usage Instructions:**  Prints a usage message if the input is incorrect.
* **`printMatrix` Method:** The `printMatrix` method is now more focused and only handles the matrix printing logic.  It receives the size `n` as input.
* **Clear Output:**  Uses `System.out.println()` to ensure each row of the matrix is printed on a new line, making the output easy to read.



Now, you have a robust and user-friendly Java program to print an n-by-n matrix!  You can change the `System.out.print("1 ")`  line in `printMatrix` to print any desired element within the matrix (e.g., `i`, `j`, a calculated value, etc.) if you need a different pattern.
