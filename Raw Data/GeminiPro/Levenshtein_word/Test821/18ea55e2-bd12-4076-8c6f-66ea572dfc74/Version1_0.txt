```java
public class CheckJavaInstallation {

    public static void main(String[] args) {
        System.out.println("Java Version: " + System.getProperty("java.version"));
        System.out.println("Java Runtime Version: " + System.getProperty("java.runtime.version"));
        System.out.println("Java Home: " + System.getProperty("java.home"));
        System.out.println("Java Vendor: " + System.getProperty("java.vendor"));
        System.out.println("Java Vendor URL: " + System.getProperty("java.vendor.url"));
        System.out.println("Java Class Path: " + System.getProperty("java.class.path")); 

        // A more concise way to check (if you just want to know IF it's installed):
        if (System.getProperty("java.version") != null) {
            System.out.println("\nJava is installed on this system.");
        } else {
            System.out.println("\nJava is NOT installed on this system."); // Highly unlikely to reach here if the code runs at all.
        }
    }
}
```


This code fulfills the prompt's requirements by:

1. **Using only `public static void main(String[] args)`:** The entire logic resides within the main method.
2. **Checking for Java installation:** It utilizes `System.getProperty()` to retrieve Java-related system properties.  If these properties are accessible, it implies Java is installed. The output mirrors the sample information provided in the prompt.
3. **Java 1.8 compatibility:** The code is compatible with Java 1.8 (and later versions).


**How to run the code:**

1. **Save:** Save the code as `CheckJavaInstallation.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile it using: `javac CheckJavaInstallation.java`
3. **Run:** Execute the compiled code using: `java CheckJavaInstallation`


The output will display the Java version, runtime version, home directory, vendor, vendor URL, and classpath, confirming that Java is installed and providing details about the installation.  It will also explicitly state "Java is installed on this system."
