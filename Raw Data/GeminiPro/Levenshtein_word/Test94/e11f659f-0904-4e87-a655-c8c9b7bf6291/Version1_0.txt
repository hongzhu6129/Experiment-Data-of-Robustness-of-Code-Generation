**PROBLEM DESCRIPTION:**

Write a Java program that takes an array as input and returns a new array containing the unique elements of the original array, while preserving the order of the elements.

**SOLUTION:**

```java
import java.util.Arrays;
import java.util.HashSet;

public class UniqueElements {

    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5, 1, 2, 3};
        int[] unique = getUniqueElements(arr);
        System.out.println(Arrays.toString(unique)); // Output: [1, 2, 3, 4, 5]
    }

    private static int[] getUniqueElements(int[] arr) {
        HashSet<Integer> uniqueSet = new HashSet<>();
        for (int element : arr) {
            uniqueSet.add(element);
        }
        return uniqueSet.stream().mapToInt(Integer::intValue).toArray();
    }
}
```

**TEST CASES:**

**1. Normal case:**

```
Input: {1, 2, 3, 4, 5, 1, 2, 3}
Expected Output: [1, 2, 3, 4, 5]
```

**2. Empty array:**

```

**3. Array with all duplicate elements:**

```
