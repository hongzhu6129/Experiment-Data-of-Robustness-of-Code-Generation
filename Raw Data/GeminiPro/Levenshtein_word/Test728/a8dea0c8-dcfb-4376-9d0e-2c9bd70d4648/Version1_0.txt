```java
import java.util.Arrays;

public class BinarySearch {

    public static int binarySearch(int[] arr, int target) {
        // Sort the array before performing binary search
        Arrays.sort(arr);

        int low = 0;
        int high = arr.length - 1;

        while (low <= high) {
            int mid = low + (high - low) / 2; // Avoid potential overflow

            if (arr[mid] == target) {
                return mid;  // Element found at index mid
            } else if (arr[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }

        return -1; // Element not found
    }

    public static void main(String[] args) {
        int[] arr = {12, 4, 7, 9, 2, 15, 6}; // Example array
        int target = 9; // Element to search for

        int index = binarySearch(arr, target);

        if (index != -1) {
            System.out.println("Element " + target + " found at index: " + index);
        } else {
            System.out.println("Element " + target + " not found in the array.");
        }


        // Example with a target that is NOT present
        target = 100; 
        index = binarySearch(arr, target);

        if (index != -1) {
            System.out.println("Element " + target + " found at index: " + index);
        } else {
            System.out.println("Element " + target + " not found in the array.");
        }

    }
}
```


Key improvements in this version:

1. **Handles unsorted arrays:** The `binarySearch` method now sorts the input array using `Arrays.sort(arr)` before starting the search.  This ensures that binary search works correctly even if the input isn't initially sorted.

2. **Overflow protection:** The midpoint calculation `int mid = low + (high - low) / 2;`  prevents potential integer overflow that could occur with `(low + high) / 2` if `low` and `high` were very large.

3. **Clearer output:** The `main` method now provides more informative output messages indicating whether the target element was found and its index if it was.

4. **Added test case:** The `main` method now includes a second test case demonstrating the output when the target element is *not* in the array.
