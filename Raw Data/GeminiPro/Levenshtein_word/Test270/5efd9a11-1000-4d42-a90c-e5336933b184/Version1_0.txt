```java
import java.util.Scanner;

public class ToLowerCase {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Get the input string from the user.
        System.out.println("Enter a string: ");
        String inputString = scanner.nextLine();

        // Convert the input string to lowercase.
        String lowercaseString = inputString.toLowerCase();

        // Print the lowercase string.
        System.out.println("The lowercase string is: " + lowercaseString);
    }
}
```

**Test Cases:**

* **Corner Cases:**
    * Empty string: ""
    * String with only uppercase characters: "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    * String with only lowercase characters: "abcdefghijklmnopqrstuvwxyz"
    * String with both uppercase and lowercase characters: "Hello World"

* **Regular Cases:**
    * String with a mix of uppercase and lowercase characters: "This is a Test"
    * String with special characters: "This is a Test!@#$%^&*()"
    * String with numbers: "This is a Test123"
    * String with accented characters: "This is a Tï¿½st"
