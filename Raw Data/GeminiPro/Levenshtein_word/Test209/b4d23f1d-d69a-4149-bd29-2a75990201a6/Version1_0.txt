class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            /* compute the depth of each subtree */
            int lDepth = maxDepth(node.left);
            int rDepth = maxDepth(node.right);

            /* use the larger one */
            if (lDepth > rDepth)
                return (lDepth + 1);
            else
                return (rDepth + 1);
        }
    }

    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.right.left = new Node(6);


        System.out.println("Height of tree is : " + tree.maxDepth(root));

        // Another Example
         root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.left.left = new Node(5);

        System.out.println("Height of tree is : " + tree.maxDepth(root));


    }
}

**Explanation:**

1. **Node Class:** Represents a node in the binary tree.  It has `data` to store the value and `left` and `right` pointers to its children.

2. **maxDepth(Node node):**
   - **Base Case:** If the `node` is `null` (empty subtree), the depth is 0.
   - **Recursive Step:**  It recursively calculates the depth of the left subtree (`lDepth`) and the right subtree (`rDepth`).  The depth of the current node is then 1 + the maximum of `lDepth` and `rDepth`.

3. **main(String[] args):**
   - Creates a `BinaryTreeMaxDepth` object.
   - Constructs example binary trees.
   - Calls `maxDepth()` to calculate and print the height/depth of the trees.
