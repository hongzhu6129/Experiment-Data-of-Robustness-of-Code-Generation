```java
public class SumOfTwoNumbers {

    public static void main(String[] args) {
        // Declare two integer variables
        int num1 = 74;
        int num2 = 36;

        // Calculate the sum
        int sum = num1 + num2;

        // Print the sum to the console
        System.out.println(num1 + " + " + num2 + ": " + sum); 

        // Alternatively, using formatted output:
        System.out.printf("%d + %d: %d\n", num1, num2, sum);


        // Demonstrating taking input from command line arguments (optional):
        if (args.length == 2) {
            try {
                int arg1 = Integer.parseInt(args[0]);
                int arg2 = Integer.parseInt(args[1]);
                int argSum = arg1 + arg2;
                System.out.printf("%d + %d: %d\n", arg1, arg2, argSum);
            } catch (NumberFormatException e) {
                System.err.println("Invalid input: Command line arguments must be integers.");
            }
        }

    }
}
```


**Explanation and Educational Points:**

1. **Variables:**  We declare variables `num1` and `num2` of type `int` (integer) to store the two numbers.
2. **Assignment:** We assign the values 74 and 36 to these variables using the `=` operator.
3. **Sum Calculation:** The `+` operator performs addition. The result of `num1 + num2` is stored in the `sum` variable.
4. **Output:**  
   - `System.out.println()` prints the sum and the original numbers to the console.  The `+` operator, when used with strings, performs concatenation (joining strings together).
   - `System.out.printf()` provides formatted output. `%d` is a placeholder for integers.  `\n` creates a newline at the end.
5. **Command-Line Arguments (Optional):**
   - The `args` array in `main` holds any command-line arguments passed to the program.
   - `args.length` checks how many arguments were provided.
   - `Integer.parseInt()` converts string arguments to integers.
   - The `try-catch` block handles potential `NumberFormatException` if the user enters non-numeric arguments.

**How to Compile and Run:**

1. **Save:** Save the code as `SumOfTwoNumbers.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and type: `javac SumOfTwoNumbers.java`
3. **Run:**  Type: `java SumOfTwoNumbers`  (or `java SumOfTwoNumbers <number1> <number2>` to provide input from the command line).
