```java
public class StringArrayGenerator {

    public static void main(String[] args) {
        if (args.length == 0) {
            System.out.println("Please provide an array length as a command-line argument.");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);

            if (n <= 0) {
                System.out.println("Array length must be greater than 0.");
                return;
            }

            String[] stringArray = new String[n];
            for (int i = 0; i < n; i++) {
                stringArray[i] = String.valueOf(i); // Convert integer to string
            }

            // Print the array (optional)
            System.out.print("[");
            for (int i = 0; i < n; i++) {
                System.out.print(stringArray[i]);
                if (i < n - 1) {
                    System.out.print(", ");
                }
            }
            System.out.println("]");


        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please provide an integer for the array length.");
        }


    }
}
```

**How to Compile and Run:**

1. **Save:** Save the code as `StringArrayGenerator.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile it using: `javac StringArrayGenerator.java`
3. **Run:**  Execute the compiled program, providing the array length as a command-line argument: `java StringArrayGenerator 5`  (This will create an array of length 5 containing "0", "1", "2", "3", "4").



**Explanation:**

1. **Command-line Arguments:** The `args` array in `main` holds the command-line arguments passed to the program.  `args[0]` is the first argument, which we use to get the desired array length.
2. **Error Handling:** We include error handling to check if a command-line argument is provided and if it's a valid integer.
3. **Array Creation:** We create a String array of the specified length `n`.
4. **Populating the Array:**  We use a `for` loop to iterate from 0 to `n-1` and assign the string representation of each integer `i` to the corresponding element in the `stringArray`.  `String.valueOf(i)` converts the integer `i` to its string equivalent.
5. **Printing (Optional):** The code includes a section to print the generated array to the console for verification.  You can remove or comment out this part if you don't need to print the array.
