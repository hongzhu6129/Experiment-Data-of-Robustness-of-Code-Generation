```java
import java.util.Scanner;

public class ScalarTable {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter a number: ");
        int num = scanner.nextInt();

        printScalarTable(num);

        scanner.close(); 
    }

    public static void printScalarTable(int num) {
        System.out.println("Scalar Table for " + num + ":");
        for (int i = 1; i <= 10; i++) {
            System.out.println(num + " x " + i + " = " + (num * i));
        }
    }
}
```

**Test Cases:**

1. **Positive Integer:**
   Input: 5
   Expected Output:
   ```
   Scalar Table for 5:
   5 x 1 = 5
   5 x 2 = 10
   5 x 3 = 15
   5 x 4 = 20
   5 x 5 = 25
   5 x 6 = 30
   5 x 7 = 35
   5 x 8 = 40
   5 x 9 = 45
   5 x 10 = 50
   ```

2. **Zero:**
   Input: 0
   Expected Output:
   ```
   Scalar Table for 0:
   0 x 1 = 0
   0 x 2 = 0
   ...
   0 x 10 = 0
   ```

3. **Negative Integer:**
   Input: -3
   Expected Output:
   ```
   Scalar Table for -3:
   -3 x 1 = -3
   -3 x 2 = -6
   ...
   -3 x 10 = -30
   ```

4. **Large Positive Integer:**
   Input: 123456789  (or any other large integer)
   Expected Output:  The table should be correctly calculated and displayed. This tests for potential overflow issues (though Java's `int` can handle this range).

5. **Large Negative Integer:**
   Input: -987654321 (or any other large negative integer)
   Expected Output:  The table should be correctly calculated and displayed.


**How to Run the Test Cases:**

You can either:

* **Manual Input:**  Run the program multiple times, entering each test case value when prompted.
* **Automated Testing (recommended):**  Use a testing framework like JUnit.  This allows you to write separate test methods for each case and automate the process.  Here's a basic JUnit example:


```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;
import java.io.ByteArrayOutputStream;
import java.io.PrintStream;


public class ScalarTableTest {

    @Test
    void testPositiveInteger() {
        testScalarTable(5, "5 x 1 = 5\n5 x 2 = 10\n... (and so on)"); // Replace ... with the full expected output
    }

    // Add similar test methods for other cases (zero, negative, large integers)

    private void testScalarTable(int input, String expectedOutput) {
        ByteArrayOutputStream outputStream = new ByteArrayOutputStream();
        System.setOut(new PrintStream(outputStream)); 

        ScalarTable.printScalarTable(input); // Call the method you're testing

        String actualOutput = outputStream.toString().trim();
        //assertEquals(expectedOutput, actualOutput); // For exact match. You might need to adjust for line endings etc.
        assertTrue(actualOutput.contains("5 x 1 = 5")); // Example of checking for part of the output (adapt as needed)


        System.setOut(System.out); // Restore standard output
    }
}
```


With JUnit, you can easily run all the test cases at once and quickly see if any fail. This is a much more robust way to test your code.  Be sure to add JUnit to your project if you are using an IDE.
