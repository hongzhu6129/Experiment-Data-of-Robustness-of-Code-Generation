```java
import java.util.Scanner;

public class ScalarTable {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter a number: ");
        int num = scanner.nextInt();

        printScalarTable(num);
    }

    public static void printScalarTable(int num) {
        System.out.println("Scalar Table for " + num + ":");
        for (int i = 1; i <= 10; i++) {
            System.out.println(num + " x " + i + " = " + (num * i));
        }
    }
}
```

**How to Run Test Cases:**

You can either:

* **Manually:** Run the program multiple times, entering each test input.  Compare the output with the expected output.
* **Automated Tests (Recommended):**  Use a testing framework like JUnit. This is a much more robust and efficient way to test.  Here's a simple JUnit example:

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

public class ScalarTableTest {

    @Test
    void testPositiveNumber() {
        ByteArrayOutputStream outContent = new ByteArrayOutputStream();
        System.setOut(new PrintStream(outContent));

        ScalarTable.printScalarTable(5); 

        String expectedOutput = "Scalar Table for 5:\n5 x 1 = 5\n5 x 2 = 10\n... (rest of the table)\n"; // Fill in the full expected output
        assertEquals(expectedOutput, outContent.toString());

        System.setOut(System.out); // Restore standard output
    }


    // Add more @Test methods for other test cases (negative, zero, large numbers)
}
```


This JUnit example captures the printed output and compares it to the expected output string. You would need to add JUnit to your project to use this approach (which is highly recommended for any serious Java development).
