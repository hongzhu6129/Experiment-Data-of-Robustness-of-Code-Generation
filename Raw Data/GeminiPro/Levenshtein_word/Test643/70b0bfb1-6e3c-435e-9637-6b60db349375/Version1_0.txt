```java
import java.util.Arrays;

public class CreateStringFromCharArray {

    public static void main(String[] args) {
        // Create a character array
        char[] chars = {'H', 'e', 'l', 'l', 'o'};

        // Convert the character array to a String object
        String str = new String(chars);

        // Print the String object
        System.out.println(str);
    }
}
```

Test cases:

* **Normal case:** Create a String object from a character array containing valid characters.
```
Input:
char[] chars = {'H', 'e', 'l', 'l', 'o'};
Expected output:
Hello
```

* **Empty character array:** Create a String object from an empty character array.
```
Input:
char[] chars = {};
Expected output:
""
```

* **Special characters:** Create a String object from a character array containing special characters.
```
Input:
char[] chars = {'H', 'e', 'l', 'l', 'o', '!', '?'};
Expected output:
Hello!?
```

* **Null character array:** Create a String object from a null character array.
```
Input:
char[] chars = null;
Expected output:
NullPointerException
```
