```java
import java.util.Arrays;

public class StringAndCharacterArrays {

    public static void main(String[] args) {
        // Create a String object
        String str = "Hello World";

        // Create a character array
        char[] chars = new char[]{'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd'};

        // Print the String object
        System.out.println("String: " + str);

        // Print the character array
        System.out.println("Character array: " + Arrays.toString(chars));
    }
}
```

**Test cases:**

**Corner cases:**

* Empty String object
* Empty character array
* String object with only spaces
* Character array with only spaces

**Other test cases:**

* String object with different characters (uppercase, lowercase, numbers, symbols)
* Character array with different characters (uppercase, lowercase, numbers, symbols)
* String object with repeated characters
* Character array with repeated characters
* String object with Unicode characters
* Character array with Unicode characters
