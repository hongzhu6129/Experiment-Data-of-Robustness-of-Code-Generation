```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileComparator {

    public static void main(String[] args) {
        if (args.length != 2) {
            System.out.println("Usage: java FileComparator <file1> <file2>");
            return;
        }

        String file1Path = args[0];
        String file2Path = args[1];

        try (BufferedReader reader1 = new BufferedReader(new FileReader(file1Path));
             BufferedReader reader2 = new BufferedReader(new FileReader(file2Path))) {

            int comparisonResult = compareFiles(reader1, reader2);

            if (comparisonResult < 0) {
                System.out.println(file1Path + " comes before " + file2Path + " lexicographically.");
            } else if (comparisonResult > 0) {
                System.out.println(file2Path + " comes before " + file1Path + " lexicographically.");
            } else {
                System.out.println(file1Path + " and " + file2Path + " are lexicographically equal.");
            }

        } catch (IOException e) {
            System.err.println("Error reading files: " + e.getMessage());
        }
    }


    private static int compareFiles(BufferedReader reader1, BufferedReader reader2) throws IOException {
        String line1;
        String line2;

        while ((line1 = reader1.readLine()) != null && (line2 = reader2.readLine()) != null) {
            int lineComparison = line1.compareTo(line2);
            if (lineComparison != 0) {
                return lineComparison;
            }
        }


        if (line1 == null && line2 == null) { //both files ended at the same time.
            return 0;
        } else if (line1 == null) { //file1 ended first
            return -1;
        } else { //file2 ended first
            return 1;

        }
    }
}
```


Explanation and Educational points about Lexicographical Comparison:

1. **Character-by-Character Comparison:** Lexicographical comparison works like comparing words in a dictionary. It compares the files character by character.

2. **Unicode Order:**  Java uses Unicode to represent characters.  The `compareTo()` method in the `String` class compares strings based on the Unicode values of their characters.

3. **Shorter vs. Longer:** If one file is a prefix of the other (e.g., "apple" and "apples"), the shorter file comes first lexicographically.


4.  **Line by Line:** The provided code compares files line by line. If all lines are equal up to a point where one file ends, the shorter file is considered lexicographically smaller.

5. **`compareTo()` Return Value:**  The `compareTo()` method returns:
   - A negative integer if the first string is lexicographically less than the second.
   - Zero if the strings are lexicographically equal.
   - A positive integer if the first string is lexicographically greater than the second.

6. **File Handling:** The code uses `BufferedReader` for efficient reading of files.  The `try-with-resources` block ensures that files are closed properly even if exceptions occur.


How to Compile and Run:

1. **Save:** Save the code as `FileComparator.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile using:
   ```bash
   javac FileComparator.java 
   ```
3. **Run:**
   ```bash
   java FileComparator file1.txt file2.txt
   ```
   Replace `file1.txt` and `file2.txt` with the actual paths to the files you want to compare.
