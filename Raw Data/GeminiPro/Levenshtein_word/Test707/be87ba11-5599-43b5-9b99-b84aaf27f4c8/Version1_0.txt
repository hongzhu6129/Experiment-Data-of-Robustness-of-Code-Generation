```java
public class ExcelColumnName {

    public static String Column(int n) {
        StringBuilder columnName = new StringBuilder();

        while (n > 0) {
            int rem = n % 26;

            if (rem == 0) {
                columnName.insert(0, 'Z');
                n = (n / 26) - 1;
            } else {
                columnName.insert(0, (char) ((rem - 1) + 'A'));
                n = n / 26;
            }
        }
        return columnName.toString();
    }

    public static void main(String[] args) {
        System.out.println("Column name for 1: " + Column(1));       // A
        System.out.println("Column name for 28: " + Column(28));      // AB
        System.out.println("Column name for 701: " + Column(701));     // ZY
        System.out.println("Column name for 52: " + Column(52));      // AZ
        System.out.println("Column name for 702: " + Column(702));    // ZZ
        System.out.println("Column name for 705: " + Column(705));   // AAC
        System.out.println("Column name for 26: " + Column(26));   // Z
        System.out.println("Column name for 51: " + Column(51));   // AY

    }
}
```


This code efficiently handles the edge cases where remainders are zero (representing 'Z') and adjusts `n` accordingly to avoid off-by-one errors. The `StringBuilder` is used for efficient string manipulation within the loop.  The `insert(0, ...)` method is used to build the string from right to left, as the calculation proceeds from rightmost column to leftmost.
