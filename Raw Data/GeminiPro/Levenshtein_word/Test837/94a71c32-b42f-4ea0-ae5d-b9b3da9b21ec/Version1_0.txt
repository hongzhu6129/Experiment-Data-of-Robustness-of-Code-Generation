```java
import java.io.File;
import java.io.FileNotFoundException;
import java.util.ArrayList;
import java.util.Collections;
import java.util.Scanner;

public class DescendingWords {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java DescendingWords <filename>");
            System.exit(1);
        }

        String filename = args[0];
        ArrayList<String> words = new ArrayList<>();

        try (Scanner input = new Scanner(new File(filename))) {
            while (input.hasNext()) {
                String word = input.next();
                if (word.matches("^[a-zA-Z].*")) { // Check if the word starts with a letter
                    words.add(word);
                }
            }
        } catch (FileNotFoundException ex) {
            System.out.println("File not found: " + filename);
            System.exit(2);
        }

        Collections.sort(words, Collections.reverseOrder()); // Sort in descending order

        for (String word : words) {
            display(word);
        }
    }


    public static void display(String word) {
        System.out.println(word);
    }


}
```


Key improvements and explanations:

1. **Error Handling for Command-line Argument:** The code now checks if a filename is provided as a command-line argument.  If not, it prints a usage message and exits.

2. **File Not Found Handling:** The code uses a `try-catch` block to handle `FileNotFoundException`.  If the file is not found, it prints an error message and exits.

3. **Word Filtering (Starts with a Letter):**  The code uses a regular expression `^[a-zA-Z].*` to ensure that only words starting with a letter (a-z or A-Z) are added to the `words` list.

4. **Descending Sort:** `Collections.sort(words, Collections.reverseOrder());` sorts the `words` list in descending alphabetical order.

5. **Clearer Output (display method):**  The `display` method now simply prints each word on a separate line, making the output more readable.

6. **Resource Management (try-with-resources):** The `Scanner` is used inside a `try-with-resources` block. This ensures that the `Scanner` and the underlying file are closed automatically, even if an exception occurs.



**How to Compile and Run:**

1. **Save:** Save the code as `DescendingWords.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile it using:
   ```bash
   javac DescendingWords.java
   ```
3. **Run:**  Execute the compiled code, providing the text file name as a command-line argument:
   ```bash
   java DescendingWords input.txt 
   ```
   Replace `input.txt` with the actual name of your text file.  Make sure the text file is in the same directory as the compiled `.class` file, or provide the full path to the file.
