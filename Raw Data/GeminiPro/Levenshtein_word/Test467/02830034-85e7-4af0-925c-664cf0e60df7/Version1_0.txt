```java
import java.time.LocalDate;
import java.time.Period;
import java.time.format.DateTimeFormatter;
import java.time.format.DateTimeParseException;
import java.util.Scanner;

public class DateDifference {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter the first date (yyyy-MM-dd):");
        String date1String = scanner.nextLine();

        System.out.println("Enter the second date (yyyy-MM-dd):");
        String date2String = scanner.nextLine();

        try {
            DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
            LocalDate date1 = LocalDate.parse(date1String, formatter);
            LocalDate date2 = LocalDate.parse(date2String, formatter);

            Period difference = Period.between(date1, date2);

            System.out.println("Difference:");
            System.out.println("Years: " + difference.getYears());
            System.out.println("Months: " + difference.getMonths());
            System.out.println("Days: " + difference.getDays());


        } catch (DateTimeParseException e) {
            System.err.println("Invalid date format. Please use yyyy-MM-dd.");
        }
    }
}
```


Key improvements in this version:

* **Error Handling:**  It includes a `try-catch` block to handle `DateTimeParseException`. This prevents the program from crashing if the user enters an invalid date format. It provides a helpful error message.
* **Clear Prompts:** The prompts to the user are more explicit about the expected date format.
* **Modern Date/Time API:** Uses `java.time` (introduced in Java 8), which is significantly better than older classes like `Date` and `Calendar`.  `LocalDate` is specifically for dates (without time).
* **Period Class:**  Uses `Period.between()` for calculating the difference, directly giving the years, months, and days. This is cleaner and more accurate than manual calculations.
* **Output Formatting:**  The output is neatly formatted, clearly labeling the years, months, and days of the difference.


Example Usage:

```
Enter the first date (yyyy-MM-dd):
2023-10-26
Enter the second date (yyyy-MM-dd):
2024-02-10
Difference:
Years: 0
Months: 3
Days: 15
```
