```java
public class CubeChecker {

    public static void main(String[] args) {
        // Input (hardcoded for this example as per problem description)
        int num = 8; 

        // Check if the number is a perfect cube
        if (isCube(num)) {
            System.out.println("Number is a cube.");
        } else {
            System.out.println("Number is not a cube.");
        }


        // Example with different inputs (you can uncomment these to test):
         checkCube(27); // Output: Number is a cube.
         checkCube(10); // Output: Number is not a cube.
         checkCube(-8); // Output: Number is a cube.
         checkCube(0);  // Output: Number is a cube.


    }


    // Helper function to check if a number is a perfect cube
    private static boolean isCube(int num) {
        // Handle negative numbers (cube root of a negative is also negative)
        int absNum = Math.abs(num); 

        // Efficiently check for cubes using integer arithmetic 
        // (avoids floating-point inaccuracies).
        for (int i = 0; i <= Math.cbrt(absNum); i++) {
            if (i * i * i == absNum) {
                return true;
            }
        }
        return false;
    }


    //  Alternative method (demonstrates handling input directly in main)
    public static void checkCube(int num) {

        if (isCube(num)) {
            System.out.println("Number is a cube.");
        } else {
            System.out.println("Number is not a cube.");
        }
    }
}

```


**Explanation and Improvements:**

1. **`isCube()` Helper Function:**
   - This function encapsulates the cube-checking logic, making the `main` method cleaner and more readable.
   - It handles negative numbers correctly by taking the absolute value (`absNum`).
   - It uses an efficient integer-based approach to avoid potential floating-point errors.

2. **Integer Arithmetic:**
   - The code uses `i * i * i` instead of `Math.pow()` to check for cubes. This avoids potential inaccuracies with floating-point calculations.

3. **Handling Negative Numbers:**
   - The `isCube()` function uses `Math.abs()` to handle negative numbers, ensuring that the cube root check works correctly for both positive and negative inputs.

4. **`checkCube()` example:**
   - The `checkCube()` function demonstrates how to handle user input or different test cases directly within the `main` method (if you weren't hardcoding the input as specified in the original problem.)

5. **Efficiency:** The loop runs up to the cube root of the absolute value of the input number (`Math.cbrt(absNum)`), making it more efficient than checking all numbers up to `num` itself.
