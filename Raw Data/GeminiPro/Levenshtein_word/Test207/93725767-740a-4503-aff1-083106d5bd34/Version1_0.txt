```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            /* compute the depth of each subtree */
            int lDepth = maxDepth(node.left);
            int rDepth = maxDepth(node.right);

            /* use the larger one */
            if (lDepth > rDepth)
                return (lDepth + 1);
            else
                return (rDepth + 1);
        }
    }


    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.left.left.left = new Node(6);


        System.out.println("Height of tree is : " +
                tree.maxDepth(root));


        //Another Example:
        Node root2 = new Node(1);
        root2.left = new Node(2);
        root2.right = new Node(3);
        root2.left.left = new Node(4);
        root2.right.left = new Node(5);
        root2.right.right = new Node(6);
        root2.right.right.left = new Node(7);


        System.out.println("Height of tree is : " +
                tree.maxDepth(root2));

    }
}
```


The term "exceeds measuring" isn't standard terminology. I've interpreted it to mean finding the maximum depth (or height) of the binary tree.  This is a common tree operation.

**Explanation:**

1. **Node Class:** Represents a node in the binary tree. It stores the data and references to the left and right children.

2. **maxDepth(Node node):**
   - **Base Case:** If the node is null (empty tree or end of a branch), the depth is 0.
   - **Recursive Step:**
     - Calculate the depth of the left subtree (`lDepth`).
     - Calculate the depth of the right subtree (`rDepth`).
     - Return the larger of the two depths, plus 1 (to account for the current node).

3. **main(String[] args):**
   - Creates example binary trees.
   - Calls the `maxDepth` method to calculate and print the height of the trees.


**How it works:**

The `maxDepth` function recursively traverses the tree.  At each node, it compares the depths of its left and right subtrees.  The recursion naturally unwinds, and the final result is the maximum depth of the entire tree.
