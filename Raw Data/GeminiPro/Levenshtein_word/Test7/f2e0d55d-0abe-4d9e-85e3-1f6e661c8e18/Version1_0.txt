```java
import java.util.Scanner;

public class AverageCalculator {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        System.out.print("Enter the first number: ");
        double num1 = input.nextDouble();

        System.out.print("Enter the second number: ");
        double num2 = input.nextDouble();

        System.out.print("Enter the third number: ");
        double num3 = input.nextDouble();

        double average = (num1 + num2 + num3) / 3.0;  // Use 3.0 for floating-point division

        System.out.println("The average is: " + average);

        input.close(); // Close the scanner
    }
}
```

**Test Cases:**

**Normal Cases:**

1. **Input:** 10, 20, 30   **Expected Output:** 20.0
2. **Input:** 1.5, 2.5, 3.5  **Expected Output:** 2.5
3. **Input:** -5, 0, 5    **Expected Output:** 0.0

**Corner/Edge Cases:**

4. **Input:** Integer.MAX_VALUE, Integer.MAX_VALUE, Integer.MAX_VALUE  **Expected Output:**  (Should handle potential overflow gracefully, likely resulting in Infinity)
5. **Input:** Integer.MIN_VALUE, Integer.MIN_VALUE, Integer.MIN_VALUE  **Expected Output:** (Should handle potential underflow gracefully, likely resulting in -Infinity)
6. **Input:** 0, 0, 0       **Expected Output:** 0.0
7. **Input:** Double.MAX_VALUE, Double.MAX_VALUE, Double.MAX_VALUE **Expected Output:** Infinity
8. **Input:** Double.MIN_VALUE, Double.MIN_VALUE, Double.MIN_VALUE  **Expected Output:** (A very small positive number close to 0)
