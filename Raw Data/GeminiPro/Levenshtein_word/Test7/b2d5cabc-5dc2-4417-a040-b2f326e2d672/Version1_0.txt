```java
import java.util.Scanner;

public class AverageCalculator {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter three numbers:");

        double num1 = scanner.nextDouble();
        double num2 = scanner.nextDouble();
        double num3 = scanner.nextDouble();

        double average = calculateAverage(num1, num2, num3);

        System.out.println("The average is: " + average);
        scanner.close();
    }

    public static double calculateAverage(double num1, double num2, double num3) {
        return (num1 + num2 + num3) / 3;
    }
}
```


**Test Cases:**

**Normal Cases:**

1. **Input:** 10, 20, 30   **Expected Output:** 20.0
2. **Input:** 1.5, 2.5, 3.0  **Expected Output:** 2.3333333333333335
3. **Input:** -5, 0, 5   **Expected Output:** 0.0

**Edge Cases:**

4. **Input:** 0, 0, 0   **Expected Output:** 0.0  (All zeros)
5. **Input:** Integer.MAX_VALUE, Integer.MAX_VALUE, Integer.MAX_VALUE   **Expected Output:** 2.147483647E9 (Handling large numbers, potential overflow if using int instead of double)


**Corner Cases:**

6. **Input:** Double.MAX_VALUE, Double.MAX_VALUE, Double.MAX_VALUE **Expected Output:** Infinity (Handling the maximum possible double value)
7. **Input:** Double.MIN_VALUE, Double.MIN_VALUE, Double.MIN_VALUE **Expected Output:** 4.9E-324  (Handling the minimum positive double value)
8. **Input:**  1.0, Double.NaN, 2.0  **Expected Output:** NaN (Handling NaN - Not a Number. Any calculation involving NaN results in NaN)
