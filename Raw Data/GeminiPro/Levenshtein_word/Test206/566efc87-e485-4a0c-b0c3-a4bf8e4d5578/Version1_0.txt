```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            /* compute the depth of each subtree */
            int lDepth = maxDepth(node.left);
            int rDepth = maxDepth(node.right);

            /* use the larger one */
            if (lDepth > rDepth)
                return (lDepth + 1);
            else
                return (rDepth + 1);
        }
    }


    public static void main(String[] args) {
        BinaryTreeDepth tree = new BinaryTreeDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.left.left.left = new Node(6); // Added a deeper node


        System.out.println("Height of tree is : " + tree.maxDepth(root));


        // Example with a different tree structure
        Node root2 = new Node(10);
        root2.left = new Node(20);
        root2.right = new Node(30);
        root2.left.left = new Node(40);
        root2.left.right = new Node(50);
        
        System.out.println("Height of tree 2 is : " + tree.maxDepth(root2)); 
    }
}
```


**Explanation and Improvements:**

1. **Node Class:** A `Node` class is defined to represent the nodes of the binary tree. Each node holds an integer `data` value and references to its left and right children.

2. **maxDepth(Node node):** This recursive method calculates the maximum depth (height) of the binary tree.
   - **Base Case:** If the `node` is `null` (empty subtree), the depth is 0.
   - **Recursive Step:**  It recursively calculates the depth of the left and right subtrees (`lDepth` and `rDepth`). The maximum of these depths + 1 (for the current node) is returned.

3. **main(String[] args):**
   - Creates instances of the `BinaryTreeDepth` class and `Node` objects to build sample binary trees.  I've added a second example tree (`root2`) to demonstrate how it works with different structures.
   - Calls the `maxDepth()` method to calculate and print the height of the created trees.



**How to Compile and Run:**

1. **Save:** Save the code as `BinaryTreeDepth.java`.
2. **Compile:** Open a terminal or command prompt and navigate to the directory where you saved the file. Compile the code using:
   ```bash
   javac BinaryTreeDepth.java
   ```
3. **Run:** Execute the compiled code:
   ```bash
   java BinaryTreeDepth 
   ```


The output will show the height/maximum depth of the example binary trees.
