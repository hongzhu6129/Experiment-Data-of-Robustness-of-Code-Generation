```java
public class StringPermutationsWithRepetition {

    public static void main(String[] args) {
        String inputString = "abc";
        permutationWithRepeation(inputString);
    }

    private static void permutationWithRepeation(String inputString) {
        showPermutation(inputString, "");
    }

    private static void showPermutation(String inputString, String currentPermutation) {
        if (currentPermutation.length() == inputString.length()) {
            System.out.println(currentPermutation);
            return;
        }

        for (int i = 0; i < inputString.length(); i++) {
            showPermutation(inputString, currentPermutation + inputString.charAt(i));
        }
    }
}
```


Explanation:

1. **`main(String[] args)`:**
   - This is the entry point of the program.  It takes the input string (e.g., "abc").
   - It calls `permutationWithRepeation()` to start the permutation process.

2. **`permutationWithRepeation(String inputString)`:**
   - This method initializes the permutation process.
   - It calls `showPermutation()` with the input string and an empty string representing the current permutation being built.

3. **`showPermutation(String inputString, String currentPermutation)`:**
   - This is the recursive method that generates the permutations.
   - **Base Case:** If the length of the `currentPermutation` equals the length of the `inputString`, it means a complete permutation has been formed.  It prints the permutation and returns.
   - **Recursive Step:** It iterates through each character of the `inputString`.  For each character, it adds it to the `currentPermutation` and recursively calls `showPermutation()` again. This explores all possible combinations by adding each character at each position.
