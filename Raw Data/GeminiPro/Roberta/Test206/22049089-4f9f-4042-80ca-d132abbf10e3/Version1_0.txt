```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            // Compute the depth of each subtree
            int leftDepth = maxDepth(node.left);
            int rightDepth = maxDepth(node.right);

            // Use the larger one
            return Math.max(leftDepth, rightDepth) + 1; 
        }
    }


    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);

        System.out.println("Height of tree is : " + tree.maxDepth(root));


        // Another test case (just for demonstration)
        Node root2 = new Node(1);
        root2.left = new Node(2);
        root2.right = new Node(3);
        root2.left.left = new Node(4);
        root2.left.left.right = new Node(5);  // Deeper on the left


        System.out.println("Height of tree 2 is : " + tree.maxDepth(root2));



    }
}
```


Explanation:

1. **Node Class:** Represents a node in the binary tree.  It has `data`, `left` child, and `right` child.

2. **maxDepth(Node node):**
   - **Base Case:** If `node` is `null` (empty tree or we've reached the end of a branch), the depth is 0.
   - **Recursive Step:**
     - Recursively calculate the `leftDepth` and `rightDepth` of the left and right subtrees.
     - Return `1 + max(leftDepth, rightDepth)`.  We add 1 because the current node itself adds one level to the depth.  We take the maximum of the left and right depths because the depth of a tree is determined by its longest branch.

3. **main(String[] args):**
   - Creates sample binary trees to test the `maxDepth` method.
   - Prints the calculated height/depth of the trees.
