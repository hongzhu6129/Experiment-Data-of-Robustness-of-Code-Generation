```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. n must be an integer.");
        }
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("[" + i + "," + j + "] "); // Print matrix element
            }
            System.out.println(); // Newline after each row
        }
    }
}
```


**Explanation and How to Run:**

1. **Compilation:** Save the code as `MatrixPrinter.java` and compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java
   ```

2. **Execution:** Run the compiled code from your terminal, providing the matrix size `n` as a command-line argument:
   ```bash
   java MatrixPrinter 5  // Creates a 5x5 matrix
   java MatrixPrinter 3  // Creates a 3x3 matrix
   ```

**Key Improvements:**

* **Input Validation:** The `main` method now includes error handling:
    * It checks if exactly one command-line argument (the matrix size `n`) is provided.
    * It uses a `try-catch` block to handle potential `NumberFormatException` if the input is not a valid integer.
    * It checks if `n` is positive.
* **Clearer Output:** The `printMatrix` method now prints each matrix element in the format `[i,j]` for better readability.
* **Concise and Efficient:** The code is written in a straightforward and efficient way, avoiding unnecessary complexity.


**Example Output (for `n=3`):**
