```java
public class MatrixPrinter {

    public static void main(String[] args) {
        if (args.length != 1) {
            System.out.println("Usage: java MatrixPrinter <n>");
            return;
        }

        try {
            int n = Integer.parseInt(args[0]);
            if (n <= 0) {
                System.out.println("n must be a positive integer.");
                return;
            }
            printMatrix(n);
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter an integer.");
        }
    }

    public static void printMatrix(int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print("[" + i + "," + j + "] "); // Print matrix element (row, column)
            }
            System.out.println(); // Newline after each row
        }
    }
}
```


**Explanation and How to Run:**

1. **Compilation:** Save the code as `MatrixPrinter.java` and compile it using a Java compiler:
   ```bash
   javac MatrixPrinter.java
   ```

2. **Execution:** Run the compiled code, providing the size of the matrix (n) as a command-line argument:
   ```bash
   java MatrixPrinter 5  // This will print a 5x5 matrix
   ```

**Key Improvements in this Version:**

* **Input Validation:**  The `main` method now checks if the correct number of arguments is provided and if the input is a valid positive integer. It handles potential `NumberFormatException` if the user enters non-numeric input.
* **Clearer Output:** The `printMatrix` method now prints the matrix elements in a more readable format, e.g., `[0,0] [0,1] ...`  making it easy to see the row and column indices.
* **Error Handling:** Includes error messages to guide the user on correct usage.
* **Adherence to Method Signatures:**  Strictly uses only `main` and `printMatrix(int)` as required.
