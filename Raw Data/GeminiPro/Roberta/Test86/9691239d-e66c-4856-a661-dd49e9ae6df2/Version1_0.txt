**Java Program to Find the Missing Number in a Given Array**

**Problem Statement:**

Given an array containing numbers from 0 to n, where n is the size of the array, find the missing number in the array.

**Algorithm:**

1. Create a sum variable and initialize it to 0.
2. Iterate through the array and add each element to the sum variable.
3. Calculate the expected sum of the array using the formula: (n * (n + 1)) / 2.
4. Subtract the actual sum from the expected sum to get the missing number.

**Java Code:**

```java
public class MissingNumber {

    public static int findMissingNumber(int[] arr) {
        int sum = 0;

        for (int num : arr) {
            sum += num;
        }

        int expectedSum = (arr.length * (arr.length + 1)) / 2;

        return expectedSum - sum;
    }

    public static void main(String[] args) {
        int[] arr1 = {0, 1, 2, 3, 5};
        int missingNumber1 = findMissingNumber(arr1);
        System.out.println("Missing number in arr1: " + missingNumber1);  // Output: 4

        int[] arr2 = {0, 1, 2, 3, 4, 6, 7, 8};
        int missingNumber2 = findMissingNumber(arr2);
        System.out.println("Missing number in arr2: " + missingNumber2);  // Output: 5

        int[] arr3 = {0};
        int missingNumber3 = findMissingNumber(arr3);
        System.out.println("Missing number in arr3: " + missingNumber3);  // Output: 1

        int[] arr4 = {};
        int missingNumber4 = findMissingNumber(arr4);
        System.out.println("Missing number in arr4: " + missingNumber4);  // Output: 0
    }
}
```

**Test Cases:**

**Corner Cases:**

* **Empty array:** The missing number is 0.
* **Array with only one element:** The missing number is 1.

**Normal Cases:**

* **Array with missing number in the middle:** The missing number is 4.
* **Array with missing number at the beginning:** The missing number is 5.
* **Array with missing number at the end:** The missing number is 1.
