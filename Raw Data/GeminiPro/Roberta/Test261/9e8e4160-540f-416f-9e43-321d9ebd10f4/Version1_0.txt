```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Initialize with 1 as the first element is unique

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                nums[uniqueCount] = nums[i];
                uniqueCount++;
            }
        }
        return uniqueCount;

    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int newLength1 = removeDuplicates(nums1);
        System.out.println("Updated length: " + newLength1); // Output: 2
        System.out.println("Updated array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, newLength1))); // Output: [1, 2]

        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int newLength2 = removeDuplicates(nums2);
        System.out.println("Updated length: " + newLength2); // Output: 5
        System.out.println("Updated array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, newLength2))); // Output: [0, 1, 2, 3, 4]

        // Test with an empty array
        int[] nums3 = {};
        int newLength3 = removeDuplicates(nums3);
        System.out.println("Updated length (empty array): " + newLength3); // Output: 0


    }
}
```


Key improvements and explanations in this code:

1. **Handles Empty Array:**  Added a check at the beginning of `removeDuplicates` to handle the case of an empty input array gracefully.

2. **Clearer Variable Names:**  `uniqueCount` is used instead of `j` to make the purpose of the variable more explicit.

3. **In-Place Modification:** The function modifies the original `nums` array in-place, shifting the unique elements to the beginning.

4. **Returns the Length:** The function returns the length of the updated subarray (containing only unique elements), as required by the problem statement.

5. **Comprehensive Test Cases:** Included test cases with different input arrays, including an empty array, to demonstrate the functionality.

6. **Prints Updated Array (for demonstration):** In the `main` method, I've added code to print the *updated* portion of the array after removing duplicates.  This uses `Arrays.copyOfRange` to create a new array containing only the first `newLength` elements, which are the unique ones. This makes it easier to see the result of the `removeDuplicates` operation.  This extra printing is for demonstration purposes in `main` and isn't strictly part of the `removeDuplicates` method's core functionality.
