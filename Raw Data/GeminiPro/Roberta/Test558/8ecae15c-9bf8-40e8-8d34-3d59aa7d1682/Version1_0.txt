```java
import java.util.Arrays;

public class UpdateArrayElement {

    public static void main(String[] args) {
        // Example usage:
        int[] originalArray = {1, 2, 3, 4, 5};
        int indexToUpdate = 2;  // Update the element at index 2 (which is currently 3)
        int newValue = 10;

        // Update the array element
        if (indexToUpdate >= 0 && indexToUpdate < originalArray.length) {
            originalArray[indexToUpdate] = newValue;
            System.out.println("Updated array: " + Arrays.toString(originalArray)); 
        } else {
            System.out.println("Invalid index.  Index must be between 0 and " + (originalArray.length - 1));
        }


        // Another example with error handling for invalid input (e.g., from user input)
        String indexStr = "4"; // Simulate getting index as a string
        String valueStr = "25";

        try {
            int index = Integer.parseInt(indexStr);
            int value = Integer.parseInt(valueStr);

            if (index >= 0 && index < originalArray.length) {
                originalArray[index] = value;
                System.out.println("Updated array (after second update): " + Arrays.toString(originalArray));
            } else {
                System.out.println("Invalid index. Index must be within the array bounds.");
            }


        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter valid integers for the index and value.");
        }



    }
}
```


Explanation and improvements:

1. **Error Handling:**  The code now includes error handling to prevent `ArrayIndexOutOfBoundsException` if the provided index is out of range.  It checks if `indexToUpdate` is within the valid bounds of the array (0 to length - 1).

2. **Clearer Output:**  The `Arrays.toString()` method is used to print the updated array in a readable format.

3. **Example Usage:**  The `main` method demonstrates how to use the update logic with an example array and values.

4. **Robust Input Handling:** The second example shows how you might handle input (e.g., if you were getting the index and new value from the user as strings).  It uses `Integer.parseInt()` to convert the strings to integers and includes a `try-catch` block to handle potential `NumberFormatException` if the input is not a valid integer.


This improved version is more robust, user-friendly (with informative error messages), and provides a clearer demonstration of how to update an array element in Java.
