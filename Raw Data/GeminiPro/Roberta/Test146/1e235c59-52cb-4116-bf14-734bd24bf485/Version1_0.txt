```java
import java.util.Arrays;

public class CreateStringObjectAndCharArray {

    public static void main(String[] args) {
        // Create a String object
        String str = "Hello World";

        // Create a character array
        char[] chars = {'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd'};

        // Print the String object
        System.out.println(str);

        // Print the character array
        System.out.println(Arrays.toString(chars));
    }
}
```

**Test cases:**

* **Normal case:**
    * Input: `str = "Hello World"`, `chars = {'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd'}`
    * Output:
        ```
        Hello World
        [H, e, l, l, o,  , W, o, r, l, d]
        ```

* **Empty String:**
    * Input: `str = ""`, `chars = {}`
    * Output:
        ```
        
        []
        ```

* **Null String:**
    * Input: `str = null`, `chars = null`
    * Output:
        ```
        null
        null
        ```

* **String with special characters:**
    * Input: `str = "!@#$%^&*()_+=-"`, `chars = {'!', '@', '#', '$', '%', '^', '&', '*', '(', ')', '+', '-', '='}`
    * Output:
        ```
        !@#$%^&*()_+=-
        [!, @, #, $, %, ^, &, *, (, ), +, -, =]
        ```

* **String with spaces:**
    * Input: `str = "Hello World With Spaces"`, `chars = {'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd', ' ', 'W', 'i', 't', 'h', ' ', 'S', 'p', 'a', 'c', 'e', 's'}`
    * Output:
        ```
        Hello World With Spaces
        [H, e, l, l, o,  , W, o, r, l, d,  , W, i, t, h,  , S, p, a, c, e, s]
        ```
