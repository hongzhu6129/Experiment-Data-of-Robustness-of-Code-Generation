```java
import java.util.LinkedList;

public class LinkedListInsertion {

    public static void main(String[] args) {

        // Create a LinkedList of Strings
        LinkedList<String> myList = new LinkedList<>();

        // Inserting at the beginning (head) of the list
        myList.addFirst("Apple"); // Inserts "Apple" at the beginning
        myList.addFirst("Banana"); // Inserts "Banana" before "Apple"
        myList.push("Orange");    // Another way to insert at the beginning (same as addFirst)


        // Inserting at the end (tail) of the list
        myList.addLast("Grape");  // Inserts "Grape" at the end
        myList.add("Mango");     // Another way to insert at the end (same as addLast)


        // Print the updated LinkedList
        System.out.println("Linked List after insertions: " + myList);



        // Example to show how elements are ordered:
        System.out.println("\nIterating and accessing elements:");
        for (int i = 0; i < myList.size(); i++) {
            System.out.println("Element at index " + i + ": " + myList.get(i)); 
        }

         // Demonstrate removing elements:
        String removedFirst = myList.removeFirst(); // Removes and returns the first element
        String removedLast = myList.removeLast(); // Removes and returns the last element
        System.out.println("\nLinked List after removing first and last: " + myList);
        System.out.println("Removed first: " + removedFirst);
        System.out.println("Removed last: " + removedLast);



    }
}
```


Key points explained in the code and comments:

1. **Creating a LinkedList:** `LinkedList<String> myList = new LinkedList<>();` creates a new LinkedList that will store String objects.  You can change `<String>` to other types like `<Integer>`, `<Double>`, or even custom objects.

2. **Inserting at the beginning (head):**
   - `addFirst("element")`:  Adds the specified element to the beginning of the list.
   - `push("element")`: This is equivalent to `addFirst()` and also inserts at the beginning.

3. **Inserting at the end (tail):**
   - `addLast("element")`: Adds the specified element to the end of the list.
   - `add("element")`:  When used with a LinkedList, this is equivalent to `addLast()` and adds to the end.

4. **Printing the list:** `System.out.println(myList);` directly prints the contents of the LinkedList.

5. **Iterating and Accessing:** The `for` loop demonstrates how to iterate through the LinkedList using an index and `myList.get(i)` to retrieve elements.

6. **Removing elements:**  `removeFirst()` and `removeLast()` are used to remove and return the first and last elements, respectively.  This is included to provide a more complete example of common LinkedList operations.
