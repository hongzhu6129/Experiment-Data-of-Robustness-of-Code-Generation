```java
import java.util.Scanner;

public class LargestOfTwo {

    public static int findLargest(int num1, int num2) {
        return Math.max(num1, num2); // Using Math.max for simplicity
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the first integer: ");
        int num1 = scanner.nextInt();

        System.out.print("Enter the second integer: ");
        int num2 = scanner.nextInt();

        int largest = findLargest(num1, num2);
        System.out.println("The largest number is: " + largest);

        scanner.close(); 
    }
}
```

**Test Cases:**

| Test Case | Input 1 | Input 2 | Expected Output |
|---|---|---|---|
| Positive Numbers | 10 | 5 | 10 |
| Negative Numbers | -5 | -10 | -5 |
| Zero and Positive | 0 | 5 | 5 |
| Zero and Negative | 0 | -5 | 0 |
| Both Zero | 0 | 0 | 0 |
| Equal Positive Numbers | 15 | 15 | 15 |
| Equal Negative Numbers | -8 | -8 | -8 |
| Large Positive and Small Negative | 1000 | -1 | 1000 |
| Small Positive and Large Negative | 1 | -1000 | 1 |
| Integer.MAX_VALUE and Integer.MIN_VALUE | 2147483647 | -2147483648 | 2147483647 |  // Corner case: testing extreme values


**How to Run the Test Cases:**

1. **Manually:** You can run the program multiple times, entering the input values for each test case and verifying the output.

2. **Using JUnit (or similar testing framework):**  This is the preferred approach, especially for larger projects.  Here's an example using JUnit:

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

public class LargestOfTwoTest {

    @Test
    void testPositiveNumbers() {
        assertEquals(10, LargestOfTwo.findLargest(10, 5));
    }

    @Test

        assertEquals(-5, LargestOfTwo.findLargest(-5, -10));
    }

    // ... Add more test cases similarly ...

    @Test
    void testIntegerMaxMin() {
        assertEquals(Integer.MAX_VALUE, LargestOfTwo.findLargest(Integer.MAX_VALUE, Integer.MIN_VALUE));
    }
}
```

By using a testing framework, you can automate the testing process and ensure that your code behaves correctly under different conditions.  This makes it easier to catch bugs early on and maintain code quality.
