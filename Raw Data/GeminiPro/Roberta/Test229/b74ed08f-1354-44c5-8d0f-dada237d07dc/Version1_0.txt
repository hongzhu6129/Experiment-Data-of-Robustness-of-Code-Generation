**Java Code for Currency Formatting**

```java
import java.text.NumberFormat;
import java.util.Locale;

public class CurrencyFormatter {

    public static void main(String[] args) {
        double amount = 12345.67;
        String formattedAmount = NumberFormat.getCurrencyInstance(Locale.US).format(amount);
        System.out.println(formattedAmount); // Output: $12,345.67
    }
}
```

**Test Cases**

**Positive Test Cases:**

* Input: 1000.50
* Expected Output: $1,000.50

* Input: 0.01
* Expected Output: $0.01

**Negative Test Cases:**

* Input: -100.00
* Expected Output: -$100.00 (negative sign is preserved)

* Input: 123456789101112.13
* Expected Output: $123,456,789,101,112.13 (large amount with many digits)

**Corner Cases:**

* Input: 0.00
* Expected Output: $0.00 (handles zero amounts)

* Input: null
* Expected Output: $0.00 (handles null input)

* Input: "abc"
* Expected Output: $0.00 (handles non-numeric input)
