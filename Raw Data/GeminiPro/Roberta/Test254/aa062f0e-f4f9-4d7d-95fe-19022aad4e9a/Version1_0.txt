```java
import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Initialize with 1 as the first element is always unique

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) { // Compare with the previous element
                nums[uniqueCount] = nums[i]; // Place the unique element at the correct index
                uniqueCount++; 
            }
        }

        return uniqueCount;
    }


    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Length without duplicates: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: 2, [1, 2]


        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Length without duplicates: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: 5, [0, 1, 2, 3, 4]
        
        int[] nums3 = {}; // Empty array
        int len3 = removeDuplicates(nums3);
        System.out.println("Length without duplicates: " + len3+ ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Output: 0, []

        int[] nums4 = null; // Null array
        int len4 = removeDuplicates(nums4);
        System.out.println("Length without duplicates: " + len4); // Output: 0


        int[] nums5 = {1,2,3,4,5};
        int len5 = removeDuplicates(nums5);
        System.out.println("Length without duplicates: " + len5 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums5, 0, len5))); // Output: 5, [1, 2, 3, 4, 5]

    }
}
```

1. **Handles Edge Cases:**  The code now correctly handles empty (`{}`) and `null` arrays, returning 0 in both cases.

2. **Clearer Variable Names:** Using `uniqueCount` instead of just `k` makes the code more readable.

3. **Detailed Comments:** Comments explain the logic step by step.

4. **Comprehensive Test Cases:** The `main` method includes tests with different scenarios (duplicates, no duplicates, empty array, null array) to demonstrate the code's correctness.

5. **Output Formatting:** The output now clearly shows the length of the unique elements and the modified array (up to the `uniqueCount` index).  `Arrays.copyOfRange` ensures we only print the relevant portion of the array, not the original values after that index.  This is important because `removeDuplicates` modifies the input array in-place.
