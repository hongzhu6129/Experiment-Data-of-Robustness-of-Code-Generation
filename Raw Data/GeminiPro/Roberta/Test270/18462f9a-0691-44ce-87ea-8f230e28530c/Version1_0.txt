```java
import java.util.Locale;

public class ConvertToLowerCase {

    public static void main(String[] args) {
        String input = "HELLO WORLD";
        String output = input.toLowerCase(Locale.ROOT);
        System.out.println(output); // hello world
    }
}
```

Test cases:

* **Regular case:**
    * Input: "Hello world"
    * Output: "hello world"
* **Uppercase:**
    * Input: "HELLO WORLD"
    * Output: "hello world"
* **Numbers:**
    * Input: "12345"
    * Output: "12345"
* **Special characters:**
    * Input: "!@#$%^&*()"
    * Output: "!@#$%^&*()"
* **Empty string:**
    * Input: ""
    * Output: ""
* **Null string:**
    * Input: null
    * Output: null
