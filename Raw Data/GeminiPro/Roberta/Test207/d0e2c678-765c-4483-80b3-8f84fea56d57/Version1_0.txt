```java
class Node {
    int data;
    Node left, right;

    Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            // Recursively compute the depth of each subtree
            int leftDepth = maxDepth(node.left);
            int rightDepth = maxDepth(node.right);

            // Use the larger one
            if (leftDepth > rightDepth)
                return (leftDepth + 1);
             else
                return (rightDepth + 1);
        }
    }


    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);


        System.out.println("Height of tree is : " + tree.maxDepth(root));


        // Another Example:
        Node root2 = new Node(1);
        root2.left = new Node(2);
        root2.right = new Node(3);
        root2.left.left = new Node(4);
        root2.left.left.left = new Node(5); // Deeper left subtree

         System.out.println("Height of tree is : " + tree.maxDepth(root2));

    }
}
```


Key improvements and explanations in this code:

1. **Clear Node Class:**  A separate `Node` class is defined to represent the nodes of the binary tree.  This is standard practice and makes the code more organized.

2. **Recursive `maxDepth()`:**  The `maxDepth()` function is implemented recursively, which is the most elegant and efficient way to solve this problem.  The base case is when `node` is null (empty subtree), in which case the depth is 0. Otherwise, it calculates the depths of the left and right subtrees and returns the larger of the two, plus 1 (for the current node).

3. **Example Usage in `main()`:** The `main()` method demonstrates how to create a sample binary tree and then call the `maxDepth()` function to get the height. Two examples are provided to illustrate different tree structures.

4. **Comments:**  The code is well-commented to explain the logic.

5. **Improved Formatting:** The code is formatted for better readability.
