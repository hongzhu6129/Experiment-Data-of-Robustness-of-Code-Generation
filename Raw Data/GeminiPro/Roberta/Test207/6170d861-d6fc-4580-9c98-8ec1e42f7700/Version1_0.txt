```java
class Node {
    int data;
    Node left, right;

    public Node(int item) {
        data = item;
        left = right = null;
    }
}

public class BinaryTreeMaxDepth {

    public int maxDepth(Node node) {
        if (node == null) {
            return 0;
        } else {
            /* compute the depth of each subtree */
            int lDepth = maxDepth(node.left);
            int rDepth = maxDepth(node.right);

            /* use the larger one */
            if (lDepth > rDepth)
                return (lDepth + 1);
            else
                return (rDepth + 1);
        }
    }

    public static void main(String[] args) {
        BinaryTreeMaxDepth tree = new BinaryTreeMaxDepth();

        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.left.right.left = new Node(6);


        System.out.println("Height of tree is : " + tree.maxDepth(root));

       //Another test case
        Node root2 = new Node(10);
        root2.left = new Node(20);
        root2.right = new Node(30);
        root2.left.left = new Node(40);
        root2.left.right = new Node(50);

        System.out.println("Height of tree is : " + tree.maxDepth(root2));


    }
}
```


**Explanation:**

1. **Node Class:** Represents a node in the binary tree. It contains the data and pointers to the left and right children.

2. **maxDepth(Node node):**
   - **Base Case:** If the node is null (empty subtree), the depth is 0.
   - **Recursive Step:**
     - Calculate the depth of the left subtree (`lDepth`) recursively.
     - Calculate the depth of the right subtree (`rDepth`) recursively.
     - Return the maximum of `lDepth` and `rDepth`, plus 1 (to account for the current node).

3. **main(String[] args):**
   - Creates a `BinaryTreeMaxDepth` object.
   - Creates a sample binary tree.
   - Calls `maxDepth()` with the root node to calculate the maximum depth/height.
   - Prints the result.



This implementation uses a simple recursive approach to efficiently determine the maximum depth of the binary tree.  It adheres to the requested method signatures.
