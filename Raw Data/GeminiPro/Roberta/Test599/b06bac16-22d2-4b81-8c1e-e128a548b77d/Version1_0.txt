```java
import java.util.LinkedList;

public class CompareLinkedLists {

    public static void main(String[] args) {
        // Create two LinkedLists
        LinkedList<Integer> list1 = new LinkedList<>();
        LinkedList<Integer> list2 = new LinkedList<>();

        // Populate list1
        list1.add(1);
        list1.add(2);
        list1.add(3);

        // Populate list2 - Example 1: Identical to list1
        list2.add(1);
        list2.add(2);
        list2.add(3);

        System.out.println("List 1: " + list1);
        System.out.println("List 2: " + list2);
        System.out.println("Are lists equal (Example 1)? " + list1.equals(list2));


        // Clear list2 and repopulate - Example 2: Different elements
        list2.clear();
        list2.add(4);
        list2.add(5);
        list2.add(6);

        System.out.println("\nList 1: " + list1);
        System.out.println("List 2: " + list2);
        System.out.println("Are lists equal (Example 2)? " + list1.equals(list2));



        // Clear list2 and repopulate - Example 3: Same elements, different order
        list2.clear();
        list2.add(3);
        list2.add(2);
        list2.add(1);

        System.out.println("\nList 1: " + list1);
        System.out.println("List 2: " + list2);
        System.out.println("Are lists equal (Example 3)? " + list1.equals(list2));


        // Clear list2 and repopulate - Example 4: One list is a subset of the other
        list2.clear();
        list2.add(1);
        list2.add(2);


        System.out.println("\nList 1: " + list1);
        System.out.println("List 2: " + list2);
        System.out.println("Are lists equal (Example 4)? " + list1.equals(list2));


        // Example 5: Comparing with null
        list2 = null; 
        System.out.println("\nList 1: " + list1);
        System.out.println("List 2: " + list2);
        System.out.println("Are lists equal (Example 5)? " + (list1 == list2)); // Use == for null check

    }
}
```


Explanation:

1. **Create LinkedLists:** We create two `LinkedList` objects, `list1` and `list2`, using the `LinkedList<>()` constructor.  We've used Integer as the generic type, but you can use any other object type.

2. **Populate the Lists:**  The `add()` method adds elements to the end of the linked lists.

3. **Compare using `equals()`:** The core of the comparison is done by the `equals()` method. This method, inherited from the `AbstractList` class, compares the lists element by element and in order.  It returns `true` if the lists are of the same size and contain the same elements in the same order. Otherwise, it returns `false`.

4. **Illustrative Examples:** The code demonstrates various scenarios, including identical lists, lists with different elements, lists with the same elements in a different order, and a list being a subset of another.  This helps understand the behavior of the `equals()` method thoroughly.

5. **Null Check (Example 5):**  It's important to handle cases where one or both of the lists might be `null`.  Attempting to call `equals()` on a `null` object will result in a `NullPointerException`. The example shows how to safely check for null using `==` before attempting the comparison.


This comprehensive example makes it clear how to compare `LinkedLists` in Java and handles different scenarios you might encounter, including the essential null check.
