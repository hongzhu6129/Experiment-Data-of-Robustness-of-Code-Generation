```java
public class MetastasizeArrays {

    public static void main(String[] args) {
        int[] array1 = {1, 2, 3, 4, 5};
        int[] array2 = {6, 7, 8, 9, 10};

        // Check if arrays are of the same length.  If not, we can't metastasize.
        if (array1.length != array2.length) {
            System.out.println("Arrays must be of the same length for metastasization.");
            return; 
        }

        int[] result = new int[array1.length * 2];

        // Interleave elements
        for (int i = 0; i < array1.length; i++) {
            result[2 * i] = array1[i];
            result[2 * i + 1] = array2[i];
        }

        // Print the resulting array
        System.out.print("Metastasized Array: [");
        for (int i = 0; i < result.length; i++) {
            System.out.print(result[i]);
            if (i < result.length - 1) {
                System.out.print(", ");
            }
        }
        System.out.println("]");
    }
}
```


This code takes two integer arrays (array1 and array2) and "metastasizes" them, meaning it interleaves their elements. It first checks if the arrays have the same length.  If they don't, it prints an error message and exits. Otherwise, it creates a new array (result) twice the size of the original arrays.  It then iterates through the original arrays, placing elements from array1 at even indices and elements from array2 at odd indices in the result array.  Finally, it prints the metastasized array.


Example Output (with the given array1 and array2):

```
Metastasized Array: [1, 6, 2, 7, 3, 8, 4, 9, 5, 10]
```
