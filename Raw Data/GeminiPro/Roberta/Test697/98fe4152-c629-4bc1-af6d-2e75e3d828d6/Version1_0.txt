class Solution {
    /**
     * Counts the number of integers from 1 to n (inclusive) that do not contain the digit 7.
     *
     * @param n The upper limit (inclusive).
     * @return The count of numbers without the digit 7.
     */
    static int count_nums_not_7(int n) {
        int count = 0;
        for (int i = 1; i <= n; i++) {
            if (!containsDigit7(i)) {
                count++;
            }
        }
        return count;
    }

    /**
     * Checks if a number contains the digit 7.
     *
     * @param num The number to check.
     * @return True if the number contains the digit 7, false otherwise.
     */
    private static boolean containsDigit7(int num) {
        while (num > 0) {
            if (num % 10 == 7) {
                return true;
            }
            num /= 10;
        }
        return false;
    }


    public static void main(String[] args) {
        int n = 20;  // Example: Count numbers without 7 from 1 to 20
        int result = count_nums_not_7(n);
        System.out.println("The count of numbers without 7 from 1 to " + n + " is: " + result);


        n = 100; // Example 2: Count numbers without 7 from 1 to 100
        result = count_nums_not_7(n);
        System.out.println("The count of numbers without 7 from 1 to " + n + " is: " + result);

    }
}

3. **Efficient Digit Check:**  Uses modulo (`%`) and integer division (`/`) for an efficient digit-by-digit check.
4. **Comprehensive Main Method:**  Provides example usage with different input values to demonstrate the function.
5. **Comments:** Added comments to explain the purpose and logic of the code clearly.
6. **Robustness:** Handles edge cases (e.g., n = 0) correctly.  It will return 0 if n is 0, which is logical as there are no numbers between 1 and 0 (inclusive).
