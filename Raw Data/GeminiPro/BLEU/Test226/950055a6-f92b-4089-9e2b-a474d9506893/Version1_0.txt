```java
import java.util.Arrays;

class Solution {
    /**
     * Given a sorted array nums, remove the duplicates in-place such that each element appears only once and returns the new length.
     * Do not allocate extra space for another array; you must do this by modifying the input array in-place with O(1) extra memory.
     *
     * Example 1:
     * Input: nums = [1,1,2]
     * Output: 2, nums = [1,2,_]
     * Explanation: Your function should return k = 2, with the first two elements of nums being 1 and 2 respectively.
     * It does not matter what you leave beyond the returned k (hence they are underscores).
     *
     * Example 2:
     * Input: nums = [0,0,1,1,1,2,2,3,3,4]
     * Output: 5, nums = [0,1,2,3,4,_,_,_,_,_]
     * Explanation: Your function should return k = 5, with the first five elements of nums being 0, 1, 2, 3, and 4 respectively.
     * It does not matter what you leave beyond the returned k (hence they are underscores).
     *
     * @param nums a sorted integer array
     * @return the new length of the array with duplicates removed
     */
    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueIndex = 0; // Index to track the position of unique elements

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[uniqueIndex]) {
                uniqueIndex++;
                nums[uniqueIndex] = nums[i]; 
            }
        }

        return uniqueIndex + 1;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: 2, [1, 2]


        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: 5, [0, 1, 2, 3, 4]


        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Length: " + len3 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Output: 0, []

        int[] nums4 = null;
        int len4 = removeDuplicates(nums4);
        System.out.println("Length: " + len4);  // Output: 0


        int[] nums5 = {1};
        int len5 = removeDuplicates(nums5);
        System.out.println("Length: " + len5 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums5, 0, len5))); // Output: 1, [1]
    }
}
```


Key improvements and explanations:

1. **Clearer Variable Names:** Using `uniqueIndex` instead of just `i` or `j` for the index tracking unique elements makes the code easier to understand.

2. **Comprehensive Comments:**  The Javadoc comment at the beginning of the `removeDuplicates` method explains its purpose, parameters, return value, and provides examples, making it an educational piece of code.  In-line comments also help to clarify the logic.

3. **Edge Case Handling:** The code explicitly handles cases where the input array is `null` or empty, ensuring robustness.

4. **Concise Logic:** The core logic for removing duplicates is kept efficient and straightforward using a single `for` loop and the `uniqueIndex` to track the position for placing the next unique element.

5. **Illustrative `main` Method:** The `main` method now demonstrates the function's behavior with multiple test cases, including edge cases (empty array, null array, single-element array), showcasing its usage and output.  The use of `Arrays.copyOfRange` in the `main` method ensures we only print the relevant part of the array (up to the new length).



This revised code provides a more educational and robust solution to the problem of removing duplicates from a sorted array while adhering to the specified method signatures.
