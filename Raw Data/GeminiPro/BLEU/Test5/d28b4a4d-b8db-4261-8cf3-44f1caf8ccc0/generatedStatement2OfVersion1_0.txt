

**Test Cases:**

**Normal Cases:**

1. **Input:** 10, 20, 30
   **Expected Output:** 20.0

2. **Input:** 1.5, 2.5, 3.5
   **Expected Output:** 2.5

3. **Input:** -5, 0, 5
   **Expected Output:** 0.0


**Edge/Corner Cases:**

4. **Input:** 0, 0, 0
   **Expected Output:** 0.0

5. **Input:** Integer.MAX_VALUE, Integer.MAX_VALUE, Integer.MAX_VALUE  (Test overflow, though Java handles this with doubles)
   **Expected Output:** 2.147483647E9

6. **Input:** Integer.MIN_VALUE, Integer.MIN_VALUE, Integer.MIN_VALUE (Test underflow)
   **Expected Output:** -2.147483648E9

7. **Input:** Double.MAX_VALUE, Double.MAX_VALUE, Double.MAX_VALUE (Test double overflow)
   **Expected Output:** Infinity

8. **Input:** Double.MIN_VALUE, Double.MIN_VALUE, Double.MIN_VALUE (Test double min value handling)
   **Expected Output:** 4.9E-324

9. **Input:** 1, 2, Double.NaN (Test NaN input)
   **Expected Output:** NaN  (Any operation involving NaN results in NaN).
