

**Test Cases:**

1. **Standard Case:**
   - Input: `start = 1`, `end = 99`
   - Expected Output: Odd numbers from 1 to 99 printed, each on a new line.  (1, 3, 5, ... , 97, 99)

2. **Start Greater than End:**
   - Input: `start = 10`, `end = 5`
   - Expected Output: "Invalid input range."

3. **Negative Input:**
   - Input: `start = -5`, `end = 5`
   - Expected Output: "Invalid input range."  (Although some odd numbers exist in this range, the problem specifies 1 to 99).

4. **Single Odd Number:**
   - Input: `start = 7`, `end = 7`
   - Expected Output: `7`

5. **Single Even Number:**  
   - Input: `start = 8`, `end = 8`
   - Expected Output: *No output* (because 8 is even).

6. **Start Even, End Odd:**
   - Input: `start = 2`, `end = 7`
   - Expected Output: `3`, `5`, `7`


7. **Start Odd, End Even:**
   - Input: `start = 3`, `end = 8`
   - Expected Output: `3`, `5`, `7`
