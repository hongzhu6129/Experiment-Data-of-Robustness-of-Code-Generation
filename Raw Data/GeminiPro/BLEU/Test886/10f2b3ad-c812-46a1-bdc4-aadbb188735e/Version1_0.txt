```java
public class MaxSubarray {

    public static void main(String[] args) {
        int[] nums1 = {-2, 1, -3, 4, -1, 2, 1, -5, 4};
        System.out.println("Maximum contiguous sum is " + max_SubArray(nums1)); // Output: 6

        int[] nums2 = {-1};
        System.out.println("Maximum contiguous sum is " + max_SubArray(nums2)); // Output: -1
        
        int[] nums3 = {5, 4, -1, 7, 8};
        System.out.println("Maximum contiguous sum is " + max_SubArray(nums3)); // Output: 23


    }

    public static int max_SubArray(int[] nums) {
        int maxSoFar = nums[0];  // Initialize with the first element
        int currentMax = nums[0];

        for (int i = 1; i < nums.length; i++) {
            // Decide whether to extend the current subarray or start a new one
            currentMax = Math.max(nums[i], currentMax + nums[i]);

            // Update the overall maximum if the current subarray sum is greater
            maxSoFar = Math.max(maxSoFar, currentMax);
        }

        return maxSoFar;
    }
}
```


Kadane's Algorithm Explanation:

1. **Initialization:**
   - `maxSoFar`: Stores the maximum subarray sum seen so far. Initialized with the first element of the array.
   - `currentMax`: Stores the maximum subarray sum ending at the current position. Also initialized with the first element.

2. **Iteration:**
   - The loop iterates from the second element (`i = 1`) to the end of the array.
   - Inside the loop:
     - `currentMax = Math.max(nums[i], currentMax + nums[i]);`: This is the core of Kadane's algorithm.  It checks if it's better to start a new subarray at the current element (`nums[i]`) or extend the previous subarray (`currentMax + nums[i]`).
     - `maxSoFar = Math.max(maxSoFar, currentMax);`: Updates `maxSoFar` if the current subarray sum is greater than the maximum seen so far.

3. **Return:**
   - After the loop completes, `maxSoFar` will hold the maximum subarray sum, which is returned.


Example:

For `nums = {-2, 1, -3, 4, -1, 2, 1, -5, 4}`:

| i | nums[i] | currentMax | maxSoFar |
|---|---|---|---|
| 0 | -2 | -2 | -2 |
| 1 | 1 | 1 | 1 |
| 2 | -3 | -2 | 1 |
| 3 | 4 | 4 | 4 |
| 4 | -1 | 3 | 4 |
| 5 | 2 | 5 | 5 |
| 6 | 1 | 6 | 6 |
| 7 | -5 | 1 | 6 |
| 8 | 4 | 5 | 6 |


The final `maxSoFar` is 6, which is the correct maximum subarray sum (corresponding to the subarray `[4, -1, 2, 1]`).
