```java
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;

public class StringToDateConverter {

    public static void main(String[] args) {
        // String to Date conversion
        String dateString = "2023-11-20"; // Example date string
        SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd"); // Define the date format

        try {
            Date date = dateFormat.parse(dateString);
            System.out.println("Converted Date: " + date);


            // Date to String Conversion
            SimpleDateFormat outputFormat = new SimpleDateFormat("MMMM dd, yyyy"); //  Different output format
            String convertedString = outputFormat.format(date);
            System.out.println("Converted String: " + convertedString);


            //Demonstrating with another date format (e.g., dd/MM/yyyy)
            String dateString2 = "25/12/2024";
            SimpleDateFormat dateFormat2 = new SimpleDateFormat("dd/MM/yyyy");

            Date date2 = dateFormat2.parse(dateString2);
            System.out.println("Converted Date 2: " + date2);

            SimpleDateFormat outputFormat2 = new SimpleDateFormat("EEEE, MMM dd, yyyy"); // Another output format
            String convertedString2 = outputFormat2.format(date2);
            System.out.println("Converted String 2: " + convertedString2);



        } catch (ParseException e) {
            System.err.println("Invalid date format. Please use yyyy-MM-dd or dd/MM/yyyy"); // Handle parsing errors
        }
    }
}
```


Key improvements in this version:

* **Error Handling:** Includes a `try-catch` block to handle `ParseException` if the input date string is not in the expected format.  This makes the code more robust.
* **Clearer Output:**  Improved output messages to clearly indicate which conversion is being demonstrated.
* **Multiple Formats Demonstrated:** Shows how to handle different input date formats (yyyy-MM-dd and dd/MM/yyyy) and how to format the output date in various ways using different `SimpleDateFormat` patterns.
* **Comments:** Added comments explaining the code for better readability.


How to compile and run:

1. **Save:** Save the code as `StringToDateConverter.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and run: `javac StringToDateConverter.java`
3. **Run:**  `java StringToDateConverter`


The output will show the date conversions in different formats. If you provide an invalid date string, the error message will be displayed.
