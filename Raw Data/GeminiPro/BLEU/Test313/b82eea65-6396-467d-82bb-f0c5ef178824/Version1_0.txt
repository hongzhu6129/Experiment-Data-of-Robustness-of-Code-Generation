    public static int[] partitionArray2(int[] arr) {
        if (arr == null) {
            return null; // Handle null input
        }
        int[] partitionedArr = new int[arr.length];
        int evenIndex = 0;
        int oddIndex = arr.length - 1;

        for (int num : arr) {
            if (num % 2 == 0) {
                partitionedArr[evenIndex++] = num;
            } else {
                partitionedArr[oddIndex--] = num;
            }
        }

        return partitionedArr;
    }
}

1. **Handles Null Input:** The code now explicitly checks for `null` input and returns `null` in that case. This prevents `NullPointerExceptions`.

3. **Enhanced For Loop:** Using an enhanced `for` loop ( `for (int num : arr)` ) simplifies iterating through the array.

4. **In-Place Partitioning (Optional):**  If the requirement is to modify the original array *in-place* (without creating a new array),  you'd need a different approach using two pointers and swapping elements.  However, the provided prompt implied creating a new, partitioned array.
