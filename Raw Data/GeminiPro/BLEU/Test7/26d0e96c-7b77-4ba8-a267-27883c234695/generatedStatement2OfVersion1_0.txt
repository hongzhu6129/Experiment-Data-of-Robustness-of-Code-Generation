

**Test Cases:**

**Normal Cases:**

1. **Input:** num1 = 10, num2 = 20, num3 = 30
   **Expected Output:** 20.0

2. **Input:** num1 = 5.5, num2 = 10.5, num3 = 15.0
   **Expected Output:** 10.333333333333334

3. **Input:** num1 = -5, num2 = 0, num3 = 5
   **Expected Output:** 0.0

**Corner Cases:**

4. **Input:** num1 = 0, num2 = 0, num3 = 0
   **Expected Output:** 0.0  (Testing all zeros)

5. **Input:** num1 = Integer.MAX_VALUE, num2 = Integer.MAX_VALUE, num3 = Integer.MAX_VALUE // Or use Double.MAX_VALUE
   **Expected Output:**  Infinity (or a very large number close to infinity) (Testing overflow)


6. **Input:** num1 = Integer.MIN_VALUE, num2 = Integer.MIN_VALUE, num3 = Integer.MIN_VALUE // Or use Double.MIN_VALUE (Not really a practical corner case, but good for completeness).
   **Expected Output:**  -Infinity (or a very large negative number) (Testing underflow)


7. **Input:** num1 = Double.NaN, num2 = 10, num3 = 20  //Testing NaN input
   **Expected Output:** NaN
