import java.util.Arrays;

public class StringDuplicates {

    static void CountCharacters(String str, int[] charCounts) {
        str = str.toLowerCase(); // Case-insensitive counting
        for (char c : str.toCharArray()) {
            if (c >= 'a' && c <= 'z') {
                charCounts[c - 'a']++;
            }
        }
    }

    static void showDuplicates(String str) {
        int[] charCounts = new int[26]; // Assuming only lowercase English alphabets
        CountCharacters(str, charCounts);

        System.out.print("Duplicate characters in \"" + str + "\": ");
        boolean foundDuplicates = false; 
        for (int i = 0; i < 26; i++) {
            if (charCounts[i] > 1) {
                System.out.print((char) ('a' + i) + " ");
                foundDuplicates = true;
            }
        }

        if (!foundDuplicates) {
            System.out.print("None");
        }
        System.out.println(); // Add a newline for better formatting

    }

    public static void main(String[] args) {
        showDuplicates("Programming");
        showDuplicates("Java");
        showDuplicates("abracadabra");
        showDuplicates("hello");
        showDuplicates("wxyz");  // Example with no duplicates
    }
}

1. **Case-Insensitive Counting:**  The `CountCharacters` method now converts the input string to lowercase using `str.toLowerCase()` before counting.  This ensures that 'P' and 'p' are treated as the same character.

2. **Clearer Output:** The `showDuplicates` method now provides clearer output:
   - It prints the original string being analyzed.
   - It explicitly states "None" if no duplicates are found.
   - A newline is added at the end of each output for better readability.

3. **`charCounts` Array Size:** The `charCounts` array is initialized to size 26 (number of lowercase English letters).  This is a reasonable assumption for many cases, but if you need to handle a wider range of characters (uppercase, special characters, etc.), you would need to adjust the array size and the logic within `CountCharacters` accordingly. 

4. **`foundDuplicates` flag:**  This boolean flag is used to print "None" only if no duplicates have actually been found. This prevents unnecessary printing of "None" even when duplicates are present.
