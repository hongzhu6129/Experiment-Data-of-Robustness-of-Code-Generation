```java
import java.util.Arrays;

public class SortIntArray {

    public static void sort(int[] arr) {
        Arrays.sort(arr);
    }

    public static void main(String[] args) {
        // The original error "invalid literal for int() with base 10: 'computer'" 
        // occurs when you try to convert a non-numeric string (like "computer") 
        // into an integer using Integer.parseInt() or similar.

        // This example demonstrates how to handle such a situation gracefully.

        String[] stringArray = {"123", "456", "computer", "789", "0"}; 
        int[] intArray = new int[stringArray.length];  // Initialize an array to hold integers
        int validIntegers = 0; // Keep track of valid integers

        for (int i = 0; i < stringArray.length; i++) {
            try {
                intArray[validIntegers] = Integer.parseInt(stringArray[i]);
                validIntegers++;
            } catch (NumberFormatException e) {
                System.err.println("Invalid integer format: " + stringArray[i]);
                // We handle the exception and simply continue to the next element.
            }
        }

        // Create a new array with the correct size to store only valid integers
        int[] sortedIntArray = new int[validIntegers];
        System.arraycopy(intArray, 0, sortedIntArray, 0, validIntegers);

        // Now we sort and print only the valid integers.
        sort(sortedIntArray);

        System.out.print("Sorted Integer Array: ");
        for (int num : sortedIntArray) {
            System.out.print(num + " ");
        }
        System.out.println();


        // Example using only int arrays (for demonstration of the sort method)
        int[] numbers = {5, 2, 8, 1, 9, 4};
        sort(numbers);

        System.out.print("Sorted numbers: ");
        for (int num : numbers) {
            System.out.print(num + " ");
        }
        System.out.println();
    }
}
```



Explanation and how it addresses the error:

1. **Error Context:** The error "invalid literal for int() with base 10: 'computer'" arises when you attempt to convert a non-numeric string into an integer.

2. **Handling the Error:**  The code demonstrates the use of a `try-catch` block.  The `try` block attempts the conversion using `Integer.parseInt()`. If the string is not a valid integer, a `NumberFormatException` is thrown.  The `catch` block handles this exception, prints an error message, and allows the program to continue processing other elements.

3. **Storing Valid Integers:** The code uses a `validIntegers` counter and `System.arraycopy()` to create a new array containing only the valid integers parsed from the string array. This ensures that the `sort()` method operates only on valid integer data.

4. **Sorting and Printing:**  The `sort()` method (using `Arrays.sort()`) sorts the valid integers.  The code then prints the sorted array.


This revised version demonstrates how to handle the error gracefully, avoiding program crashes and providing useful feedback to the user. It also demonstrates the `sort()` method's functionality with a simple integer array.
