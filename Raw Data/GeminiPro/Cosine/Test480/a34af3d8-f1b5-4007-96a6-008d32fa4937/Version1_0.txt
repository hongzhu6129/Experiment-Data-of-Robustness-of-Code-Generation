import java.util.TimeZone;

public class TimeZones {

    public static void main(String[] args) {
        String[] availableIDs = TimeZone.getAvailableIDs();

        System.out.println("Available Time Zones (UTC/GMT):");

        for (String id : availableIDs) {
            TimeZone tz = TimeZone.getTimeZone(id);

            // Filter and display only UTC/GMT time zones
            if (id.startsWith("Etc/GMT") || id.startsWith("UTC") || id.startsWith("GMT")) {  
                 System.out.println(id + ": " + tz.getDisplayName()); // Optionally display the longer name as well

**Explanation and Improvements:**

1. **Filtering for UTC/GMT:**  The code now efficiently filters for time zones that start with "Etc/GMT", "GMT", or "UTC".  This provides a more precise list of relevant time zones.

2. **`getDisplayName()` (Optional):**  I've added `tz.getDisplayName()` to show the longer, more human-readable name of the time zone along with its ID. This is optional but can make the output clearer.

3. **Alternative Using `getRawOffset()` (Commented out):**  I've included a commented-out section that demonstrates how to use `getRawOffset()` to filter for time zones with offsets that are exact multiples of one hour (as GMT/UTC time zones are).
