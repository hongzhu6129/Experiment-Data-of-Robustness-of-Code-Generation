```java
import java.util.Scanner;

public class DecimalToBinary {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);
        System.out.print("Input any decimal number: ");
        int decimal = input.nextInt();

        // Using Integer.toBinaryString() for simplicity and efficiency
        String binary = Integer.toBinaryString(decimal); 

        System.out.println("Equivalent binary number: " + binary);
        input.close();

    }
}
```


Explanation:

1. **Input:** The program takes decimal input from the user using a `Scanner`.
2. **Conversion:** The core of the conversion is done by `Integer.toBinaryString(decimal)`. This built-in Java method efficiently converts an integer to its binary string representation.
3. **Output:**  The program prints the resulting binary string to the console.
4. **Resource Management:** The `input.close()` statement closes the `Scanner` to release system resources.  This is good practice, especially in larger programs.

**Example Usage (Corrected for Decimal Input):**

```
Input any decimal number: 7
Equivalent binary number: 111
```
