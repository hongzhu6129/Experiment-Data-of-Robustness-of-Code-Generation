```java
public class MultiplicationTable {

    public static void main(String[] args) {
        // Check if an argument is provided
        if (args.length == 0) {
            System.out.println("Please provide a number as a command-line argument.");
            return;
        }

        try {
            int num = Integer.parseInt(args[0]);

            System.out.println("Input a number: " + num + ":");

            for (int i = 1; i <= 10; i++) {
                System.out.println(num + " x " + i + " = " + (num * i));
            }
        } catch (NumberFormatException e) {
            System.out.println("Invalid input. Please enter a valid integer.");
        }
    }
}
```

**How to compile and run:**

1. **Save:** Save the code as `MultiplicationTable.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile it using the command:
   ```bash
   javac MultiplicationTable.java 
   ```
3. **Run:**  Execute the compiled program with a number as a command-line argument:
   ```bash
   java MultiplicationTable 8 
   ```
   Replace `8` with any other number you want to generate a multiplication table for.


**Explanation:**

* **`public static void main(String[] args)`:** This is the main method where the program execution begins. The `String[] args` parameter allows you to receive command-line arguments.
* **Input Handling:** The code first checks if a command-line argument is provided (`args.length == 0`).  If not, it prints an error message and exits.
* **`Integer.parseInt(args[0])`:**  It converts the first command-line argument (`args[0]`) which is a String, into an integer and stores it in the `num` variable. It uses a `try-catch` block to handle potential `NumberFormatException` if the input is not a valid integer.
* **Loop and Output:** The `for` loop iterates from 1 to 10. Inside the loop, it prints each line of the multiplication table using `System.out.println()`. The calculation `num * i` is performed within the print statement itself.
