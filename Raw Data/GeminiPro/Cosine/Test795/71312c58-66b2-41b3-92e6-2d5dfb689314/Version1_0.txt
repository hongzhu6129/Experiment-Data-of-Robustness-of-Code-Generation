```java
public class ProductOfTwoNumbers {

    public static void main(String[] args) {

        // Check if two arguments are provided
        if (args.length != 2) {
            System.err.println("Please provide two numbers as command-line arguments.");
            return; // Exit the program if the input is invalid
        }

        try {
            // Parse the input arguments as integers
            int num1 = Integer.parseInt(args[0]);
            int num2 = Integer.parseInt(args[1]);

            // Calculate the product
            int product = num1 * num2;

            // Display the result
            System.out.println(num1 + " x " + num2 + " = " + product);

        } catch (NumberFormatException e) {
            System.err.println("Invalid input. Please enter valid numbers.");
        }
    }
}
```

**How to compile and run:**

1. **Save:** Save the code as `ProductOfTwoNumbers.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and compile using the command: `javac ProductOfTwoNumbers.java`
3. **Run:**  Execute the compiled program, providing the two numbers as command-line arguments:  `java ProductOfTwoNumbers 25 5`


This version incorporates error handling:

* **Input Validation:** It checks if exactly two arguments are provided.  If not, it prints an error message and exits gracefully.
* **Number Format Exception Handling:** It uses a `try-catch` block to handle potential `NumberFormatException` if the user enters non-numeric input. This prevents the program from crashing.


**Example Usage:**

```
> java ProductOfTwoNumbers 25 5
25 x 5 = 125

> java ProductOfTwoNumbers 10 20
10 x 20 = 200

> java ProductOfTwoNumbers  // (No arguments)
Please provide two numbers as command-line arguments.

> java ProductOfTwoNumbers a b  // (Non-numeric input)
Invalid input. Please enter valid numbers.
```
