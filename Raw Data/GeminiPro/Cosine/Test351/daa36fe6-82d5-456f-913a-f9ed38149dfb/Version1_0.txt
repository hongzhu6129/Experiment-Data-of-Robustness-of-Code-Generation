```java
import java.util.Scanner;

public classStringLength {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Get the input string from the user
        System.out.println("Enter a string: ");
        String str = scanner.nextLine();

        // Get the length of the string
        int length = str.length();

        // Print the length of the string
        System.out.println("The length of the string is: " + length);
    }
}
```

**Test cases:**

**Normal cases:**

* **Input:** "Hello"
* **Output:** 5

* **Input:** "12345"
* **Output:** 5

* **Input:** "This is a test string"
* **Output:** 19

**Corner cases:**

* **Input:** "" (empty string)
* **Output:** 0

* **Input:** null
* **Output:** 0 (the length of a null string is 0)

* **Input:** "    " (string with only whitespace characters)
* **Output:** 4 (the length of a string with only whitespace characters is the number of whitespace characters)
