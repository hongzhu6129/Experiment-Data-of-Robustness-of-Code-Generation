```java
import java.util.Arrays;
import java.util.Comparator;

public class StringSort {

    public static void main(String[] args) {
        String[] strings = {"apple", "banana", "orange", "grape", "kiwi"};

        System.out.println("Original array: " + Arrays.toString(strings));

        sort_array_by_length(strings, true); // Sort in ascending order
        System.out.println("Ascending order by length: " + Arrays.toString(strings));

        sort_array_by_length(strings, false); // Sort in descending order
        System.out.println("Descending order by length: " + Arrays.toString(strings));

    }

    public static void sort_array_by_length(String[] arr, boolean sort_asc_dsc) {
        // Use a Comparator to sort based on length
        Arrays.sort(arr, new Comparator<String>() {
            @Override
            public int compare(String s1, String s2) {
                int comparisonResult = Integer.compare(s1.length(), s2.length());
                // Reverse the order if descending sort is requested
                return sort_asc_dsc ? comparisonResult : -comparisonResult; 
            }
        });
    }
}
```


**Explanation:**

1. **`main` Method:**
   - Creates a sample `String` array.
   - Prints the original array.
   - Calls `sort_array_by_length` twice: once for ascending order (passing `true`) and once for descending order (passing `false`).
   - Prints the sorted arrays.

2. **`sort_array_by_length` Method:**
   - Takes the `String` array (`arr`) and a boolean flag (`sort_asc_dsc`) as input.
   - Uses `Arrays.sort()` with a custom `Comparator`.
   - The `Comparator` compares strings based on their lengths using `Integer.compare(s1.length(), s2.length())`. This ensures correct handling of potential integer overflow if string lengths are very large.
   - The `sort_asc_dsc` flag controls the sort order.  If `true` (ascending), the `comparisonResult` is returned directly. If `false` (descending), the result is negated (`-comparisonResult`) to reverse the comparison order.  This is a concise way to handle both ascending and descending sorts within the same `Comparator`.


**Example Output:**

```
Original array: [apple, banana, orange, grape, kiwi]
Ascending order by length: [kiwi, grape, apple, orange, banana]
Descending order by length: [banana, orange, apple, grape, kiwi]
```
