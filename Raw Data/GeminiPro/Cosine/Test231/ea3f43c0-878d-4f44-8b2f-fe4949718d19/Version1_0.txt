```java
import java.util.Arrays;

class Solution {
    /**
     * Given a sorted array nums, remove the duplicates in-place such that each element appears only once and returns the new length.
     * Do not allocate extra space for another array. You must do this by modifying the input array in-place with O(1) extra memory.
     *
     * @param nums sorted array of integers
     * @return the new length of the array after removing duplicates
     */
    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueIndex = 0; // Index to track the position of unique elements

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[uniqueIndex]) {
                uniqueIndex++;
                nums[uniqueIndex] = nums[i]; 
            }
        }

        return uniqueIndex + 1;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("New Length: " + len1 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: New Length: 2, Array: [1, 2]


        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("New Length: " + len2 + ", Array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: New Length: 5, Array: [0, 1, 2, 3, 4]

        //Test cases for empty and null arrays
        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("New Length: " + len3 + ", Array: " + Arrays.toString(nums3)); // Output: New Length: 0, Array: []

        int[] nums4 = null;
        int len4 = removeDuplicates(nums4);
        System.out.println("New Length: " + len4 ); // Output: New Length: 0


    }
}
```


Explanation and Key Improvements:

1. **Handles Empty and Null Input:**  The code now correctly handles cases where the input array is empty or null, returning 0 as expected.
2. **Clearer Variable Naming:** `uniqueIndex` more clearly indicates the purpose of the variable.
3. **In-Place Modification:** The code directly modifies the input array `nums` without using extra space, as required by the problem.
4. **Concise Logic:** The core logic for removing duplicates is efficient and easy to understand.
5. **Comprehensive Testing:** The `main` method includes tests with various inputs, including edge cases (empty and null array), to demonstrate correctness.
6. **Output Clarity:** The `main` method prints the new length and the modified portion of the array (up to the new length) for better visualization of the result.  `Arrays.copyOfRange` is used to avoid displaying the unmodified part of the original array.



This improved version addresses the potential issues and provides a more robust and clear solution to the problem.
