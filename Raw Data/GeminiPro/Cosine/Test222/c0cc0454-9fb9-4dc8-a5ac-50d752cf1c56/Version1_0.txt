import java.util.Arrays;

public class RemoveDuplicates {

    static int removeDuplicates(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }

        int uniqueCount = 1; // Initialize with 1 as the first element is always unique

        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {  // Compare with the previous element
                nums[uniqueCount] = nums[i]; // Place the unique element at the next available position
                uniqueCount++;
            }
        }

        return uniqueCount;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 1, 2};
        int len1 = removeDuplicates(nums1);
        System.out.println("Length of array after removing duplicates: " + len1 ); // Output: 2
        System.out.println("Modified array: " + Arrays.toString(Arrays.copyOfRange(nums1, 0, len1))); // Output: [1, 2]



        int[] nums2 = {0, 0, 1, 1, 1, 2, 2, 3, 3, 4};
        int len2 = removeDuplicates(nums2);
        System.out.println("Length of array after removing duplicates: " + len2); // Output: 5
        System.out.println("Modified array: " + Arrays.toString(Arrays.copyOfRange(nums2, 0, len2))); // Output: [0, 1, 2, 3, 4]

        // Test with an empty array
        int[] nums3 = {};
        int len3 = removeDuplicates(nums3);
        System.out.println("Length of array after removing duplicates: " + len3); // Output: 0
        System.out.println("Modified array: " + Arrays.toString(Arrays.copyOfRange(nums3, 0, len3))); // Output: []


         // Test with a null array
        int[] nums4 = null;
        int len4 = removeDuplicates(nums4);
        System.out.println("Length of array after removing duplicates: " + len4); // Output: 0
        // Avoid printing null array as it would throw a NullPointerException
        if(nums4!=null) {
              System.out.println("Modified array: " + Arrays.toString(Arrays.copyOfRange(nums4, 0, len4))); 
        }

    }
}

Key improvements in this version:

* **Handles edge cases:**  It now correctly handles empty (`{}`) and `null` arrays.
* **Clearer output:**  The output now explicitly states the length of the array after removing duplicates and prints the modified array (up to the new length).
* **More comprehensive testing:**  The `main` method includes more test cases, including an empty array and a null array, to demonstrate robustness.
* **Uses `Arrays.copyOfRange`:** Prevents printing elements beyond the new length after duplicate removal, avoiding confusion.
* **More efficient in-place modification:** The algorithm directly modifies the input array `nums` to store the unique elements at the beginning. This avoids creating a new array, which is more memory-efficient.



The `removeDuplicates` function now efficiently modifies the array in place and returns the correct count of unique elements even in boundary cases.  The main method provides clearer demonstrations of how the function works.
