```java
public class CheckJavaInstallation {

    public static void main(String[] args) {
        System.out.println("Java is installed on your computer.");
        System.out.println("Java version: " + System.getProperty("java.version"));
        System.out.println("Java runtime version: " + System.getProperty("java.runtime.version"));
        System.out.println("Java home: " + System.getProperty("java.home"));
        System.out.println("Java vendor: " + System.getProperty("java.vendor"));
        System.out.println("Java vendor url: " + System.getProperty("java.vendor.url"));
        System.out.println("Java class path: " + System.getProperty("java.class.path"));
    }
}
```

**Explanation:**

The fact that this code compiles and runs using a Java compiler and JVM is proof that Java is installed.  The code then uses `System.getProperty()` to retrieve and print various system properties related to the Java installation, providing further details about the installed Java version and environment.  If Java wasn't installed, the code wouldn't compile or run in the first place.
