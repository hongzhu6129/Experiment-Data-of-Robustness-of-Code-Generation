```java
import java.util.Scanner;

public class BinaryToOctal {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter a binary number: ");
        String binaryString = scanner.nextLine();

        // Handling corner cases:
        if (binaryString == null || binaryString.isEmpty()) {
            System.out.println("Invalid input: Empty string.");
            return;
        }

        for (char c : binaryString.toCharArray()) {
            if (c != '0' && c != '1') {
                System.out.println("Invalid input: Not a binary number.");
                return;
            }
        }


        long decimal = 0;
        long power = 1;
        for (int i = binaryString.length() - 1; i >= 0; i--) {
            if (binaryString.charAt(i) == '1') {
                decimal += power;
            }
            power *= 2;
        }


        String octalString = "";
        if(decimal == 0) {
            octalString = "0"; // Special case for input 0
        } else {
            while (decimal > 0) {
                octalString = (decimal % 8) + octalString;
                decimal /= 8;
            }
        }

        System.out.println("Octal equivalent: " + octalString);
        scanner.close();
    }
}
```


**Test Cases:**

1. **Valid Binary:**
   - Input: `101101`
   - Expected Output: `55`

2. **Leading Zeros:**
   - Input: `000111`
   - Expected Output: `7`

3. **Zero:**
   - Input: `0`
   - Expected Output: `0`

4. **Large Binary:**
   - Input: `11111111111111111111` (Max Binary Value within Java's `long` range)
   - Expected Output: `77777777777`

5. **Empty String:**
   - Input: ``
   - Expected Output: `Invalid input: Empty string.`

6. **Invalid Characters:**
   - Input: `101201`
   - Expected Output: `Invalid input: Not a binary number.`

7. **Invalid Characters (Spaces):**
   - Input: `101 01`
   - Expected Output: `Invalid input: Not a binary number.`

8. **Null Input:** 
   - Input: (Enter nothing and press Ctrl+D to simulate null input on some systems/IDEs)
   - Expected Output:  `Invalid input: Empty string.` (The code now handles null as an empty string)



This improved code handles the corner cases, converts the binary to decimal first and then to octal, and provides clear error messages for invalid input. It now also handles null input correctly, treating it as an empty string.  The code adheres to the requirement of only using the `main` method.
