```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class FileToArray {

    public static void main(String[] args) {
        // Replace "your_file.txt" with the actual path to your text file
        String filePath = "your_file.txt";

        try {
            List<String> lines = readLinesFromFile(filePath);

            // Convert List<String> to String[]
            String[] linesArray = lines.toArray(new String[0]);

            // Print the array (optional) to verify the result
            System.out.println(Arrays.toString(linesArray));

        } catch (IOException e) {
            System.err.println("An error occurred while reading the file: " + e.getMessage());
        }
    }


    private static List<String> readLinesFromFile(String filePath) throws IOException {
        List<String> lines = new ArrayList<>();
        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                lines.add(line);
            }
        }
        return lines;
    }
}
```


**Explanation and Improvements:**

1. **Error Handling:**  The code now includes a `try-catch` block to handle potential `IOExceptions` that might occur if the file is not found or cannot be read.  This makes the code more robust.

2. **Resource Management:**  The `BufferedReader` is used inside a try-with-resources block (`try (...) { ... }`). This ensures that the `BufferedReader` is automatically closed even if exceptions occur, preventing resource leaks.

3. **ArrayList for Dynamic Sizing:**  The code uses an `ArrayList` to store the lines initially. `ArrayList` grows dynamically, which is important because we usually don't know the number of lines in a file beforehand.

4. **Conversion to Array:** After reading all the lines into the `ArrayList`,  the code converts it to a `String[]` using `lines.toArray(new String[0])`.  This creates a new array of the correct size and copies the elements from the list.


**How to Use:**

1. **Create a text file:**  Create a text file named `your_file.txt` (or any name you prefer) in the same directory as your Java file or provide the full path to the file.
2. **Add content:** Put some text into the file, with each line representing a separate entry you want in the array.
3. **Compile and run:** Compile and run the Java code. The output will be the contents of the file, printed as a String array.


**Example `your_file.txt`:**

```
Line 1
Line 2
Line 3
```

**Output:**

```
[Line 1, Line 2, Line 3] 
```
