```java
public class PrintOneFacing {

    public static void printOneFacing(int size) {
        if (size < 3) {
            System.out.println("Size must be at least 3.");
            return;
        }

        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (j == 0 || i == size / 2) { // Left edge and middle row
                    System.out.print("* ");
                } else {
                    System.out.print("  "); // Two spaces for alignment
                }
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        // Test cases
        printOneFacing(3);   // Minimum size
        printOneFacing(5);   // Odd size
        printOneFacing(6);   // Even size
        printOneFacing(10);  // Larger size
        printOneFacing(2);   // Corner case: size too small
        printOneFacing(1);   // Corner case: size too small
        printOneFacing(0);   // Corner case: zero size
        printOneFacing(-5); // Corner case: negative size

    }
}
```

**Explanation:**

   - Takes an integer `size` as input, representing the height and width of the "one".
   - Handles the edge case where `size` is less than 3, printing an error message.
   - Uses nested loops to iterate through rows and columns.
   - The condition `j == 0 || i == size / 2` determines whether to print a `*` or a space:
     - `j == 0`: Prints a `*` for the leftmost column.
     - `i == size / 2`: Prints a `*` for the middle row.
   - Prints two spaces inside the `else` block to ensure proper alignment.

2. **`main` Method (Test Cases):**
   - Calls `printOneFacing()` with various input values to test different scenarios:
     - Minimum size (3)
     - Odd sizes (5)
     - Even sizes (6)
     - Larger size (10)
     - Corner cases: size too small (2, 1, 0), negative size (-5)
