

**Key improvements in this version:**

* **Error Handling:**  The `main` method now includes error handling for invalid input:
    * It checks if exactly one command-line argument is provided.
    * It uses a `try-catch` block to handle `NumberFormatException` in case the input is not a valid integer.
    * It checks if the input `n` is positive.
* **Clearer Output:** The `printMatrix` method now prints the matrix elements in a more readable format: `[row,col]`, making it easy to see the row and column indices.
* **Adherence to prompt:** The code strictly adheres to the requested method signatures: `public static void main(String[])` and `public static void printMatrix(int)`.
