

**Negative Cases (Corner Cases):**

* **Input:** `[]` (Empty list)
* **Expected Output:** No output

* **Input:** `null` (Null list)
* **Expected Output:** No output

* **Input:** `["1", "2", "3"]` (Non-string elements)
* **Expected Output:** No output

* **Input:** List containing duplicate elements
* **Expected Output:** Each unique element is printed only once
