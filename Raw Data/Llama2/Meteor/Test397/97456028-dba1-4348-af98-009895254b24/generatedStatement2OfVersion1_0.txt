

**Test Cases:**

* **Normal case:** "I love programming and Java"

    * Expected output: "Lorem si tsilp modor si tsem"

    * Expected output: "Hello world"
* **Empty string:** ""
    * Expected output: ""

    * Expected output: "a" (no odd-length words)
* **Corner case: String with a single odd-length word:** "programming"
    * Expected output: "gnimmargor"
* **Corner case: String with a single odd-length word and leading/trailing spaces:** "    programming    "
    * Expected output: "    gnimmargor    "
