**Problem Description**

Write a Java function that concatenates two strings with itself, ensuring that the resulting string ends in the second string.

**Format**

The format of the word to be used is:
`java <function name> <input string 1> <input string 2>`

**Test Cases**

**Normal Cases**

* Input: `java concatenate "Hello" "World"`
* Output: `"HelloWorld"`
* Input: `java concatenate "Java" "Programming"`
* Output: `"JavaProgramming"`

**Corner Cases**

* Input: `java concatenate "" "abc"`
* Output: `"abc"`
* Input: `java concatenate "123" ""`
* Output: `"123"`
* Input: `java concatenate "" ""`
* Output: `""`
