

**Test cases:**

* **Normal cases:**
    * "hello" -> "helo"
    * "world" -> "word"
    * "programming" -> "programing"
    * "java" -> "java"
    * "python" -> "python"
    * "c++" -> "c++"
    * "javascript" -> "javascript"
    * "ruby" -> "ruby"
    * "kotlin" -> "kotlin"
    * "swift" -> "swift"
* **Edge cases:**
    * "" -> ""
    * "a" -> "a"
    * "aa" -> "a"
    * "aaa" -> "a"
    * "123" -> "123"
    * "!@#$%" -> "!@#$%"
    * "Hello World" -> "Hello World" (case-sensitive)
    * "null" -> "null"
* **Corner cases:**
    * null -> throw NullPointerException
    * String with non-printable characters -> handle appropriately
