

**Test Cases:**

* **Corner Cases:**
    * Two empty strings "" + ""
    * One empty string and a non-empty string "" + "abc"
    * A non-empty string and an empty string "abc" + ""

* **Normal Cases:**
    * "abc" + "def"
    * "hello" + "world"
    * "123" + "456"

* **Edge Cases:**
    * A very long string + a very long string
    * A string with special characters + a string with special characters
    * A string with Unicode characters + a string with Unicode characters
