3. **`showPermutation` Method (Recursive):**

   - **Recursive Step:**
     - Iterates through each character of the input string.
     - For each character, it recursively calls `showPermutation` with:
       - The same input string (to allow repetition).
       - A new prefix formed by appending the current character to the existing `prefix`.
