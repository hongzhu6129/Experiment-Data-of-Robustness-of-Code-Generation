

**Test Cases:**

| Test Case | Input 1 | Input 2 | Expected Output |
|---|---|---|---|
| Positive Numbers | 10 | 5 | 10 |
| Negative Numbers | -5 | -10 | -5 |
| Zero and Positive | 0 | 5 | 5 |
| Zero and Negative | 0 | -5 | 0 |
| Equal Positive Numbers | 5 | 5 | 5 |
| Equal Negative Numbers | -5 | -5 | -5 |
| Positive and Negative | 10 | -5 | 10 |
| Negative and Positive | -10 | 5 | 5 |
| Large Positive Numbers | 100000 | 50000 | 100000 |
| Large Negative Numbers | -100000 | -50000 | -50000 |
| Integer.MAX_VALUE and other | Integer.MAX_VALUE | 10 | Integer.MAX_VALUE |  // Corner case
| Integer.MIN_VALUE and other | Integer.MIN_VALUE | -10 | -10 | // Corner case


**Explanation of Corner Cases:**

* **Integer.MAX_VALUE and other:** This tests the behavior when one of the inputs is the largest possible integer value.  The program should still correctly identify `Integer.MAX_VALUE` as the largest.
* **Integer.MIN_VALUE and other:**  This tests the behavior when one of the inputs is the smallest possible integer value. The program should correctly identify the *other* number as the largest in this case.


**How to run the test cases (Manually):**

1. Compile the Java code: `javac LargestOfTwo.java`
2. Run the code: `java LargestOfTwo`
3. Enter the input values for each test case when prompted.
4. Verify that the output matches the expected output.
