1. **General Case:**

   - Expected Output: Before swapping: a = 5, b = 10

2. **Zero Values:**
   - Input: a = 0, b = 15
   - Expected Output: Before swapping: a = 0, b = 15
                     After swapping: a = 15, b = 0

3. **Negative Values:**
   - Input: a = -3, b = 7
   - Expected Output: Before swapping: a = -3, b = 7
                     After swapping: a = 7, b = -3

                     After swapping: a = 5, b = 5

5. **Large Values:** (Tests potential integer overflow if you were using a smaller data type, though `int` is unlikely to overflow in typical scenarios)

                     After swapping: a = -2147483648, b = 2147483647
