

**Test cases:**

* **Normal case:**
    * Input: "Hello world", "world"
    * Output: true
* **Case where the sequence is at the beginning of the string:**
    * Input: "world Hello", "world"
    * Output: true
* **Case where the sequence is at the end of the string:**
    * Input: "Hello world", "ld"
    * Output: true
* **Case where the sequence is in the middle of the string:**
    * Input: "Hello world", "lo wo"
    * Output: true
* **Case where the sequence is not in the string:**
    * Input: "Hello world", "xyz"
    * Output: false
* **Case where the sequence is empty:**
    * Input: "Hello world", ""
    * Output: true
* **Case where the string is empty:**
    * Input: "", "world"
    * Output: false
* **Case where the sequence is longer than the string:**
    * Input: "Hello", "Hello world"
    * Output: false
* **Case where the string and the sequence are the same:**
    * Input: "Hello", "Hello"
    * Output: true
