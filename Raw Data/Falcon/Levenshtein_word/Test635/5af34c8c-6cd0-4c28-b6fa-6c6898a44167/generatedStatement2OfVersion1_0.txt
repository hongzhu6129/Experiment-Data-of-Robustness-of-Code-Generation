


**Explanation:**

1. **`main` Method:**
   - The `main` method is the entry point of the program.
   - It first demonstrates example usage by directly calling `areBothBetweenZeroAndOne` with hardcoded values and printing the result.
   - Then, it checks if two command-line arguments are provided using `args.length == 2`.
   - If arguments are provided:
     - It uses `Double.parseDouble` to convert the string arguments to double values.
     - It calls `areBothBetweenZeroAndOne` with the parsed doubles.
     - It handles potential `NumberFormatException` if the arguments are not valid numbers.
   - If no or an incorrect number of arguments are provided, it prints a message to the user.


2. **`areBothBetweenZeroAndOne` Method:**
   - This helper method takes two `double` variables as input.
   - It returns `true` if *both* numbers are strictly greater than 0 and strictly less than 1.
   - It uses the logical AND operator (`&&`) to combine the conditions.


**How to run with command-line arguments:**

1. **Compile:** `javac DoubleChecker.java`
2. **Run:** `java DoubleChecker 0.3 0.9` (replace `0.3` and `0.9` with your desired numbers)
