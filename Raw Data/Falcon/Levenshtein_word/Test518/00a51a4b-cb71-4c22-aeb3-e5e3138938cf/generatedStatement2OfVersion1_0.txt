


To compile and run this from the command line:

1. **Save:** Save the code as `StringConcatenation.java`.
2. **Compile:** Open a terminal or command prompt, navigate to the directory where you saved the file, and type: `javac StringConcatenation.java`
3. **Run:**  `java StringConcatenation` (This will use the default "Hello" and " World!" strings.)
4. **Run with user input:** `java StringConcatenation "First String" "Second String"` (Replace with your desired strings).  Make sure to use double quotes if your strings contain spaces.
