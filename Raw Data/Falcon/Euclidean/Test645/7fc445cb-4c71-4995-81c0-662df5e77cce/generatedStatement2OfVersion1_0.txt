

**Test Cases:**

**Corner Cases:**

* **Empty string:** The program should handle an empty string input and print an appropriate message.
* **Single character string:** The program should handle a string with only one character and print that character as the most frequent.

**Regular Test Cases:**

* **String with multiple occurrences of the same character:** The program should find the character with the highest number of occurrences.
* **String with equal occurrences of multiple characters:** The program should print all the characters with the highest number of occurrences.
* **String with special characters or spaces:** The program should ignore special characters and spaces when counting occurrences.
* **String with uppercase and lowercase characters:** The program should treat uppercase and lowercase characters as the same.
