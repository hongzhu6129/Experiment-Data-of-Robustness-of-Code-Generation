

**Test Cases:**

Here are some test cases covering various scenarios, including corner cases:

1. **Regular Positive Numbers:**
   - Input: 10, 20, 30
   - Expected Output: 20.0

2. **Negative Numbers:**
   - Input: -5, -10, -15
   - Expected Output: -10.0

3. **Mix of Positive and Negative Numbers:**
   - Input: -5, 10, 25
   - Expected Output: 10.0

4. **Zero Values:**
   - Input: 0, 0, 0
   - Expected Output: 0.0

5. **Large Numbers:**
   - Input: 1000000000, 2000000000, 3000000000
   - Expected Output: 2.0E9 (or 2000000000.0 depending on how the output is formatted)

6. **Small Decimal Numbers:**
   - Input: 0.1, 0.2, 0.3
   - Expected Output: 0.2

7. **Combination of Large and Small Numbers:**
   - Input: 1000000000, 0.000000001, 5
   - Expected Output: 3.3333333336666667E8 (approximately)



**How to Run the Test Cases (Manually):**

1. Compile the Java code: `javac AverageCalculator.java`
2. Run the code: `java AverageCalculator`
3. The program will prompt you to enter three numbers. Enter the test case input values.
4. Compare the program's output with the expected output.
