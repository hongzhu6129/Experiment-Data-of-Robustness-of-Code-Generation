

**Test Cases:**

**Corner Cases:**

* **Empty List:** Test with an empty list of words.
* **One-Character Words:** Test with a list of words that only contain one character.
* **Words with No Common Characters:** Test with a list of words that have no common characters.

**Regular Cases:**

* **Words with Same Characters:** Test with a list of words that contain some common characters.
* **Words with Different Characters:** Test with a list of words that contain no common characters.
* **Words with Repeated Characters:** Test with a list of words that contain repeated characters.
