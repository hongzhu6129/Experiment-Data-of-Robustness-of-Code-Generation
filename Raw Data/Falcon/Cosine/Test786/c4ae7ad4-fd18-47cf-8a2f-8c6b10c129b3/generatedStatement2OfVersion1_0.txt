

**Test Cases:**

**Valid Cases:**

* **Input:** str = "Hello World!", n = 3
* **Expected Output:** ["Hello", "World!", ""]
* **Explanation:** The string is split into three parts of equal size.

* **Input:** str = "Lorem ipsum dolor sit amet", n = 5
* **Expected Output:** ["Lorem", "ipsum", "dolor", "sit", "amet"]
* **Explanation:** The string is split into five parts of equal size.

**Corner Cases:**

* **Input:** str = null, n = 3
* **Expected Output:** IllegalArgumentException
* **Explanation:** The input string cannot be null.

* **Input:** str = "Hello World!", n = 0
* **Expected Output:** IllegalArgumentException
* **Explanation:** The number of parts must be a positive integer.

* **Input:** str = "Hello World!", n = 10
* **Expected Output:** IllegalArgumentException
* **Explanation:** The number of parts cannot be greater than the length of the string.
