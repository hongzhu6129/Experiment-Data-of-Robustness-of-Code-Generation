

**Test Cases:**

**Corner Cases:**

* Input string is null or empty
* Input string contains non-alphabetic characters
* Specified positions are null or empty
* Specified positions are not within the range of the input string
* Specified positions are not unique

**Positive Test Cases:**

* Input string is valid (e.g., "java")
* Specified positions are valid (e.g., [1, 5, 9, 13, 17])
* Output alpha-helix string is correct (e.g., "jeev")

**Negative Test Cases:**

* Input string is invalid (e.g., "123")
* Specified positions are invalid (e.g., [0, 6, 10, 14, 18])
* Output alpha-helix string is incorrect (e.g., "jv")
