

**Test Cases:**

Here are some test cases covering various scenarios, including corner and edge cases:

1. **Standard Case:**
   - Input: `start = 1`, `end = 99`
   - Expected Output: Odd numbers from 1 to 99 printed, each on a new line.

2. **Start and End Equal (Odd):**
   - Input: `start = 7`, `end = 7`
   - Expected Output: `7`

3. **Start and End Equal (Even):**
   - Input: `start = 8`, `end = 8`
   - Expected Output:  (Nothing printed as 8 is even)


4. **Start Greater Than End:**
   - Input: `start = 10`, `end = 5`
   - Expected Output: `IllegalArgumentException`

5. **Start Less Than 1:**
   - Input: `start = -1`, `end = 5`
   - Expected Output: `IllegalArgumentException`

6. **End Less Than 1:**
   - Input: `start = 1`, `end = -5`
   - Expected Output: `IllegalArgumentException`


7. **Small Range with Even Start:**
   - Input: `start = 2`, `end = 5`
   - Expected Output: `3`
                   `5`

8. **Small Range with Odd Start:**
   - Input: `start = 3`, `end = 6`
   - Expected Output: `3`
                   `5`

9. **Large Range:**
   - Input: `start = 1`, `end = 10001`  (Test a larger range to ensure efficiency)
   - Expected Output: Odd numbers from 1 to 10001, each on a new line.
