

**Test cases including corner cases:**

**Test case 1:**

* Input 1: "" (empty string)
* Input 2: "" (empty string)
* Expected output: true

**Test case 2:**

* Input 1: "Hello World"
* Input 2: "Hello World"
* Expected output: true

**Test case 3:**

* Input 1: "Hello"
* Input 2: "World"
* Expected output: false

**Test case 4:**

* Input 1: "Hello World!"
* Input 2: "Hello World"
* Expected output: false (strings are not equal due to the exclamation mark)

**Test case 5:**

* Input 1: null
* Input 2: null
* Expected output: false (null is not equal to null)

**Test case 6:**

* Input 1: "Hello World"
* Input 2: new String("Hello World")
* Expected output: true (strings are equal even though they are not the same object)

**Test case 7:**

* Input 1: "Hello\nWorld" (string with newline character)
* Input 2: "Hello\nWorld" (string with newline character)
* Expected output: true (strings are equal including whitespace characters)
