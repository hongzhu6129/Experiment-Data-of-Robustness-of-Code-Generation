Write a Java program to check whether a given number is a Disarium number or an unhappy number. A Disarium number is a number defined by the following process : Sum of its digits powered with their respective position is equal to the original number. For example 175 is a Disarium number : As 1+3+5 = 135 Some other DISARIUM are 89, 175, 518 etc. A number will be called Disarium if the sum of its digits powered with their respective position is equal with the number itself. Sample Input: 135.Expected Output
