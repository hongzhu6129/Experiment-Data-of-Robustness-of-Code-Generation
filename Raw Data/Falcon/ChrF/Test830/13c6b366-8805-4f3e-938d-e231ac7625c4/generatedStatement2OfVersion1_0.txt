


This enhanced example includes:

* **`peek()`, `peekFirst()`, `peekLast()`:** Demonstrates retrieving the first and last elements.
* **Empty List Handling:** Shows how to check for `null` when the list is empty.
* **`get()` for Specific Index:** Demonstrates `get()` for accessing elements by index and includes error handling for `IndexOutOfBoundsException`.
* **Clearer Output:** Improved output messages for better understanding.
* **Comments:** Added more explanatory comments.


This covers various scenarios and best practices for retrieving elements from a `LinkedList` without removing them in Java.
