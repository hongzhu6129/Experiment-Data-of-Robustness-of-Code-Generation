

**Test Cases with Corner Cases:**

* **Test Case 1 (No Second Non-Repeating Character):**
   - Input: "aaaa"
   - Expected Output: Character.MIN_VALUE

* **Test Case 2 (Multiple Second Non-Repeating Characters):**
   - Input: "aabbbcddd"
   - Expected Output: 'c' (or any other second non-repeating character)

* **Test Case 3 (Empty String):**
   - Input: ""
   - Expected Output: Character.MIN_VALUE

* **Test Case 4 (String with All Unique Characters):**
   - Input: "abcdefghij"
   - Expected Output: 'b' (or any other second character in the string)

* **Test Case 5 (String with Spaces and Special Characters):**
   - Input: "Hello World! 123"
   - Expected Output: ' ' (space is the second non-repeating character)
