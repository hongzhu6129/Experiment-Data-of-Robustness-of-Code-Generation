

**Test cases:**

* **Normal case:**
    * Input: `"abcde"`, `"fghij"`
    * Output: `false`
* **Same minor ends case:**
    * Input: `"abcde"`, `"fghi"`
    * Output: `true`
* **Empty string case:**
    * Input: `""`, `"abcde"`
    * Output: `false`
* **One empty string case:**
    * Input: `"abcde"`, `""`
    * Output: `false`
* **Special character case:**
    * Input: `"abcde"`, `"fghi!"`
    * Output: `false`
