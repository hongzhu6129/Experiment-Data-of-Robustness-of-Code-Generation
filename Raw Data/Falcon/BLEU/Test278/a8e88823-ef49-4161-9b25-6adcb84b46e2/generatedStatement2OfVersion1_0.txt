


Here's how this code works within the constraints of a single `main` method:

1. **Input:**  It uses a `Scanner` to get the date string from the user.
2. **Date Formatting:** It defines a `DateTimeFormatter` with the pattern "yyyy-MM-dd" to specify the expected format.
3. **Parsing:** `LocalDate.parse()` attempts to convert the input string into a `LocalDate` object.  The `try-catch` block handles potential `DateTimeParseException` if the input is not in the correct format.
4. **Extraction:** If the parsing is successful, it extracts the year, month, and day using `getYear()`, `getMonthValue()`, and `getDayOfMonth()` respectively.
5. **Output:** It prints the extracted components.
6. **Error Handling:** If the date format is invalid, it prints an error message.
7. **Resource Management:** It closes the `Scanner` to release the resource.
