

**Test Cases:**

Here's a set of test cases covering various scenarios, including corner cases:

1. **Normal Case:**
   - Input: `10 20 30`
   - Expected Output: `20.0`

2. **Negative Numbers:**
   - Input: `-5 0 5`
   - Expected Output: `0.0`

3. **Decimal Numbers:**
   - Input: `2.5 3.7 1.8`
   - Expected Output: `2.6666666666666665`  (or approximately 2.67)

4. **Zeroes:**
   - Input: `0 0 0`
   - Expected Output: `0.0`

5. **Large Numbers:**
   - Input: `1000000 2000000 3000000`
   - Expected Output: `2000000.0`

6. **Combination of Positive, Negative, and Zero:**
   - Input: `-10 0 20`
   - Expected Output: `3.3333333333333335` (or approximately 3.33)


7. **Invalid Input (Non-numeric):**
   - Input: `a b c`
   - Expected Output: `"Invalid input. Please enter numbers only."`  (Error message)

8. **Invalid Input (Partial Numeric):**
   - Input: `1 2 a`
   - Expected Output: `"Invalid input. Please enter numbers only."` (Error message)
