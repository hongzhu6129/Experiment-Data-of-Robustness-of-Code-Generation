

**Test Cases:**

**Corner Cases:**

* **Empty vocabulary:** All sets are empty.
* **Single element vocabulary:** All sets have only one element.

**Nominal Cases:**

* **Small vocabulary:** Sets have a small number of elements (e.g., 5).
* **Large vocabulary:** Sets have a large number of elements (e.g., 100).
* **Balanced vocabulary:** All sets have approximately the same number of elements.
* **Unbalanced vocabulary:** One or more sets have significantly more or fewer elements than the others.

**Additional Test Cases:**

* **Long story:** Generate a story with more than 10 sentences.
* **Specific vocabulary:** Use a custom vocabulary for testing, such as a specific domain or genre.
* **Fixed seed:** Use a fixed random seed to generate a consistent story for debugging purposes.
