Here's a set of test cases covering various scenarios, including corner cases:

1. **Normal Case:**

2. **Empty String:**

   - Expected Output: Letters: 0, Spaces: 0, Numbers: 0, Other: 0

3. **Only Letters:**

4. **Only Spaces:**

5. **Only Numbers:**

   - Expected Output: Letters: 0, Spaces: 0, Numbers: 10, Other: 0

6. **Only Special Characters:**

   - Input: `"   This string has leading and trailing spaces   "`

9. **String with multiple consecutive spaces:**
    - Input: `"This  string  has  multiple   spaces"`
