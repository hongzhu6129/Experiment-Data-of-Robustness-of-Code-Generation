

**Test cases:**

* **Normal case:** A list of products is given, and the program should return the products that contain only one word.
* **Empty list:** The program should return an empty list if the input list is empty.
* **Null list:** The program should throw a NullPointerException if the input list is null.
* **List with products containing multiple words:** The program should return a list with the products that contain only one word.
* **List with products containing special characters:** The program should return a list with the products that contain only one word, regardless of whether they contain special characters.
* **List with products containing numbers:** The program should return a list with the products that contain only one word, regardless of whether they contain numbers.
