How Saddleback Search Works:

The algorithm starts at the top-right corner of the matrix.  It compares the current element with the target:

* **If they are equal:** The target is found, and the row/column are returned.
* **If the current element is smaller than the target:** The target cannot be in the current row (since the row is sorted in ascending order).  Move down to the next row.
* **If the current element is larger than the target:** The target cannot be in the current column (since the column is sorted in ascending order).  Move left to the previous column.

This process continues until the target is found or the boundaries of the matrix are reached. This approach efficiently eliminates portions of the search space with each comparison.
