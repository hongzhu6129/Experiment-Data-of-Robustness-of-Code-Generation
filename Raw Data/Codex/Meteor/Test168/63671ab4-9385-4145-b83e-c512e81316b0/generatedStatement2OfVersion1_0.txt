


Explanation of the code using only `main`:

1. **Array Declaration and Initialization:**
   - `int[] originalArray = {1, 2, 3, 4, 5};`  declares and initializes an integer array.

2. **Index and New Value:**
   - `int indexToUpdate = 2;` specifies the index of the element to update (0-based indexing).
   - `int newValue = 10;`  holds the value you want to put in the array.

3. **Bounds Checking (Important):**
   - `if (indexToUpdate >= 0 && indexToUpdate < originalArray.length)` is crucial to prevent `ArrayIndexOutOfBoundsException`. It makes sure the index is within the valid range of the array.

4. **Update Element:**
   - `originalArray[indexToUpdate] = newValue;` updates the element at the specified index directly.

5. **Print Updated Array:**
   - `System.out.println("Updated Array: " + Arrays.toString(originalArray));` prints the modified array using `Arrays.toString()` for easy viewing.

6. **Error Handling:**
   - The `else` block handles cases where the index is invalid, printing an error message and leaving the original array unchanged.


The included helper function `updateElement()` demonstrates how to create a *new* array with the updated element if you don't want to modify the original one. It uses `Arrays.copyOf()` to create a copy of the array before updating the element in the copy.  It also demonstrates good practice by checking the index and returning `null` (or you could throw an exception) if the index is invalid.
