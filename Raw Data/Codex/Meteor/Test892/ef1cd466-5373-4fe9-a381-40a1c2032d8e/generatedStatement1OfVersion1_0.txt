**Problem Description:**

Write a Java program that finds four substrings within a given string, such that they are all the same and appear at specified positions.

**Input Format:**

The input is a string consisting of lowercase and uppercase English letters. The positions where the substrings should appear are specified as four integers.

**Output Format:**

The output is the four substrings found at the specified positions, or "Not found" if no such substrings exist.
