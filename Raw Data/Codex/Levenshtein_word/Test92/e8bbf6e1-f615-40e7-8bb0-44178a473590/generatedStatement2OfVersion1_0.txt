


**Test Cases:**

Here are some test cases covering various scenarios, including corner cases:

1. **Regular Positive Numbers:**
   - Input: 5, 10, 15
   - Expected Output: 10.0

2. **Zero Included:**
   - Input: 0, 5, 10
   - Expected Output: 5.0

3. **Negative Numbers:**
   - Input: -5, -10, -15
   - Expected Output: -10.0

4. **Mixed Positive and Negative:**
   - Input: -5, 0, 5
   - Expected Output: 0.0

5. **Large Numbers:**
   - Input: 1000000, 2000000, 3000000
   - Expected Output: 2000000.0

6. **Decimal Numbers:**
   - Input: 2.5, 3.7, 4.2
   - Expected Output: 3.466666666666667

7. **Corner Case:  All Zeros:**
   - Input: 0, 0, 0
   - Expected Output: 0.0
