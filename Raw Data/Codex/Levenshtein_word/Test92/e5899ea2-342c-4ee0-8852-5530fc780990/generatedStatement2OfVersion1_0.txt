* Input: num1 = 10, num2 = 20, num3 = 30

**2. Negative Numbers:**

* Input: num1 = -5, num2 = 15, num3 = 5
* Expected Output: 5.0

**3. Zeroes:**

* Input: num1 = 0, num2 = 0, num3 = 0
* Expected Output: 0.0

**4. Large Numbers:**

* Input: num1 = 1000000, num2 = 2000000, num3 = 3000000
* Expected Output: 2000000.0

**5. Decimal Numbers:**

* Input: num1 = 2.5, num2 = 3.7, num3 = 1.8
* Expected Output: 2.6666666666666665

**6.  All Same Numbers:**

* Input: num1 = 7, num2 = 7, num3 = 7
* Expected Output: 7.0
