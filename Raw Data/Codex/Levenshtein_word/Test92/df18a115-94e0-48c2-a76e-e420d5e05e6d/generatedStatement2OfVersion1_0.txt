   - Input: -5, -10, -15
   - Expected Output: -10.0

3. **Zero Values:**
   - Input: 0, 0, 0
   - Expected Output: 0.0

7. **Edge Case - Very Large Numbers (potential overflow with integers, but handled correctly with doubles):**

   - Expected Output: A very large number (approximately  1.157920892373162E+308 / 1.333... which should not overflow a `double`)

8. **Edge Case - Very Small Numbers (close to zero):**
   - Input: 0.0000001, 0.0000002, 0.0000003
   - Expected Output: 2.0E-7
