

**Test Cases:**

**Corner Cases:**

* **Empty string:** "" -> ""
* **Single-character string:** "a" -> "aa"

**Normal Cases:**

* "Hello" -> "HHeelllloo"
* "12345" -> "1122334455"
* "abcde" -> "aabbccddee"

**Edge Cases:**

* **String containing spaces:** "Hello world" -> "HHeelllloo  wworrlldd"
* **String containing special characters:** "Hello@123" -> "HHeelllloo@@112233"
