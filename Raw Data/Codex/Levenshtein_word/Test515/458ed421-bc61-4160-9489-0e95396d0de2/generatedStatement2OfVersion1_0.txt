**Corner Cases:**

* Empty string: `""`
* Single character string: `"a"`
* Palindromic string: `"bab"`

* Null string: `null`

**General Cases:**

* Odd-length string with a single character at the center: `"ac"`
* String with no palindromic substring: `"qwer"`
* String with multiple palindromic substrings: `"abccba"`
