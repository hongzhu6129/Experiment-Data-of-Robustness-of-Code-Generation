


Key improvements in this version:

* **Clearer Examples:** Includes examples searching for both existing and non-existing keys.
* **Value Search:** Demonstrates how to search for a value (though less efficient than key search).
* **Comments:** Added explanatory comments to improve understanding.
* **Efficiency in Value Search:** Includes a `break` statement in the value search loop to exit once the value is found, improving efficiency.
* **Error Handling:**  Handles the case where the value is not found.
