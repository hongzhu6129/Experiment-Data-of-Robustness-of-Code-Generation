The fixed-size array method is still shown (commented out) but is not recommended unless you absolutely know the number of lines beforehand.

6. **File Path:** The file path is now a variable, making it easier to change the file you're reading from.  Remember to replace `"your_file.txt"` with the actual path.


How to use:

1. **Create a text file:** Create a text file named `your_file.txt` (or whatever you named it in the `filePath` variable) in the same directory as your Java code.  Add some text to the file, each line on a separate line.

2. **Compile and run:** Compile and run the Java code. The output will display the lines of the file stored in the array.
