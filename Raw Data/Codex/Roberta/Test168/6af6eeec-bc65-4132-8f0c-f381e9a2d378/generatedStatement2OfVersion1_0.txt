


Key improvements and explanations:

1. **Generics:** The `findFirstAndLast` method now uses generics `<T>`, making it work with LinkedLists of any type (Integer, String, etc.), not just integers.
2. **ListIterator:**  Using `ListIterator` is more efficient than iterating through indices with a for loop because we don't need to calculate indices separately.  `ListIterator` gives you the index directly with `nextIndex()`. We iterate through the LinkedList only once.
3. **Clearer Output:** The output now explicitly states when an element is not found, making the educational aspect clearer.
4. **Multiple Examples in `main`:** The `main` method now includes multiple examples:
   - Finding occurrences of an integer.
   - Finding occurrences of a string.
   - Handling the case where the target element is not present. This helps demonstrate the robustness of the code.
5. **Comments:**  Added more comments to explain the logic and the purpose of the different sections of the code.



This improved version is more flexible, efficient, and easier to understand for educational purposes.
