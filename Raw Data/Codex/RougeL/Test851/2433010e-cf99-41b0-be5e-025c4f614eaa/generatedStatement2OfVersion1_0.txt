


This code demonstrates:

1. **Creating a HashSet:** `HashSet<String> hashSet = new HashSet<>();`  (You can replace `String` with any other data type).

2. **Checking for emptiness using `isEmpty()`:**  This is the most direct and efficient way.

3. **Adding elements:** `hashSet.add("element");`

4. **Checking again after adding elements.**

5. **Illustrating a less efficient way (using `size()`):** While it works, `isEmpty()` is preferred.

6. **`clear()` method:**  Empties the HashSet.

7. **Final check after clearing.**


This provides a comprehensive educational example within a single `main` method, as requested.
