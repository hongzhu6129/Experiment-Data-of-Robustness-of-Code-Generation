

**Test Cases:**

**Corner Cases:**

* **Empty List:** Pass an empty list as input. Expected output: An empty list.
* **List with Only Duplicates:** Pass a list where all elements are duplicates. Expected output: An empty list.
* **Null String:** Pass a list containing null strings. Expected output: A list with null strings removed.

**Normal Cases:**

* **List with Duplicates:** Pass a list with several duplicate strings. Expected output: A list with the duplicates removed.
* **List with No Duplicates:** Pass a list with no duplicate strings. Expected output: The same list.
* **List with Mixed Case:** Pass a list with strings of mixed case. Expected output: A list with the duplicates removed, regardless of case.
* **List with Special Characters:** Pass a list with strings containing special characters. Expected output: A list with the duplicates removed, including special characters.
